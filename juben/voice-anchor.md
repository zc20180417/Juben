# Voice Anchor

用于锁定关键角色的说话方式，防止全员同腔。

## 使用原则
- Writer 优先读取本文件；未列出的角色再回退到 `character.md`
- 若本文件只剩模板占位，视为未填写，直接回退到 `character.md`
- 没有原著时，只保留语音指纹；有原著时再补“原著样句 + 抽象特征”
- 这里只存说话方式，不复述人物经历
- 外显台词和 `os` 分开看：外显更克制，`os` 可以更短、更狠，但不能全员金句化

## 核心角色
### 沈青鸾
- 角色功能：女主 / 扮弱上位 / 借刀复仇
- 语速：对外偏慢偏轻，越在强者面前越像收着说；`os` 更短、更冷
- 句式：外显多短句，常拆成两三拍说完；先自贬、再递刀；`os` 常用半句、单词句
- 称呼习惯：对上位者用“臣妾 / 女儿 / 皇上”；对亲近者直接叫“娘 / 翠竹”
- 常用词：不敢、位卑、知道、明白、等、活着
- 禁用词：不当面逞强叫板；不长篇宣言；不说热闹俏皮话
- 狠法：姿态低，话头软，真正的刀放在停顿、转折和没说满的后半句里
- 抽象特征（可选）：
  - 人前像退，人后像算
  - 很少把计划一次说穿
  - `os` 要像心里突然亮出来的刀，不像海报金句

### 萧景珩
- 角色功能：帝王 / 权力刀柄 / 被挑起征服欲的男人
- 语速：稳，压人，不急不慢
- 句式：问句多，命令句短，常连下两三句把人逼住；少解释
- 称呼习惯：自称“朕”；对下多直呼“你”或直接点身份
- 常用词：朕、谁准、怎么、过来、抬头、有意思
- 禁用词：不说絮叨情话；不说市井玩笑；不做长篇自我剖白
- 狠法：先看、先问、再轻描淡写地下判词，让人生畏比当场咆哮更重
- 抽象特征（可选）：
  - 台词里自带掌控感
  - 对沈青鸾时会多一点试探和玩味
  - 真动情时也不该一下变成抒情男主

### 皇后
- 角色功能：后宫秩序持有者 / 终极后宫反派
- 语速：慢，稳，像每个字都掂过轻重
- 句式：规整短句多，常一句一句落地；先摆事实，再压名分
- 称呼习惯：自称“本宫”，很少喊人全名，更多点身份或位份
- 常用词：本宫、不急、规矩、物证、定夺、胆子
- 禁用词：不泼妇式辱骂；不情绪化抢话；不说粗鄙狠话
- 狠法：用名分、规矩、证据和站位压人，像审人，不像吵架
- 抽象特征（可选）：
  - 表面平，底下全是刀
  - 越要置人于死地，越不提高声
  - 她的威压来自“秩序”，不是来自“疯”

### 宸妃
- 角色功能：前期立威用的大反派 / 张扬外放型敌人
- 语速：快，冲，带压迫感
- 句式：命令句、反问句多，喜欢连打；经常一句一句把人往台前逼
- 称呼习惯：自称“本宫”，爱直接点人名或位份
- 常用词：过来、怎么、本宫、听不见、谁给你们的胆子
- 禁用词：不装温柔；不绕弯铺垫；不说太端着的宫规腔
- 狠法：当众羞辱、立刻发号施令，让人下不来台
- 抽象特征（可选）：
  - 她的狠是响的、亮的、要让全场都看见
  - 情绪上来就会继续往前顶，不会自己收刹车
  - 不是深谋型，是立威型

### 沈夫人
- 角色功能：家宅旧秩序的执行者 / 女主家仇线主敌
- 语速：平时稳冷，发火时更硬，不乱
- 句式：训诫句、命令句为主；先压身份，再下规矩，再给威胁
- 称呼习惯：张口常带“相府 / 沈家 / 出身”，很少给人平等称呼
- 常用词：放肆、规矩、相府、沈家、出身、听明白了、贱骨头
- 禁用词：不撒娇；不装可怜；不说市井口水话
- 狠法：把恶意包进体面、家法和规矩里，越像在管教越毒
- 抽象特征（可选）：
  - 她不是纯泼，而是冷着压
  - 末路时也会先嘴硬，再漏出求生
  - 她的核心不是骂人，是“你天生低我一等”

### 沈明珠
- 角色功能：嫡姐 / 嫡庶镜像对手 / 情绪外放型反派
- 语速：快，急，情绪一上来就失控
- 句式：抱怨句、反问句、命令句多；喜欢重复“我不 / 凭什么 / 她也配”
- 称呼习惯：常喊“母亲”；对下人和庶妹多直压
- 常用词：母亲、我不、凭什么、走、她也配、死也不
- 禁用词：不该像皇后那样稳；不该像沈青鸾那样藏；不说太老练的权谋话
- 狠法：撒泼、抢话、踩人，靠身份优越感压人
- 抽象特征（可选）：
  - 她的坏要带“被宠坏”的底色
  - 一急就会把心思全说脸上
  - 她最怕失去体面和高位

### 翠竹
- 角色功能：贴身丫鬟 / 女主情感支点 / 反应器
- 语速：快，脆，反应先于思考
- 句式：直句、问句多，常连着追问；少绕弯
- 称呼习惯：常叫“小主 / 小姐”，自称“奴婢”
- 常用词：怎么了、怎么办、您、小主、奴婢、真、快
- 禁用词：不文绉绉；不说深谋远虑的长句；不装老成
- 狠法：替主子先急、先骂、先冲，忠心是明着摆出来的
- 抽象特征（可选）：
  - 她负责把紧张和担心外化出来
  - 说话比主子更直、更热
  - 她不是谋士，是最先跳起来护主的人

### 淑妃
- 角色功能：中期慢刀子型反派 / 借手杀人者
- 语速：人前慢柔，人后更冷
- 句式：表面周全，句子里带弯；被逼急时会突然变短、变狠
- 称呼习惯：自称“臣妾”，对皇后常叫“娘娘”
- 常用词：臣妾、明白、冤枉、娘娘、怎么、不能再等
- 禁用词：不应长期像宸妃那样炸；不应像皇后那样过分端稳；不说粗暴叫骂口
- 狠法：不亲手弄脏，爱借毒、借谣、借别人去下手
- 抽象特征（可选）：
  - 正常状态是绵里藏针
  - 失控状态才会露出急和恨
  - 她的危险感来自“柔”，不是来自“响”
