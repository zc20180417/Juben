## 主线

1992 年，刚从粮校毕业的青年陆行川回到江北县，正撞见父亲准备把家里承包的旧冷库和制冰车间低价转让给外地倒爷。陆行川带着上一世记忆知道：接下来几年，县域食品、冷链配送、连锁终端、包装材料、大宗原料价格和出口订单都会陆续改变地方商业格局。第一集开场不是讲重生，而是合同即将盖章，他当场按住印泥，把“没人愿意接手的烂冷库”变成翻盘入口。

全剧主线是“用未来认知重建县域供应链”，而不是复刻原著的养猪、饲料、猪价和大豆线。陆行川先保住冷库，通过冰块、果蔬预冷和冻品周转拿到第一笔现金；再用速冻食品、包装材料、县城连锁便利店、冷链车队、原料采购风控、北方淀粉作物基地、食品机械改造和科研合作，把一个快倒闭的家庭小厂做成区域食品供应链集团。

每一阶段都必须有现场阻力：父亲怕赔光家底，外地倒爷催签，银行嫌抵押物破旧，县里担心安全事故，竞品断渠道，冷库断电事故暴露管理漏洞，原料涨价挤压利润，边境订单存在汇率和运输风险，科研合作怕企业短期化。爽感来自“别人只看见一座赔钱冷库，陆行川看见的是食品加工、冷链、终端和原料控制权”。

## 改编重构蓝图

原作核心叙事功能是：重生者在家业即将被卖掉时抢回主动权，用未来认知连续识别时代窗口，从单一家庭资产做成多板块实业集团。新剧保留这个功能，不保留原作具体行业和事件外壳。

新剧核心设定方案：把“养猪场/饲料/猪价/大豆/农机”改为“旧冷库/制冰车间/速冻食品/县域冷链/包装材料/连锁终端/大宗原料风控/食品装备/加工适配科研”。家庭资产从牲畜资产变为冷链基础设施，第一桶金从卖猪价差变为冰块、果蔬预冷和冻品周转；产业链从养殖-饲料-食品扩展，转译为冷链-加工-包装-渠道-原料-装备-科研。

原作功能到新设定的转译原则：凡是原作的“价格窗口”，新剧改成“供需错配、冷链稀缺、原料涨价、渠道缺货、设备升级窗口”；凡是原作的“政府支持”，新剧改成“就业、安全生产、县域食品工业示范、冷链基础设施”；凡是原作的“期货对冲”，新剧改成“白糖、淀粉、油脂等原料采购风控”，强调退出线而非赌博；凡是原作的“科研合作”，新剧改成“加工适配、保鲜工艺、速冻配方和设备改造”。

可保留的是叙事功能、因果压力、人物关系位置、阶段爽点和组织升级节奏。必须改写的是人物姓名、企业名称、行业术语、交易对象、关键道具、现场动作、证据形态、事件触发方式和标志性场景。每集 source.map 必须写清“原作功能”与“新剧事件”之间的转译关系。

后续边界：map-book 不得把原著事件逐集平移；每集都要有 `source_function`、`new_episode_event`、`setting_translation`、`must_change_surface`、`do_not_copy`。writer 不得复用原文台词、标志性事件组合或专有道具，即使人物名已改也视为失败。

## 角色弧光

- 人物改名映射：陆行川 -> 原著林远；陆振海 -> 原著林建军；段启荣 -> 原著王成富；贺远桥 -> 原著高伟良；梅知夏 -> 原著刘玉芳；沈砚山 -> 原著李星和；乔南枝 -> 原著李雨欣；韩策 -> 原著梁永辉；秦峥 -> 原著刘浩；叶承稷 -> 原著孙宇；许望舒 -> 原著张海明。
- 陆行川：从“知道未来、急着救家”的青年，成长为能让父亲、银行、县里、职业经理人和科研人员一起下注的供应链集团掌舵人。他不能说出未来，只能把判断转化为订单、合同、产能、风控和组织。
- 陆振海：旧冷库承包人，务实、怕债、重面子。前期认为儿子疯了，中段用老经验不断提醒风险，后段成为企业的乡土根基和现场压舱石。
- 段启荣：开局倒爷型压力人物，代表“低价拿走旧冷库、把陆家踢出局”的错误路径。功能是制造逼签和第一场反击。
- 贺远桥：县里审慎干部，支持必须来自税收、就业、安全和产业示范结果，不做主角外挂。
- 梅知夏：职业经理人，负责把家庭小厂升级成组织化企业。她必须敢问授权、流程、账目和风险边界。
- 沈砚山：生产负责人，掌控冷库、速冻线、品控和设备维护，所有戏围绕“能否稳定、安全、低成本交付”。
- 乔南枝：品牌和渠道负责人，负责速冻食品、终端铺货、包装和区域渠道战。
- 韩策：大宗原料风控负责人，不是赌徒，所有交易都要绑定仓位、退出线和采购需求。
- 秦峥：退伍执行负责人，负责车队、仓储秩序和突发冲突处理。
- 叶承稷、许望舒：后段科研与食品工程转化人物，负责把企业从规模扩张提升到工艺、设备和长期技术壁垒。

## 关系变化

- 父子关系：从合同桌前撕破脸，到陆振海被订单和账本逐步说服，再到他以老厂主身份替儿子压住工人、供应商和地方关系。认可来自事实，不来自鸡汤。
- 陆行川与县里：从“年轻人拿旧冷库吹牛”到“就业、税收、食品安全和冷链基础设施都能落地”。每次支持都必须经过现场检查和结果验证。
- 陆行川与职业经理人：从一个人拍板变成组织分权。梅知夏、沈砚山、乔南枝、韩策加入后，必须让专业判断限制主角，而不是只替主角报喜。
- 陆行川与市场对手：早期是倒爷和本地竞品，中段是渠道封锁、食品安全事故、原料涨价，后段是跨区域订单、风控和技术壁垒。
- 企业内部：从家庭冷库到供应链集团，会议可以出现，但必须成为选择、授权、追责和资源调度现场，不能写成纯汇报。

## 关键反转

- 开局反转：所有人以为旧冷库是包袱，陆行川知道它即将变成县域食品供应链入口。
- 现金流反转：父亲以为不卖就等死，第一批冰块/预冷订单让冷库当天回血。
- 银行反转：破旧设备原本贷不到款，现场订单、温控记录和县里观察资格让抵押物重新被看见。
- 生产反转：制冷机事故不是灾难收场，而是陆行川借此建立品控、维修和交付纪律。
- 渠道反转：竞品封锁货架，乔南枝用样板店、冷柜和社区终端反开渠道。
- 安全危机反转：食品安全质疑不能靠嘴解释，必须用检测、追溯和责任切割反打。
- 风控反转：大宗原料交易不是暴富桥段，而是采购成本和供应安全的保险。
- 北方基地反转：不是简单倒货，而是原料、设备和加工适配的长期布局。
- 科研反转：最终壁垒不是现金，而是保鲜工艺、速冻配方、设备改造和稳定供应能力。
- 富豪榜收口：外界以为陆行川靠运气翻身，最后发现他握住的是冷链、加工、渠道、原料和技术的完整系统。

## 结局闭环

第 40 集收在“高层例会 + 富豪榜消息”上：陆行川听完冷库、速冻食品、渠道终端、原料风控、北方基地和食品工程合作的汇报，明确下一步扩产、品牌战、风控收手、设备升级和科研投入。富豪榜消息传来时不能只写炫富，而要形成首尾闭环：第一集他按住的是旧冷库转让合同，最后他握住的是区域食品供应链的方向盘。

情绪闭环是陆振海从“你别把家败光”到亲眼看见儿子把旧冷库变成集团；事业闭环是从保住家业到供应链集团阶段成型；开放钩子是富豪榜让企业进入更大牌桌，下一阶段面对的不再是县城倒爷，而是全国品牌、资本和技术竞争。
