## Episode Mapping
- EP-11：第10章至第11章旧案回忆、家宴冲突、失踪疑点。
  - 家宴从“补团圆”开始，但第一句或第一动作就碰到旧案禁区。 -> 沈知微逼问当年失踪细节，沈怀章与许兰因回答前后矛盾。 -> 姜照眠发现自己的被弃与沈知微失踪可能互相牵连。 -> 沈雨棠试图用哭闹转移焦点，被姐妹当场压回证据问题。 -> 结尾落在一个旧物、旧照片或旧证人名字出现。
  - knowledge_boundary:
    - 姐妹知道旧案不是普通意外
    - 父母知道自己当年有隐瞒但未必知道幕后全貌
    - 沈雨棠知道继续查会危及自己。
- EP-12：第11章至第12章调查档案、旧证人、幕后名字浮出。
  - 开场是档案或证人被截，不能只是坐着看资料。 -> 顾砚舟和姜照眠分头推进，一个护现场，一个追问关键人。 -> 证据必须通过行动抢回来：追车、堵门、抢录音、保护证人或现场交锋。 -> 周振岳名字第一次进入主线，显示他与沈家旧案和沈雨棠利益相关。 -> 结尾让周振岳反向盯上姜照眠，危机升级。
  - knowledge_boundary:
    - 主角团知道周振岳可疑但不掌握全部链条
    - 周振岳知道有人在查他
    - 沈雨棠开始和幕后线靠近。
- EP-13：第12章至第13章追查升级、周振岳反制、主角团被威胁。
  - 开场以威胁短信、被跟踪或证人失联切入。 -> 姜照眠坚持继续查，顾砚舟从保护变成并肩行动。 -> 沈知微拿出当年记忆碎片，但记忆会带来情绪冲击和现实风险。 -> 周振岳安排假线索，把主角团引向危险地点。 -> 结尾形成硬钩：姜照眠或沈知微被困，手机信号断或门从外锁上。
  - knowledge_boundary:
    - 主角团知道被反盯
    - 周振岳知道她们接近真相
    - 沈家父母知道事态危险但不知道是否该继续遮丑。
- EP-14：第13章至第14章危险脱身、会议室诱捕、反向栽赃。
  - 开场从被困现场脱身，不跳过危机解决过程。 -> 主角团设计让周振岳以为自己占上风，现场必须有双方博弈。 -> 周振岳露出一段关键口供或行为破绽，但马上启动反咬。 -> 反咬目标指向沈知微或姜照眠，让真相线从胜利变成更大危机。 -> 结尾落在警方、媒体或沈家内部对沈知微产生怀疑。
  - knowledge_boundary:
    - 主角团掌握部分关键证据但不完整
    - 周振岳知道如何利用沈家旧伤制造分裂
    - 外界只看到被剪裁的证据。
- EP-15：第14章至第15章沈知微被栽赃、姐妹信任与沈家站队。
  - 开场是沈知微被公开质疑或带走配合调查，不能慢铺。 -> 沈家父母再次本能动摇，暴露他们最深的不信任。 -> 姜照眠在证据不完整时选择站姐姐，和沈家发生正面对抗。 -> 顾砚舟提出可行救援路径，但姜照眠必须亲自做关键选择。 -> 结尾卡在新的反证线索出现：沈雨棠与周振岳之间有一处无法解释的联系。
  - knowledge_boundary:
    - 姜照眠不知道完整反证，但相信沈知微
    - 沈知微知道部分旧案记忆可能被人利用
    - 沈雨棠以为自己暂时安全。
