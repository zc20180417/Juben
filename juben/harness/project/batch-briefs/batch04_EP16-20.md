## Episode Mapping
- EP-16：第15章至第16章沈知微反证、沈雨棠关键漏洞。
  - 开场用审讯、听证或家庭对质切入，沈知微必须在压力下自证。 -> 沈知微拿出反证，但现场有人质疑证据来源或完整性。 -> 姜照眠补上被忽略的细节，把沈雨棠拉入同一证据链。 -> 沈雨棠试图弃车保帅，把责任推给周振岳或旧佣人。 -> 结尾让一个关键物证指向沈雨棠亲自参与。
  - knowledge_boundary:
    - 主角团开始接近完整真相
    - 沈雨棠知道自己可能保不住
    - 沈家父母第一次无法继续自欺。
- EP-17：第16章后段至第17章真相对质、沈雨棠崩盘。
  - 开场把沈雨棠推到必须解释时间线的位置。 -> 姜照眠和沈知微分别用现场经历、档案和证人压缩她撒谎空间。 -> 沈雨棠情绪失控，暴露她长期恐惧失去沈家位置的真实动机。 -> 沈怀章与许兰因终于面对自己偏心造成的后果，但不能用痛哭抵消责任。 -> 结尾阶段大爽点：沈雨棠被带走或公开失去沈家庇护。
  - knowledge_boundary:
    - 主角团知道沈雨棠参与核心陷害
    - 旧案仍可保留周振岳更深责任
    - 沈家父母知道无法再“两边都要”。
- EP-18：第17章后续，沈家道歉、姐妹关系稳定、姜照眠重新定义亲情。
  - 开场不是大团圆，而是沈家父母迟来的道歉遭遇姜照眠冷静回应。 -> 姜照眠承认血缘，但拒绝把伤害一笔勾销。 -> 沈知微站在姜照眠身边，姐妹关系从共同作战转为日常互护。 -> 顾砚舟不抢家庭戏，只在边界外给姜照眠可选择的退路。 -> 结尾落在姜照眠愿意留下吃一顿饭，但不是完全原谅。
  - knowledge_boundary:
    - 所有人知道伤害不可逆
    - 姜照眠知道自己可以选择修复节奏
    - 顾砚舟知道此时不能越界。
- EP-19：结局前情感线，顾砚舟准备求婚、姜照眠确认自我。
  - 开场用顾砚舟准备求婚被姜照眠察觉或误会切入。 -> 姜照眠先处理事业发布、家庭边界或个人选择，不把婚姻当唯一终点。 -> 顾砚舟明确尊重她是否接受，而不是用盛大安排逼她答应。 -> 沈知微提醒她：被爱不是补偿，被尊重才是答案。 -> 结尾卡在顾砚舟真正求婚前一刻或戒指出现。
  - knowledge_boundary:
    - 姜照眠知道顾砚舟有心意但未确认具体安排
    - 顾砚舟知道她最怕被安排
    - 家人希望她幸福但不能替她答应。
- EP-20：求婚场面、情感确认、主线关系闭环前半。
  - 开场从求婚现场的一个意外或中断切入，保持戏剧张力。 -> 顾砚舟不说“我救你”，而是承认自己曾经伤害、后来学会等待。 -> 姜照眠回应的重点是“我选择你”，不是“我终于被人要了”。 -> 沈家人与沈知微作为见证，态度要克制，不能抢主角情绪。 -> 结尾释放大情绪：姜照眠答应同行，但婚礼和姐姐线留到后续。
  - knowledge_boundary:
    - 两人正式确认关系
    - 沈家知道姜照眠不会因婚姻回到旧控制
    - 沈知微的情感归宿尚未完成。
