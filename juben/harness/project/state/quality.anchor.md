## 代表性打法
- 代表性台词风格：
  - 沈如月：外显更稳，真正的锋利藏在短促问句和最后一击的决定里
  - 裴砚亭：压短句，不废话，重要情意只通过托付与动作放出
  - 权谋配角：说信息时要带立场和代价，不能只当喇叭
- 场景打法：
  - EP-06：用双信 + 父亲下跪完成真相揭示，不靠长解释台词
  - EP-08：用断珠、穿线、显字把宴会场写成权谋机关场
  - EP-10：用虎符、名单、耳语秘密完成从夫妻对峙到共同执局的切换
  - EP-11：用虎符、账本与帝王现身把“翻盘”写成权力视觉冲击，而不是纯口头宣判
  - EP-12：用童年画、糖人与旧纸折痕完成情感解释，不用长篇告白替代画面
  - EP-14：用撤画、拥抱、府门改口把“家庭秩序重写”落到可见物件和动作上
  - EP-15：用妆案、回廊、高阁三层空间完成“自我选择 → 共同生活 → 新朝晨光”的主题闭环
- 留白手法：
  - 真情真相分两步给，先给任务与物证，再给终局解释
  - 集尾优先把“下一步必须做什么”钉住，而不是把情绪写满
  - 终局闭环不靠口号，优先让人物站位、被记名、撤旧画、女子入署这些外部动作自己说话
