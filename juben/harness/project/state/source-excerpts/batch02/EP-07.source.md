第9章 养猪场扩张计划

林建军和外面其它人想的一样，觉得猪肉价格虽然回升了，但是最多也就赚个几万块而已。

　　当知道林远手里还完债还剩18万之后，整个人都愣在了原地。

　　“儿子，你有这本事就算是前两年猪肉行情不好的时候估计也能赚钱！”

　　林建军越看越惊奇，不愧是我的种，就是厉害。

　　林远这也算是对这个时代的养殖户进行降维打击了，等到未来三元猪普及了，料肉比整体降下去了，想这样捡钱就难了，那时候就是刺刀见红的时候了。

　　“这钱你打算怎么花？继续扩建养猪场吗？现在镇上没有空余的养猪场了，想要扩大规模只能去别的镇子，或者自己建了。”

　　林建军有些好奇地问道，他也想知道自己儿子手上有这么一大笔钱，会怎么花。

　　“肯定是要继续扩大养猪场的规模，不过别的镇子确实不太方便，现在有钱了，我打算自建一座存栏1000头猪的养猪场！”

　　林远说话的时候不自觉地攥紧了拳头，语气也带着一丝兴奋。

　　“存栏1000头猪的养猪场？你手上的这些钱应该不够吧？”

　　林建军现在虽然只是经营了一个砖厂，但也算是进入建材行业了，稍微估算了一下猪舍的建造成本，就知道这么大的场子，林远手上的这点儿钱根本不够。

　　听到林建国的疑问，林远有些不好意思地看了眼林建国。

　　“你不会又想继续赊砖头吧？这可不是之前那个小猪舍，这么大一个养猪场，得要多少砖头？我这是小本生意，可赊不起。”

　　林建国连连摆手，让林远放弃这个想法，林远的养猪场需要扩张，自己的砖厂就不需要扩张吗？

　　自己还没有退休呢？正是奋斗的年纪，怎么会为了儿子的事业放弃自己的事业。

　　林远叹了口气，直接上存栏一千头猪的养猪场确实不太现实。

　　“我打算一期一期建，第一期先将大洼村的养猪场规模扩张到500头！然后母猪猪舍也再扩建一下，让母猪的存栏数量能达到100头。”

　　大洼村养猪场就是林远家最开始的养猪场，以现在乡镇的基建水平，直接上马那种大规模的养猪场不太现实，也不太实惠。

　　林远打算用模块化建设的方式一点点扩建原本的养猪场，每个小模块就是一个存栏100头猪的猪舍。

　　连带配套设施，每个猪舍的建造成本大概在两万左右，以林远的资金情况，同时上马五个猪舍，剩下的钱依然能够维持养猪场的正常运转。

　　“我就是一个劳碌命啊！”

　　当天晚上，林远再次在自己房间的书桌上核算了起来。

　　这次先后需要养一千头猪仔，加上林远有钱了，饲料肯定不能像之前那样拖欠半年在付款了。

　　“还好有账期这个东西！”

　　林远想到这，嘴角扯出一抹笑容，虽然这玩意儿很恶心，但是对自己来说确实有好处。

　　林远打算谈好几家固定的原料供应商，然后设置三个月账期，能极大缓解林远的现金压力。

　　特别是出栏前三个月是猪食量最大的时候，实际上比之前还要省。

　　林远打算为这个板块预留3万的流动资金。

　　“可惜工人的工资不能拖三个月！”

　　随着养猪场规模扩大，林远手底下的员工应该能达到30人的规模，每个月光工资就得花四五千，这方面也得预留3万的流动资金。

　　剩下的两万，则是用来买仔猪、种猪和打疫苗的费用。

　　说到仔猪，之前引进的那50头母猪现在已经到了发情期了，需要尽快引进种猪配种。

　　等三个多月后，这些母猪产仔，下的猪仔刚好可以放到新建的五个猪舍里。

　　至于现在这几个猪圈，还是和上一批一样，仔猪一部分自产，一部分外购。

　　这一次这10头母猪比较争气，再加上林远的精心照顾，一共下了150个猪仔，林远只需要再买330头仔猪就行，明天去国营养猪场的时候刚好一起谈下来。

　　有了上一次的经验，这一次去国营养猪场就不用找自己老师带路了，林远一个人骑车来了国营养猪场，然后直接找到了李厂长。

　　李厂长在见到林远手上的手提袋之后，脸上也瞬间露出了热情的笑容。

　　“林厂长，现在仔猪的价格要比去年高上不少，如果还要买三元猪的话，需要一块五一斤。”

　　林远听到李厂长的话在心里松了口气，比自己预想的还要低一些，正常来说，仔猪的价格要是生猪价格的1.5到2倍，李厂长这个价格也就是这个时期了。

　　如果半年后自己再来时还没有搞定仔猪的自产问题，想要再买仔猪的话，价格估计就要到3块钱以上了。

　　可惜林远对于这个年代的猪价只有一个模糊的印象，只知道猪价一直在涨，具体到某个月就不太清楚了，不然的话，还能想办法囤一批仔猪赚一笔差价。

　　“行，那还是麻烦李厂长安排人把仔猪送到养猪场去，货到付款。”

　　林远也没有讨价还价，直接同意了李厂长的价格。

　　“还有就是我打算引进几头种猪，不知道李厂长您这边能不能卖？”

　　“你打算买几头？”

　　李厂长听到林远的话也来了兴趣，国营养猪场为了维持仔猪的繁衍体系，种猪还是有一定的存栏数量，考虑到种猪的价格，如果林远要的不多，确实可以做这笔生意。

　　“我打算引进两头杜洛克公猪，不知道咱们能不能卖？”

　　林远思索片刻，报出了自己的需求，自家猪圈目前有60头母猪，两头种猪的话，虽然会累点儿，但也能应付过来。

　　君不见这个年代多少村子都是一个村共用一头种猪的。

　　“两头的话，那还好，我到时候让人和仔猪一起送过去，不过这个价格的话...”

　　“1500块一头！”林远直接报出了自己的价格，没办法，物以稀为贵，这年代一头种猪的价格就是这么离谱。
