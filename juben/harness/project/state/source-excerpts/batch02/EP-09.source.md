## Original Excerpt
第13章 首富

林建军听到林远的话，也点了点头，土窑虽然投资低，但是缺点实在是太多了，最开始选择投资土窑也是因为手里钱不够。

　　自己的砖厂能赚钱很大程度上是因为现在的建材需求足够旺盛，加上比标准的砖便宜一分钱，这才能卖出去。

　　一旦未来市场没有这么火热了，自己这种砖厂是第一个凉的。

　　“你觉得新砖厂建多大的比较合适？”

　　或许是因为林远这半年时间展露出来的能力，林建军决策时不知不觉间已经习惯先听一下儿子的意见。

　　“未来市场会是什么样的，咱们都不确定，所以肯定不能疯狂举债扩张，我觉得咱们先建一个20门轮窑的小型轮窑厂就行。

　　整体投资大概需要30万，凭借咱们两个手上的现金足够了。”

　　虽然这样对资金会有一些压力，不过林建军原本的土窑砖厂又不会拆掉，每天都在疯狂赚钱，并不会对砖厂和养猪场的经营产生影响。

　　听到林远的建议，林建军狠狠地吸了一口烟，皱眉思考起来。

　　林远的建议对他来说确实有些保守了，就算没有林远的十几万，他都打算贷款硬上24门轮窑，让砖厂年产量突破一千万块。

　　注意到林建军的表情变化，林远也紧张起来。

　　前世林建军就是一直在疯狂扩张，在这个时期直接贷款建了一座24门轮窑的砖厂，随后又在新砖厂投产两个月后，通过借钱和贷款建了一座更大的36门轮窑的砖厂。

　　结果新砖厂建到一半，国家就直接宣布要对房地产进行调控，然后林建军砖厂的订单便腰斩再腰斩。

　　银行那边看着林建军砖厂生意不好，也开始催林建军还款，而林建军前后贷了近百万，私生活又比较奢侈，哪里能还的上这么多钱。

　　之前有多风光，后面就有多狼狈，林建军也因此破产，并且还欠下了大笔外债，成为一切悲剧的开始。

　　“只建一个20门轮窑是不是太保守了？镇政府那边已经答应会帮我协调一笔贷款，以砖厂现在的条件，将之前那三万块给还了之后，再重新贷一笔，贷个二十万的话还是很轻松的，还起来也不会太难。

　　要建新砖厂的话就直接上24门轮窑，建一个中型的砖厂。”

　　林建军将烟头掐灭，缓缓开口道，林建军还是坚持原本的计划，不过林远的加入倒是让林建军可以少贷一些钱，只需要贷款20万就够了，压力相对来说要减小了不少。

　　林远听到林建军的话，也不得不再次思索起来。

　　两个人加起来，足足40万的贷款，对于林家来说，压力确实有点儿大，虽然林远重生后就决定不走林建军的那种举债扩张的路线。

　　但真到了这一步，面对着财富翻倍的诱惑，确实很难控制住。

　　“爸，你说的有道理，是我保守了。”

　　林远深呼了一口气，缓缓开口道，至少砖厂还能疯狂赚钱半年时间，半年后凭借新建的轮窑厂也能维持一定营收。

　　只要不像前世那样疯狂，就不会出太大问题。

　　和林建军聊完，林远便再次回到了自己的养猪场。

　　砖厂后面的事情，有林建军在，也不用林远操心，林远这次就单纯是来给林建军送钱的。

　　坐在自己的办公室，看着书桌上的日历，林远才发现现在时间已经快到了过年的时候了。

　　好在无论是养猪场还是砖厂，都是建在农村，不需要考虑过年期间缺乏工人的情况。

　　现在距离过年还有十天左右，足够给两个建筑打完地基了，等过完年刚好直接开始地面建筑的建造。

　　这样等到三四月份就能投入使用，开始赚钱了。

　　无论是养猪还是建材，红利期或者说暴利期有限，必须得抓紧时间，尽可能地在红利期内赚更多的钱。

　　在金钱的力量下，即使现在已经快过年了，但是林远还是很快便找到了足够的施工队。

　　随着几十万砸下去，大洼村瞬间多了两个巨大的工地，无论是林远的养猪场还是林建军的砖厂，在这个年代都算得上是相当大的投资了。

　　在大洼村和白水镇都造成了不小的轰动，白水镇的人都在猜测林家现在到底有多少钱，白水镇首富的名头也被安在了林建军身上。

　　两笔巨大的投资，甚至还引来了县城电视台的记者对林建军和林远两人进行了采访。

　　林远也借着这个机会发布了自己的“招贤令”，起步200块的工资在荷县引起了不小的轰动。

　　没办法，这个年代的中专生和大学生想让他们来自己这儿，即使有足够高的工资，林远依然没有太多信心。

　　因此将条件下放到了高中，高中生的工资200，中专生350，大学生500。

　　至于中专生的工资为什么这么高，主要还是看重了其潜力，这个年代，特别是农村，都是考不上中专才会选择去高中拼一把。

　　其潜力绝对值得林远花大代价拉拢。

　　要不是开500的工资太离谱，担心会破坏市场被警告，林远都打算直接开500了。

　　至少相比于在这个时期堪称天之骄子的大学生，中专生至少还有希望能挖过来。

　　竟然真有不少人拨打了林远留下的电话，确认招工信息的真实性。

　　可惜来打电话的都是高中生，没有林远期待的中专生和大学生。

　　“大势如此，只能看高镇长那边给不给力了。”

　　林远再次挂掉一个求职者的电话，无奈地叹了口气，想让那些中专生拒绝分配的工作，来自己这个小养猪场确实太难了。

　　不过能招揽十几名高中生，已经在足够让林远高兴一阵了，在这个年代，还是这种农村养殖企业，想要招揽优秀的人才实在是太难了。

　　这些学生的年纪都很小，脑子也比较灵活，培养培养应该也勉强能用了。

　　没有过多犹豫，林远当即向那些表达意向的求职者发出邀请，邀请他们来自家养殖场参加面试。

　　1993年1月20号，距离过年还有一周时间，十几名年轻人来到了大洼村的养猪场。

　　看着热闹的施工现场，一个个都陷入了震撼之中。
