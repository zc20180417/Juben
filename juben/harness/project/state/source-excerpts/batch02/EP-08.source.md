## Original Excerpt
第7章 苏雨柔滚出苏家，姐姐归来

苏家被傅氏断了合作后，资金链瞬间吃紧，公司岌岌可危。苏雨柔把所有的恨都算在了时鸢头上。她不甘心自己锦衣玉食的人生就此毁掉，更不甘心傅斯年把所有温柔都给了那个她看不起的真千金。这天，时鸢在鸢设计工作室召开高定系列新品发布会，全城媒体、名流悉数到场，正是她事业最风光的时刻。苏雨柔混在人群里，眼底淬满毒汁。轮到模特走秀时，意外突然发生——压轴高定礼裙的裙摆，竟被人恶意划破一道长长的口子，精致的蕾丝碎成一片，彻底毁了。全场哗然！记者们的镜头疯狂对准那件报废的裙子，闪光灯闪个不停。“天啊，高定被毁了！”“鸢设计要出大事了！”苏雨柔躲在人群后，嘴角勾起得意的笑。她就是要毁了时鸢，让她身败名裂！工作室的员工吓得脸色发白，手足无措。刘美兰甚至还在一旁假惺惺地叹气：“唉，好好的发布会怎么出这种事，鸢鸢也太不小心了。”就在场面即将失控时——时鸢站在台上，眉眼依旧清冷，没有半分慌乱。她只是淡淡抬眼，看向监控方向，薄唇轻启：“调监控。”话音刚落，大屏幕瞬间切换画面。清晰的录像里，苏雨柔鬼鬼祟祟溜进后台，用美工刀划破裙子的全过程，被播得明明白白！“是苏雨柔！”“苏家那个养女居然干这种事！”“太恶毒了吧，居然故意破坏！”所有人的目光，像刀子一样扎在苏雨柔身上。她脸色惨白，连连后退，尖叫道：“不是我！是他们P图！是时鸢陷害我！”“陷害你？”一道冷冽低沉的声音从门口传来。傅斯年迈步走进来，身后跟着律师与保镖，气场凛冽，步步逼向苏雨柔。他直接将一叠文件甩在她面前，声音冷得结冰：“你买通工作人员的记录、凶器上的指纹、以及你私下造谣时鸢的证据，全都在这里。”“苏雨柔，恶意破坏商业展会、诽谤他人、蓄意滋事，你觉得，你跑得掉？”苏雨柔腿一软，直接瘫坐在地上。刘美兰疯了一样冲上去护着她：“傅斯年！你不能这样！雨柔不是故意的！你放过她吧！”“放过她？”傅斯年冷笑一声，伸手将台上的时鸢轻轻护到身边，动作温柔，语气却狠绝至极：“她敢动我的人，就该付出代价。”“从今天起，苏雨柔所有的社交账号、学历背景、设计作品，我会让人全部公开——包括你多年来抄袭他人设计的证据。”“我会让整个海城都知道，苏家捧在手心里的千金，是个心肠歹毒的小偷。”苏雨柔浑身发抖，崩溃大哭：“我错了！我真的错了！时鸢，求你饶了我！”时鸢垂眸，目光淡漠地看着她，没有一丝同情。“我从来没有招惹过你。”“你占了我的人生，享了十几年的福，还不知足。”“现在，该还了。”她转头对律师吩咐：“按法律程序走，该起诉起诉，该赔偿赔偿。”“还有——”时鸢的目光扫向脸色死灰的苏振宏夫妇，声音清晰而冰冷：“苏雨柔恶意破坏我的发布会，给我造成的损失，由苏家全额承担。”一句话，压垮了苏家最后一根稻草。苏振宏看着满地狼藉，再看看瘫在地上撒泼的苏雨柔，终于彻底清醒。就是这个女人，毁了他的家，毁了他的公司，毁了他和亲生女儿唯一的机会！他气得浑身发抖，扬手狠狠一巴掌甩在苏雨柔脸上！“滚！”“我们苏家没有你这种狼心狗肺的东西！”“从今天起，解除收养关系，你给我滚出苏家，永远不要再出现！”这一巴掌，彻底打醒了所有人。刘美兰看着面目狰狞的苏雨柔，再想想被她们伤透的时鸢，眼泪终于落了下来，却一句话都不敢再说。保镖上前，直接将哭喊挣扎的苏雨柔拖了出去。从此，她从风光无限的苏家千金，变成人人唾弃的过街老鼠，彻底滚出了海城上流社会。发布会现场恢复平静。所有媒体看向时鸢的目光，只剩下敬畏与崇拜。而她身边的傅斯年，微微低头，语气温柔得能化开冰雪：“没事了，有我在。”时鸢抬眸，撞进他深邃温柔的眼底。这一次，她没有躲开。窗外阳光正好，台上的女人光芒万丈，身边的男人护她周全。所有恶人得到报应，所有委屈终被抚平。尘埃落定，苏雨柔狼狈离去，再无立足之地。就在苏家恢复平静，时鸢渐渐放下过往心结之时，一道熟悉又陌生的身影，出现在了苏家别墅门口。女子眉眼与时鸢有七分相似，气质温婉却带着几分历经世事的沉稳，望着时鸢的眼眸里，盛满了失而复得的激动与心疼。她，就是时鸢失散多年的亲生姐姐——苏清月。
