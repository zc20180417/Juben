## Original Excerpt
第11章 高伟良的支持

林远看着眼前这两人，只感觉有些眼熟，却又想不起具体是谁，毕竟两人确实没有什么交集。

　　“林老板，你好，我是白水镇的镇长高伟良，终于见到你了。”

　　高伟良注意到林远眼中的疑惑，笑着说道。

　　“高..镇长！您过来，是”

　　听到高伟良的自我介绍，林远被吓了一跳，说话的声音都有些结巴。

　　“哈哈，林远你别紧张，你扩建养猪场的事儿，镇上很重视，所以我过来看看你这边是不是需要什么帮助。”

　　“帮助？”

　　听到高镇长的话，林远眼睛一亮，嘴角扯出笑容，不过很快恢复镇定。

　　“高镇长，我刚卖完猪，资金暂时还比较充足，需要帮助的地方主要就是人才的稀缺。

　　养猪，特别是大规模养猪，也是需要有技术要求的，防疫、治病甚至喂猪和清理猪圈都需要一定的技术。

　　我这么大一个养猪场，真正算得上懂养猪技术的只有我自己，光着几百头猪都有些忙不过来了，需要找兽医站的兽医帮忙。

　　但是一直这样也不是一个办法，而且没有人才我根本没办法扩大养猪场，只能勉强维持现在的规模。”

　　林远苦着个脸，向高伟良说起自己养猪遇到的困难。

　　这也是他决定养猪前没有想到的，这个年代，特别是乡下，想要得到人才，哪怕是养猪的人才，都太难了。

　　村里招募来的那些野路子，干那种几十头猪的小养猪场还行，根本满足不了大型养猪场的需要。

　　“没想到林远你年纪轻轻，看问题竟然这么透彻。”

　　高伟良听到林远的话也闪过一丝意外，之前自己也和不少民间的小老板打过交道，一提到帮助，都是各种要钱，看问题也都是浮于表面。

　　像林远这种直接抓住核心，让政府帮忙寻找专业人才的还是第一个。

　　“你的问题我记下了，政府这边会尽力帮你寻找，你还有别的需求吗？比如资金和土地之类的。”

　　高伟良思索片刻，没有敢打包票，在这个年代想让那些专业人才来私企，还是这种小养猪场，难度确实有些大。

　　“谢谢高镇长，只要愿意来我这边，每个月的工资中专生三百块起步，大学生五百起步，上不封顶。”

　　虽然没有得到保证，但是能让高伟良帮忙就已经很不错了，一个镇长怎么也比自己一个养猪场的小老板有面子吧。

　　“至于土地和资金，这方面确实也有需求，如果可以的话，我想要拿下大洼村养猪场周围150亩的荒地，我计划在未来几年，建造一个每年能够出栏五万头猪的大型养猪场！

　　资金这边我也想要和镇信用社沟通一下贷款的问题。”

　　林远稍微思考了片刻，又继续说出了自己的需求，既然高镇长都让自己说了，那自己也不用太客气。

　　“你倒是自信。”

　　高伟良听到林远的豪言壮语，脸上也是露出了笑容，还是年轻好啊，光这份自信就超过了白水镇大部分老板了。

　　而且万一真的建成了，按照现在的猪价，那可是至少两千万的营收啊！已经相当于白水镇现在一半的GDP了。

　　“你今天回去准备一下，明天你来镇政府，我到时候约上土地局和信用社的负责人，咱们一起讨论一下。”

　　高伟良看着紧张的林远，缓缓开口，表达了自己的支持。

　　“谢谢高镇长！我回去就好好准备，您放心，我一定会努力将养猪场建成全鲁省甚至全国最大的养猪场的！”

　　林远大喜过望，激动地握住高伟良的手，没想到高镇长这么给力，这下自己可以把步子迈得更大一点！

　　次日，镇政府的一间会议室里。

　　土地局和信用社的负责人看着自己手上的计划书，眉头紧皱，时不时又用笔在本子上记下什么。

　　“高镇长，不是我们不想支持，实在是林老板的规划太夸张了，这可是整整一百亩地，想要将其完全利用，花费的资金可是海量的。

　　按照林老板的计划，第一批只建造15个可以容纳一百头猪的猪舍，连十亩地都用不了，想让我们一口气批下150亩确实有些难。”

　　土地局的负责人率先开口，表达了自己的困难。

　　高伟良点了点头，又看向了信用社的负责人。

　　“高镇长，以林老板的实力，想要贷款，我们信用社肯定是非常乐意的，但是20万这个数字实在是太高了，并且林老板还不能拿出足够的抵押物，凭借那未建成的猪舍，想贷款这么多，我们这边也很难操作。”

　　信用社的负责人也同样一脸为难。

　　“有困难是正常的，今天把大家叫到一起，就是为了解决困难，研究出一个最合适的方案。”

　　高镇长听到两位负责人的话，也是感觉有些头大，不过还是坚定地表达了自己的想法。

　　林远的养猪场，布置镇上，就连县里现在也在关注，养猪场发展得怎么样，这可关系到了他高伟良未来的前途。

　　“林远，你是怎么想的？说说你的想法。”

　　“两位领导提出的问题都非常到位，我也理解两位领导的顾虑，我是这么想的，土地的话，我们可以签一个对赌协议，我保证一年内会开发20%的土地，实现每年出栏一万头的目标；

　　两年内开发50%的土地，实现每年出栏三万头猪的目标，三年内将150亩地全部开发，建成一个年出栏五万头猪的大型养猪场。

　　如果相应时间内，达不到设定目标，政府这边可以将土地收回。”

　　“斯！”

　　听到林远的话，几位领导心里都忍不住吸了一口冷气，年轻人就是猛，这是真的敢想敢说啊。

　　“至于贷款，我同样可以签一个定向协议，保证将这20万贷款全部用在猪圈的扩建上，新投资的几座猪舍，其价值绝对超过20万。

　　还款问题也不用担心，几位领导可能不清楚，我现在养猪场里面的猪全部都是三元猪，价格能比市面上普通的猪肉贵上几毛钱，而且存活率至今仍保持在100%，赚取的利润决定能够支撑还款和养猪场的扩张。”

　　“啪啪啪！”

　　林远说完，高伟良带头鼓起掌来，看向林远的眼神，带着掩饰不住的欣赏。
