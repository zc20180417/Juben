## Original Excerpt
第15章 求婚

压在苏家二十多年的阴霾，在这一刻，彻底烟消云散。苏父苏母老泪纵横，紧紧握住两个女儿的手，哽咽得说不出一句完整的话。这么多年的恐惧、愧疚、担忧，在真相大白、恶人落网的这一刻，尽数化作失而复得的庆幸。苏清月轻轻拍着时鸢的背，眼底是化不开的温柔：“小鸢，都结束了，以后再也没有人能欺负你。”时鸢靠在姐姐怀里，紧绷了一整晚的神经终于松懈，鼻尖一酸，眼泪无声落下。这不是委屈，不是恐惧，而是苦尽甘来的释然。而自始至终，有一道目光，从未从她身上移开过。傅斯年一步步走近，黑色西装衬得他身姿挺拔，墨色深眸里，只盛得下她一人。他没有说话，只是伸出手，稳稳将时鸢从苏清月怀里轻轻牵过来。掌心的温度滚烫而坚定，让她瞬间安定。苏清月看着两人相握的手，眼底泛起笑意，自觉后退一步，将空间留给他们。所有人的目光，都悄然落在这对历经风雨的恋人身上。傅斯年抬手，轻轻拭去时鸢眼角的泪，动作温柔得近乎虔诚。“时鸢。”他开口，声音低沉磁性，穿透寂静，落在每一个人心尖。“从我第一次见到你，你满身锋芒，却藏着一身委屈。我心疼你的倔强，欣赏你的坚韧，更想把全世界的安稳，都捧到你面前。”“过去的黑暗，我没能全程参与；但往后的每一天，我都不想缺席。”话音落下，他缓缓单膝跪地。全场瞬间屏息。苏父苏母捂住嘴，又惊又喜；苏清月含笑望着，满眼祝福。傅斯年从西装内袋里取出一只丝绒戒指盒，轻轻打开——一枚设计简约却极致璀璨的钻戒静静躺在中央，钻石切割完美，在灯光下折射出细碎又耀眼的光，内侧，刻着极小的两个字：鸢·年。“我不要你再独自扛风雨，不要你再步步为营，不要你再受半分委屈。”他抬眸，眼底是从未有过的认真与深情：“时鸢，嫁给我。往后，我护你，爱你，宠你，一辈子。”空气安静得能听见心跳。时鸢垂眸，望着跪在自己面前、放下所有骄傲与权势的男人，眼泪再次滑落，这一次，却是滚烫的幸福。那些颠沛流离，那些阴谋算计，那些孤立无援……全都在这一刻，有了最好的归宿。她轻轻点头，声音带着哭腔，却无比清晰：“我愿意。”三个字落下。傅斯年眼中瞬间亮起漫天星光，他郑重地将戒指戴在她的无名指上，起身，将她紧紧拥入怀中，力道之大，仿佛要将她揉进骨血里。“谢谢你，来到我身边。”全场响起热烈的掌声与祝福。苏清月走上前，笑着拍了拍傅斯年的肩：“傅总，我把妹妹交给你了，你敢让她受委屈，我第一个不饶你。”傅斯年拥着时鸢，沉声承诺：“此生绝不负她。”月光透过落地窗洒进来，温柔包裹着相拥的恋人，也照亮了一家人团圆的笑脸。所有风雨终成过往，所有黑暗迎来黎明。时鸢靠在傅斯年怀里，听着他沉稳有力的心跳，轻轻闭上眼。她终于明白——越过山海，跨过阴谋，她等来的，不只是真相与亲情，还有一份一生一世、至死不渝的爱。
