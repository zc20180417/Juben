## Original Excerpt
第32 章 万家超市

对于专业上的事情，高伟良也很难提供什么建议，不过其他方面，高伟良还是能提供些许帮助的。

　　菜篮子工程可是实行的市长负责制，让林市长来大洼村视察，以远丰养猪场展现出来的潜力，一定能得到重视，到时候就真的一飞冲天了。

　　“真的吗？太感谢你了，高书记，林市长什么时候能过来？”

　　林远被巨大的惊喜砸中，说话都有些语无伦次起来。

　　高伟良看到林远的表现，脸上露出笑容：

　　“这个要等你的那批母猪猪舍落成了，1000头母猪，就连曹州市的国营养猪场都没有这么多，绝对能在曹州轰动一把，再加上远丰的育肥猪的规模，林市长来视察的概率很大。”

　　高伟良也没有打包票，虽然他也有那么一点点关系，但是可没有办法直接决定一个市长的行程，最关键的还是林远这边要给力。

　　......

　　时间流逝，来到了九月份，距离高伟良视察远丰养猪场已经过去一个多月时间。

　　平静已久的猪价终于开始躁动了起来，从8月份的一块九涨到了九月初的两块，到了九月中旬，猪价已经来到了两块一毛。

　　刘志刚等猪贩子也开始兴奋起来，虽然距离林远的这一批猪出栏还有小一个月时间，但是他们已经开始提前和林远联系了。

　　一方面是猪价上涨，他们获得的利润能更高，另一方面则是林远出栏的量够大，1500头，还是三元猪，随便喝口汤就能赚不少钱。

　　不过林远直接将他们推给了刘玉芳，只需要等刘玉芳卖完猪后看一下报表就行。

　　林远自认为自己在讨价还价方面，是比不上刘玉芳的，这样还能躲躲清净，将精力放在更重要的事情上，比如林建军的超市。

　　因为林建军选择的铺面都是现成的，不需要从头开始建，只需要简单装修一下，然后把货架摆和收银设备摆上就行，所以效率很高。

　　只用了两个月时间，四家店铺便完成了选址、装修和备货，以及员工培训工作，准备在9月30号中秋节这天正式开业。

　　只是现在林远和林建军却为了超市的名字陷入苦恼之中。

　　超市这种东西以后肯定是要向连锁的方向发展，肯定不能再用建军、建远这种名字，最好有一个足够大气并且朗朗上口的名字才行。

　　“要不咱们就叫万家超市吧？”

　　林家客厅，林建军掐掉了烟，缓缓开口道，然后在纸上写下了“万家”两字。

　　“咱们是向百佳超市学习的，那咱们的目标就要超过百佳超市，我本来打算直接叫万佳超市的。

　　不过万佳这个商标已经被别人注册了，所以我打算改一个字，将‘最佳’的‘佳’，改成‘家庭’的‘家’。

　　不仅不用担心商标的问题，寓意也直接变了，意味着万千家庭的选择，显得更加温馨，名字也很大气顺口，非常适合超市行业。”

　　听完林建军的话，林远表情有些古怪，“万家”这个名字有点儿耳熟啊。

　　前世好像润华收购了万佳百货之后，就改成了润华万家，不过那是零几年的事儿了，现在确实还没有万家超市。

　　就是不知道自家用了这个名字，润华以后该怎么办。

　　“这个名字确实很不错，那就用这个名字吧。”

　　林远点了点头，他也不擅长取名，很难想到比万家更好的名字了。

　　“名字定下来了，接下来就是宣传的事儿了，虽然凭借咱们超市的现金模式，未来肯定会爆火的，但是该有的宣传还是要做的，最好能让咱们的超市一炮而红。

　　小远，你对于超市的宣传，有没有什么好办法？”

　　林建军见林远同意了自己为超市取的名字，继续开口说道。

　　听到这话，林远再次皱眉思考了起来，这个时代互联网的存在感几乎没有，几家小超市开业也用不到上电视，因此宣传超市的话只能使用一些线下宣传的方法。

　　对于这方面林远没有吃过猪肉，还没有见过猪跑吗？

　　“宣传渠道上，除了派人去发传单之外，我觉得可以直接包圆荷县所有公交车的广告位，在公交车上贴上咱们超市的宣传广告。

　　还可以雇几辆摩托车，带着外放喇叭，在全城宣传，总之要想尽一切办法，让全县人都知道咱们万家超市开业的事情。”

　　林远开口就说出了几个非常有效的宣传方式，让林建军眼前一亮，像是看一个宝物一样看着林远。

　　林远虽然被看得有些不自在，但还是继续开口道：

　　“然后就是开业大酬宾了，除了商品打折之外，咱们还可以搞一个抽奖活动，消费满50元就可以抽奖一次，奖品可以咱们可以选择摩托车、电视机、自行车，噱头一定要够。”

　　林远又说了一下自己的活动策划，本来还打算再加上特价商品的，每天一款特价商品，绝对能引爆全县。

　　不过林远感觉对于这个时代来说，打折加抽奖就够用了，甚至搭配上超市模式本身的先进性，光这些就已经有些资源浪费了，所以特价商品这招可以留着以后再用吧。

　　“哗啦啦”

　　林建军听林远说完，忍不住鼓起掌来，这次林远是真的惊艳到自己了，这一连串策略下去，万家超市想不火都难。

　　“不过，消费满50元才能抽奖，价格有些高了，满20元就行。”

　　林建军鼓完掌又补充了一句，虽然自己对超市很有信心，但是现在普通人的工资普遍只有一百多，甚至几十块。

　　想让他们一口气在超市花一半的工资确实有点儿难，这样抽奖的意义就淡了，不如将门槛定在20块，可以将人们购物的积极性彻底激发出来。

　　“是我的疏忽，50块确实有些高了。”

　　林远点了点头，虽然自己已经在努力适应这个时代的物价了，但有时候确实还会出现一些疏漏。

　　确定好宣传计划之后，林建军便开始行动起来，在荷县开始了轰轰烈烈的宣传活动。
