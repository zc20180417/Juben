## Original Excerpt
第37 章 市长视察（上）

几人一路走过母猪猪舍、育肥猪猪舍、饲料厂、发酵厂，林远不断介绍着远丰的发展情况，林正国也时不时点头，表示对林远的认可。

　　“林市长，几位领导，这里是我们的正在建造的新猪舍，随着新的一批资金到位，我们养猪场又进行了新一轮的扩张，到时候我们的育肥猪存栏量能到达一万五千头，母猪存栏量能达到2000头。

　　加上远丰在外面的一批小型养猪场，预计每年能出栏三万六头猪，向市场提供八百万斤猪肉，履行好作为菜篮子的责任。”

　　站在庞大的施工现场面前，林远看着几位领导，有些兴奋的介绍道。

　　林正国满意地点了点头：“很不错，我期待它建造完成的那一天，资金方面没有问题吧？”

　　“我父亲开了一家贸易公司，我这边占有一些股份，每个月都能拿到一笔分成，并且几个月后还会有两批猪出栏，足够保证猪舍的落地了。”

　　“做贸易可比你辛苦养猪赚钱容易多了，没有想过专心去做贸易吗？”

　　林远摇了摇头：“贸易确实赚钱，但钱再多也只是一串数字，够用就好，相比之下我更愿意做一些更有意义的事情，比如养猪，虽然现在只能做到给国人的餐桌提供更多的猪肉。

　　但是未来或许我可以继续扩大规模，改良猪种，降低猪的养殖成本，让国人实现猪肉自由。

　　事实上，如果不是因为缺钱，我也不会和我父亲去做贸易。”

　　林远说话时表情非常认真，再配合着他那张年轻的脸，看起来就更加真诚。

　　几位领导听完都不禁有些动容。

　　连林正国都没想到自己只不过是随口一问，结果林远竟然有这么大的抱负。

　　林正国拍了拍林远的肩膀。

　　“林远，希望你能记住今天说的话，也希望真的有那么一天，再带我们进去逛逛吧。”

　　很快众人便走到了正在建造的沉淀池旁边。

　　“林市长，几位领导，这是养猪场污水处理系统的沉淀池，随着养猪场规模继续扩大，养猪场产出的污水将达到一个较大的规模，将会对大洼村的生态环境产生一定影响。

　　秉着负责任的态度，我们特意引进了国内最先进的污水处理系统。

　　经过处理后的污水将是大洼村的蔬菜基地最好的水源，还有养猪场产出的粪便，经过发酵后也是极为优质的肥料。

　　蔬菜大棚的蔬菜中卖不出的部分，我们养猪场也会回收，将其作为猪饲料的一种补充，满足猪对一些微量元素的需求。

　　到时候大洼村将建立一个初步的循环系统，提高资源的利用效率避免浪费的同时，还能尽可能的保护环境。”

　　林远自信地向林正国汇报着自己的杰作，林正国的表情再次出现了一丝变化。

　　之前的内容更多的还是和居民的菜篮子有关，而这一次蕴含的信息可明显要超脱这个范畴了。

　　“林远，你给我的惊喜真是越来越多了，你的这套模式非常先进，也非常契合我国的国情，完全可以全面推广。

　　还有你身上的责任心也值得所有企业家去学习，在赚钱的时候，也不能忘记自己的社会责任。

　　这套污水处理系统不仅保护了大洼村的水源，还能惠及全村，带领大洼村一起致富，共同富裕的道路就是需要更多你这样的人站出来。”

　　林正国看林远，眼中带着毫不掩饰的欣赏，相比上一辈的企业家，林远这种年轻人身上的朝气、责任感和使命感实在是太难得了。

　　要不是需要考虑避嫌的问题，林正国都想让林远当自己干儿子了。

　　“谢谢林市长，这些都是我应该做的。”

　　“林远，远丰的发展现在有什么困难吗？”

　　等众人参观完养猪场来到远丰的会议室时，终于来到了最关键的时刻，林正国看着有些局促的林远温和地说道。

　　林远深吸了一口气，让自己冷静下来，然后开始按照提前想好的内容开始和林正国汇报起来。

　　“林市长，目前，以及未来猪肉的市场行情都非常不错，因此在现金流上我们是没有问题的，现在限制远丰继续扩张的就是土地和人才。

　　大洼村这边的养猪场的承载极限就是两万五千头，预计用不了多久就能达到极限，因此我们需要一个合适的场地建造新的、更大的养猪场。

　　如果可以的话，我希望这栋养猪场的极限承载能力能够达到十万级别的存栏量，需要的面积比较大，我们自己去审批难度可能会有些大。

　　此外就是人才了，目前远丰已经有了自己的饲料厂和母猪培养体系，但是受限于人才水平，只能做一些基础的工作，很难进行持续的优化和迭代。”

　　林正国听到这，脸上露出了好奇：“林远，你也不行吗？我听说你家的饲料就是你自己研究出来的，效果可比市面上卖的要好上不少。”

　　林远听到这话，脸上露出苦笑：“林市长，我只是一个普通的中专生，能搞出饲料来也有很大的运气因素在里面，我学习的主要内容也是技术方面的，不是偏向研发的，距离那种能够带队研究的专家，还有很远的一段距离。”

　　林正国点了点头，笑着说道：“哈哈，是我想多了，不过你还年轻，想要向这方面发展的话，完全可以利用空余时间去读个大学，相信以你的能力，应该有很多学习都愿意收的。

　　你接着说吧，除了土地和人才之外，还有那些困难？”

　　林远摇了摇头：“主要就这两个，其他的我们自己就可以轻松解决。”

　　“真的没了？错过这次机会，下次见到我可就不知道要等到什么时候了。”

　　看着林远认真的表情，林正国开玩笑似地说道。

　　“就这些，林市长。”

　　林远不是一个喜欢占国家便宜的人，有前世的记忆在，林远并不需要太担心钱的问题，只要解决这些问题，林远自然能带领远丰狂飙猛进。

　　“好，那我就帮你解决这两个问题。”

　　林正国看出林远内心的坚定，也不再多劝，他很期待林远未来能给自己带来怎样的惊喜。
