## Original Excerpt
第76 章 “三角贸易”

“小林总，远东的木材除了卖到龙国国内，还可以卖到小日子和棒子国，特别是小日子，那些商社一个个都是狗大户，结账都是用美元或者日元现结，对咱们来说，都是外汇。

　　今年，因为咱们和大毛之间的贸易摩擦，现在小日子已经成为远东商品出口的第一目标国了。

　　咱们公司还是一家港岛公司，在和小日子的贸易上更有优势，只要在远东注册一家分公司，然后想办法打通各个关节，就能将远东的木材、鱼产品还有钢材卖到小日子了。”

　　段兴站起身说了一下自己的想法，他之前就一直在和小日子做生意，很清楚小日子到底需要什么，也清楚小日子到底多有钱。

　　将远东的那些东西卖到国内不仅麻烦、赚得也不多，还是人民币，哪有卖给小日子来得舒服，只要将关节打通，就能躺着赚钱，还不需要担心坏账的问题。

　　“我觉得段总说得很对，相比国内，如果能打通各种关系，将远东的商品卖到小日子明显利润更高。

　　但是我们也不能忽视其难度与风险，以木材举例，首先是小日子那边的JAS标准远高于大毛，咱们能卖到的产品很难达到小日子的标准，很容易被降级或者拒收，很难进入高溢价的家具和装修市场。

　　其次对于一些针叶锯材大毛那边有出口管制，想要销售必须要挂靠大毛本地的特许经营商，将会被抽走不小的利润。

　　还有运输上也非常麻烦，远东的港口设备老旧、装货效率极低，很容易出现压港的情况，仓位也非常紧张，运费溢价是常态，还有大毛这边的商品包装差、装货野蛮，到小日子后的损耗同样不轻，并且只能自己承担。

　　最后就是大毛那边的外汇管制了，大毛那边出口的外汇必须汇入大毛指定的账户，一半要强制换成卢布，而卢布的汇率随时有可能暴跌，一不小心就容易亏损，另一半使用的话也要通过各项审批，找大毛的公司代理，又将被吃掉一大批利润。”

　　秦明从专业的角度一点点分析了将远东的商品卖到小日子的难度，这还只是明面上的，还有远东本地的黑帮、政府、海关这些势力的影响没有说。

　　“这也是虽然也有一些港岛公司做这方面生意，但是规模都不大的原因，虽然利润很不错，但难度也非常高，堪称九死一生。”

　　问题还是在大毛这边，听完秦明的分析，林远有些头疼，现在的大毛真的太乱了，不仅黑，还不讲规矩。

　　特别是大毛的外汇管制，必须将一半的外汇换成卢布，摆明了就是在敲竹杠，这年头卢布狗都不用，连大毛本地的木材和钢厂企业都不收。

　　完全就是一堆烫手山芋，除了用来给雇佣的大毛工人支付工资之外，能用到卢布的地方很少，最关键是这玩意儿还一直在贬值，必须得想办法尽快花出去。

　　“虽然确实有一些困难，但是这不意味着我们就要为此放弃这条渠道，有困难就去战胜困难，正是因为别人做的少，所以才是一片蓝海，只要我们真的能做起来，无论是利润还是声望，致远都能有非常可观的收获。

　　甚至哪怕利润低一些也无妨，咱们的外汇来源还是太单一了，远丰的工厂还有致远各个合作伙伴的工厂想要升级生产线离不开小日子的机器，这些都需要外汇。

　　边总，秦总，你们放开手去做就好，哪怕是失败了也不过是损失一些利润而已，咱们致远背靠远丰、万家负担的起这些损失。

　　另外，远丰这边也可以给你们提供一些支持，远丰的饲料需要大量的鱼粉，远东的渔业资源非常丰富，质量也很好，你们可以将卢布换成鱼粉卖给远丰，或者自己开办鱼粉加工厂加工鱼粉，远丰这边无限量收购。”

　　林远语气坚定地说道，在这个年代，能够换取外汇的机会无论如何也不能错过，必须要尝试一下，成功了最好，哪怕失败了，也能承担。

　　说起来也幸好远东这边除了木材和钢厂之外还有鱼粉这个特产，因为其成本主要是来自当地，所以需要大量的卢布支付工资与购买原材料，也不是那种完全的出口型商品，依然能够用卢布买到，不然这个生意就更难做了。

　　“除了这个，远丰的远东分公司这边也可以用人民币兑换一部分致远手上的卢布。”

　　林远想了想又补充了一句。

　　柴油、人工，包括地租明面上都是要用卢布结算，远东分公司这边能用到卢布的地方还是很多的。

　　远丰的外汇还是很珍贵的，用人民币兑换卢布也能减少一批外汇的开销。

　　“谢谢林总的支持！”

　　随着林远连续说出了两项对致远贸易的扶持方式，边勇和秦明脸上的表情都舒展了不少，对于致远的前途也更加有信心。

　　“我扶持致远，也是希望未来致远能够为远丰提供支持，而且在这个过程中远丰也并没有损失什么。”

　　林远摇了摇头，继续说道：

　　“前期的话，致远还是和其他边贸公司一样开展双边业务，等站稳脚跟之后再尝试开拓小日子和棒子的市场开展多边贸易。”

　　“是，林总！”

　　确定了发展方向，接下来致远的发展林远便将其放手交给了边勇和秦明，林远只是坐镇一段时间，查看边勇和秦明上传过来的各种资料。

　　过了将近一周时间，林远见致远贸易运转顺利，便和高芳芳返回了曹州。

　　下一次再来致远贸易这边，估计最快也是致远卖完第一批商品，林远还查看致远经营情况的时候了。

　　甚至更关键的节点，比如致远贸易正式开始进行三角贸易的时候，林远再亲自过来看看。

　　（从龙国购买轻工业品运到远东，然后换成原材料卖到小日子，再从小日子购买机械设备运到龙国）

　　接下来一段时间，林远的主要精力又将重新放回远丰身上。

　　刚回到养猪场，林远便看到一辆辆载满肥猪的拖拉机或者货车从养猪场驶出。
