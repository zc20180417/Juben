## Original Excerpt
第69 章 再次组局，致远贸易

而林远听完边勇的话反而松了一口气，建远贸易的性质可不是那些倒爷，反而更像那些边贸公司，甚至模式还要更加先进。

　　如今这些小倒爷退出，对林远来说反而还是一个好事儿，提前帮自己清除了一大批竞争对手，创造了一片新的蓝海市场。

　　“边哥，我们建远也打算做边贸生意，你有没有兴趣加入？”

　　林远脸上露出微笑，向边勇发出了邀请。

　　边勇先是一愣，随后皱眉思考了起来。

　　建远确实是有这个实力，如果真的想做这方面业务的话，倒是真的有可行性。

　　建远的大老板是谁，在曹州不是秘密，林建军手握万家超市这个超级公司，现在在整个鲁省商界都是排名前几的人物。

　　背靠万家这种大公司，想要搞边境贸易简直不要太容易。

　　“建远也打算进军发展远东业务吗？林总，不知道你们打算怎么解决进出口凭证和关税的事情？”

　　边勇舒展了眉头，开口问了林远一句。

　　林远嘴角勾起：“刚才我没有说清楚，不是建远，是我自己，我计划在港岛注册一家贸易公司，专门负责这方面业务，出关的话，林家和琴岛外贸公司也有合作，可以让琴岛外贸公司做代理出口。

　　至于大毛那边，就需要靠边哥你了，你应该有和那些灰关清关公司对接的门路吧。”

　　边勇眼前一亮：“小林总的计划很周全，我同意加入建远。”

　　“你不先问一下待遇吗？”

　　听到边勇这么爽快就答应了自己的要求，林远也有些意外。

　　“哈哈哈，小林总，我老家就是荷县的，对于远丰和你的名声也算是有所了解，我相信你肯定不会亏待我的。”

　　边勇大笑了一声，非常自然地给自己的新老板拍了一个马屁。

　　“不过，林总我倒是还真有一点小小的要求。”边勇有些欲言又止。

　　“尽管说，以后大家都是战友了，没有什么不能说的。”

　　“那个林总，我在北边做生意的时候也攒了一些钱，在港岛建立的这家新公司，我能不能也投一些？”

　　边勇目光热切地看着林远，刚才嘴上答应得太快，忘了提条件了，现在才突然想起来自己也可以成为合伙人的。

　　“当然没有问题，边哥你打算投多少？”

　　林远脸上露出笑容，边勇主动提投资的事情不仅没有让林远心生芥蒂，反而乐见其成，也能更放心地将新公司交给边勇，而不用担心边勇中途跑路。

　　边勇深吸了一口气，然后咬了咬牙：

　　“5万美元，这是我这些年走南闯北攒下的所有积蓄了，我愿意全部拿出来换取新公司的股份。”

　　林远打量了边勇一番，5万美元在这个年代可不是一个小数字，最重要的这五万是美元！是这个年代相当稀少的外汇！不愧是干倒爷的，出手确实不凡。

　　“新的公司我目前是计划先投资50万美元，如果没有其他人加入的话，我愿意让出15%的股份给你，多出来的5%算作边哥你凭借管理入股。

　　不过我觉得琴岛外贸公司和荷县政府应该会插一手，还有我爸那边，或许也会感兴趣，到时候可能会有一些变动，不过至少能给你10%的股份。”

　　林远思考了一会儿，缓缓开口道。

　　听到林远的解释，边勇反而更加激动：“10%的股份已经很多了，谢谢你林总，我一定会经营好新公司的。”

　　对于边勇来说，新公司插手的人越多，做大做强的可能性就越高，无论是琴岛外贸、荷县政府，还是手握万家的林建军能量都很大。

　　而管理权这个东西，包括林远在内的四位股东大概率都不会插手，公司基本上是由自己掌握，到时候他边勇或许也能成为一方大佬了。

　　“那合作愉快，边哥。”

　　林远脸上露出笑容，向边勇伸出了右手。

　　“合作愉快，林总！”

　　确定好掌舵人之后，就是拉资金、找资源，拉人入伙了。

　　因为有益农公司还有远丰系几家公司的战绩在，己方势力听说林远又要组局，都迫不及待地找上了林远。

　　就连财政现在有些紧张的荷县政府也再次咬牙抽调了一批资金出来，想要上车。

　　琴岛外贸公司更是非常清楚林远的眼光和新公司的潜力，如今有机会插手，自然也不会错过。

　　还有之前负责益农食品海外渠道的段兴业对新的公司表示了兴趣，并主动负责了新公司在港岛的注册事宜。

　　最终，经过一周时间的谈判，致远贸易有限公司正式在港岛成立，注册资本为70万美元。

　　新的公司，林远个人投资35万美元，拥有51%的股份，保持控股。

　　琴岛外贸公司投资八万美元，并且负责致远贸易的出口代理，获得15%的股份。

　　林建军投资八万美元，加上万家以及建远贸易的供应商渠道，获得14%的股份。

　　边勇投资五万美元，加上管理以及在远东的人脉，获得10%的股份。

　　荷县政府与段兴各投资3.5万美元，获得5%的股份。

　　在林远的坚持下，致远贸易虽然分出去了不少股份，但是林远和林建军两人加起来拥有65%的股份，对致远贸易的掌控程度还是很高的。

　　即使后续还有融资，想要维持住绝对控股也比较容易。

　　至于资金的话，主要是在香港想要注册公司的话，还是用美元比较方便，所以才用美元作价。

　　同时远丰的注册资金是认缴模式，实际出资，除了边勇和段兴之外，其他股东都是用的人民币。

　　毕竟致远贸易前期还是优先将国内的轻工业品卖到大毛，用人民币就足够了。

　　至于林远那35万美元，两百多万人民币是哪里来的，自然是向林建军借的，虱子多了不痒，债多了不愁，都已经借了那么多了，再借点也没啥。

　　当初林建军可是在林远的建议下成立的万家，甚至万家现在的开业活动还是用的林远最开始设计的那一套。

　　可以说万家能发展到这一步，林远绝对是有一份功劳在的，借点儿钱又怎么了？又不是不还，反正以万家现在的开店速度，账上的钱根本用不完，还不如借给林远呢。
