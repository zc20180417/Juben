## Original Excerpt
第74 章 见梁永辉，期货市场的隐患

“不过这种现代购物中心的投资很大，凭借万家的盈利能力能支撑得住吗？”

　　林远将乱七八糟的想法丢在一边，又开口提醒了一句。

　　“放心吧，现在万家超市每年的营业额都是10亿级别的，是琴岛甚至整个鲁省各个银行的优质合作对象，咱们如果要建一个天河城那样的购物中心，各个银行肯定会支持的。

　　名字我都想好了，就叫万象城，全名万象城广场，寓意包罗万象，全面对标羊城的天河城广场。”

　　林建军有些兴奋地说道。

　　而林远嘴角则再次抽了一下，如果他没有记错的话万象城好像也是未来润华集团的，这是逮着一个往死里薅呀，哪怕你叫万达广场也好呀。

　　林远暗自摇了摇头，没有打断林建军的兴致，毕竟万象城现在确实还没有呢，这个名字也确实起的很好，林远没必要阻止。

　　同时，林远心里也在考虑要不要告诉林建军万达那种城市商业综合体模式，想了想还是放弃了。

　　自己也不懂这方面，国内也没有这种模式，万一画蛇添足就糟糕了。

　　现在的话不如先照搬天河城的优秀经验，把购物中心给做起来再说，等以后经验丰富了再考虑万达那种模式。

　　“林总，老林总，你们这个万象城项目现在缺负责人吗？你们看我怎么样？”

　　见林远和林建军聊得差不多了，坐在副驾驶上的高芳芳忍不住回头毛遂自荐。

　　林远看着高芳芳，脸上带着无奈地笑容。

　　“你倒是这敢说，这可是涉及到十几个亿的大项目，你才工作多久，就敢负责这种项目？

　　别说是你了，就算是我，也得找专业的经理人负责这个项目，你现在还是先抓紧时间好好学习吧。

　　你放心，总有外放的时候。”

　　林远对于高芳芳的“野心”，一清二楚，有些无奈地说道。

　　“哈哈哈，林总，我就是开个玩笑，这点儿自知之明我还是有的，您放心，我一定努力学习，迅速成长起来，成为一个能独当一面的负责人。”

　　高芳芳也意识到自己有些膨胀了，连忙开口解释道。

　　“希望如此。”

　　林远用饱含深意的眼神看了高芳芳一眼。

　　很快，车抵达了酒店。

　　在高芳芳前去办理业务的时候，林建军拉住了林远。

　　“儿子，你老实交代，你和你这个小秘书是什么情况？”

　　林建军将脑袋凑到林远旁边，小声说道。

　　“爸，你想多了，我们就是正常的上下级关系，高芳芳眼里只有事业，而我心里也已经有别人了，所以你不用瞎猜了。”

　　林远叹了口气，有些无奈地解释道。

　　“你说什么？”林建军声音突然拔高了一截，注意到不对，才再次放低声音：

　　“你说你已经有喜欢的人了，到底是谁？我认识吗？”

　　“你不认识，不过以后你会见到的。”

　　涉及到前世，林远并不想说太多，打发了林建军，便朝着酒店走去，明天还得面试期货部的负责人呢。

　　次日，林远终于见到了自己期待已久的期货部负责人。

　　“梁先生是吧，请坐。”

　　林远手里拿着梁永辉的简历，示意梁永辉在自己面前坐下。

　　“梁先生，我看你简历上写你之前是在摩根士丹利做Quant（衍生品量化分析师），据我了解这应该是一个非常高薪的工作。

　　我想了解一下你为什么要从摩根士丹利离职回国？并且加入远丰，你应该知道远丰并不是专业的证券公司，以你的资历完全可以选择更好的岗位。”

　　等梁永辉坐好后，林远率先提出了自己的疑问。

　　“林总，您有所不知，做Quant确实收入很高，但其本质仍是基础的量化岗，在华尔街，这就是大多数华人的极限了，凭借华人的身份，很难做到管理层。

　　我在华尔街这些年钱也赚了不少，同时也没有结婚，仍然保留着龙国国籍，自然就想回来施展包袱。”

　　梁永辉说到这儿，稍微停顿了一下。

　　“至于为什么加入远丰，我原本是想凭借我之前攒下的钱先做一段时间的散户的，只是国内的期货市场...”

　　梁永辉没有继续说，但林远已经明白他的意思了。

　　“至于国内其他的期货公司，凭借我的资历虽然确实能获得不错的待遇，但也仅仅是不错而已，想要成为真正的管理岗负责独立的业务，而不是做一个量化分析师，难度依然存在。

　　因此，现在的远丰反而是我最好的选择，让我可以独自负责业务。”

　　听着梁永辉的解释，林远微微皱眉。

　　“梁先生，你可能不清楚，远丰并不是一家期货公司，我们做玉米和大豆的期货主要是为了套保（期货对冲），用来控制成本。”

　　“林总，您确定远丰只是为了做套保？2000万可不是一笔小数字。”

　　林远脸上闪过一丝意外：“你怎么知道的？”

　　梁永辉没有正面回答林远的问题：

　　“我自然有我的消息来源，只是我想提醒林总，龙国的期货市场可没有林总想的那么简单，33%的仓位还是有些高。

　　现在庄家和主力没有动手，只是因为收益还不够高，等林总买了足够的期货，并且价格也上涨了一部分之后，就是他们动手的时候。”

　　林远听着梁永辉的解释脸色有些难看，但也不得不承认梁永辉说的有道理。

　　虽然前世应该并没有出现玉米和大豆期货短时间内下降30%的情况，但是这一世有了林远入场，为了林远这个肥羊，那些机构完全有可能搞出一些前世没有发生过的事情。

　　“那梁先生，如果你担任远丰期货部部长的话，会怎么做？”
