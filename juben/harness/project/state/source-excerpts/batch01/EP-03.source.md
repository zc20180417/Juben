## Original Excerpt
第3章 扩大规模

“我就知道你盯上这笔钱了！”

　　林建军听到自家儿子的话，脸上也露出了笑容。

　　“建一个砖厂，买地、建窑、买设备这些差不多要五万，再加上一万块的备用金就够了，咱家的家底也建不了第二个，多的钱就都给你养猪。

　　今天跟老刘的这笔生意，我感觉你昨天说的还真是对的，这猪价估计真的要暴涨了，扩张养猪场确实是一个好主意。

　　这四万我分你一万，再加上家里的两万存款，三万块，够你大干一场了！”

　　听到林建国的话，林远的嘴角再也控制不住。

　　“放心吧，爸，我一定会把咱们家的养猪场发扬光大！”

　　林远一边说着，心里也开始规划起这三万块的用途。

　　自家的养猪场目前养10头母猪和100头育肥猪就是极限了，想要扩大规模，肯定得扩建养猪场，或者收购别人的养猪场。

　　我记得前世王老板除了自家的养猪场之外，还收购了隔壁村老赵家的养猪场，老赵家养猪场规模虽然比不上自家，不过也有50头的规模。

　　不过他家现在好像没有母猪，对林远扩张规模没有太大的作用，毕竟建一个能容纳一两百头猪的养猪场，加班加点也就需要一个多月而已。

　　一个五十头猪的小养猪场不值得林远特意并购，还会增加林远管理养猪场的难度。

　　当天晚上，林远便开始在书桌前详细规划了起来。

　　“按照现在的物价，一块砖要八分钱，水泥一袋26块钱，大工18块一天，小工10块一天，再加上屋顶、硬化这些，一平米猪舍造价最少也得60块。

　　要是建一个能容纳200头育肥猪的砖石猪舍，差不多要500平，光猪舍就能把3万块花完了了，还有供水、排粪系统，想要规模化养殖，这些肯定也不能放松。

　　还有以后肯定要扩大母猪的规模，母猪猪舍的成本还要更高一些。

　　这样一算，建一个养猪场的成本一点儿都不低，三万块想要大幅度扩大养猪场的规模，难度有点儿大啊。”

　　林远一边在纸上写着一串串数字，一边自言自语地说着。

　　“现在看似家底丰厚，但底子还是有点儿薄，想要通过自建猪舍扩大规模不太现实，还是得租别人的养猪场才行。

　　刚好这几年猪价一直比较低迷，应该能找到不少荒废的养猪场，可以租下来，扩大规模。

　　至于饲料这些，一头猪到出栏，即使搭配麸皮、猪草这些，也得吃至少300斤玉米，现在养猪场还有一万斤玉米，想要养300头猪，光玉米就得九万斤，还差八万斤，也不知道能不能赊欠。”

　　林远忍不住抓了抓头，越算越头大。

　　这时林远的母亲端着水果走了进来。

　　“小远，在为养猪场的事情发愁呢？别想了，先吃点儿水果。”

　　“没事儿，我能搞定，你回去休息吧。”

　　郑淑华将果盘递给了林远，林远随手拿起了一个苹果啃了起来。

　　林远看着如今容光焕发，一点儿都不显老的母亲，再想想未来父亲死后那张憔悴的面孔，脸上不由地露出微笑。

　　还好自己重生了，并且重生回到了一切都没有发生的时候，现在无疑是这个家最幸福的时候。

　　父亲是养殖大户，家境殷实，自己也考上了中专，前途光明，母亲整天也没有什么值得发愁的事情，每天脸上都带着笑容。

　　林远攥住了拳头，这一世一定要守护好眼前的这份美好。

　　“小远，你爸都和我说了，你有扩张养猪场的想法，我们都支持你，我虽然帮不了你什么，但是我可以找你舅舅帮忙。

　　他人脉广，附近有啥荒废的养猪场他都清楚，还有猪吃的饲料，我们刚开始养猪的时候还是靠你舅的面子赊下来的。”

　　郑淑华的话再次让林远陷入回忆之中，自己的舅舅郑国华是隔壁村的村支书，在附近确实称得上一句人脉广。

　　无论是前世还是这一世，舅舅对自家都非常照顾。

　　林建军能从信用社借到三万块钱，除了林建军本身的能力之外，也少不了郑国华的牵线搭桥。

　　“妈...”

　　林远欲言又止，郑淑华没有继续说什么，放下了果盘就离开了林远房间。

　　次日，林远还是和郑国华一起来到了一家已经荒废的养猪场外。

　　“舅舅，这次又要麻烦你了！”

　　“没事儿，你第一次创业，作为长辈怎么也要支持一下，顺手的事情而已。”

　　郑国华摆了摆手，表示亲戚之间不用这么客气。

　　“老郑，你过来了。”

　　一个中年男人从猪舍里面走了出来。

　　“我刚才看了一遍，虽然这个猪舍已经有一年多没用了，但是质量和环境上绝对都没有问题，稍微打扫一下就能继续使用。

　　这个就是你外甥吧，果然一表人才。”

　　介绍完猪舍，中年男人看向林远，脸色闪过一丝好奇。

　　“哈哈哈没错，这是我妹妹的儿子林远，小远，这是你张叔。”

　　郑国华笑着为林远介绍来人。

　　“张叔！”

　　林远笑着和张振江打了一个招呼。

　　“老张，你这猪舍租金怎么算的？小远可是我亲外甥，你可不能坑人。”

　　“我这猪舍比不上建军家的，但是也能装下50头育肥猪，还是标准的砖石结构，我也不坑你们，一年500块，绝对非常实惠了。”

　　林远和郑国华对视了一眼，郑国华点了点头。

　　“行，那就500块，这个猪舍我租下了，时间的话是三年。”

　　听到三年时间，张振江眼中闪过犹豫，不过思考片刻后还是点了点头。

　　“行，那就签三年的合同。”

　　很快，两人便在郑国华的见证下签了合同，当然说是合同，其实不过是一个手写的说明，证明有过交易而已。

　　签完合同，林远和郑国华便马不停蹄地前往了第二家养猪场，现在不考虑自建猪舍，那林远原本一次性出栏三百头育肥猪的计划肯定也要变一下。

　　既然知道明年猪价一定会上涨，那么林远肯定要尽可能的扩大养殖规模。

　　“还是年轻啊，放着好好的铁饭碗不要，竟然出来养猪，郑国华和林建军还跟着这个小子胡闹。

　　毛都没长其呢，就学人家创业，等赔钱了就知道做生意的不容易了，不过也多亏这个傻小子，不然我这破厂子不知道得闲到啥时候，一年500块，虽然不多，但也相当于几亩地的收成了，还不废什么力气。”
