## Original Excerpt
第6章 配合饲料

李厂长给的价格很实惠，林远也不打算继续讨价还价，毕竟是国营养猪场，一个不小心搞成侵吞国有资产就尴尬了。

　　接下来，签合同、付款一气呵成，林远后面只需要等待国营养猪场的人把仔猪送过去就行。

　　搞定了仔猪，接下来的时间，林远则全身心的投入到了配合饲料的研究与改良之中，虽然前世在原牧集团养了几十年猪，对于各种配合饲料的配比熟记于心。

　　但终究年代不同，各种条件的差距也很大，甚至连猪的品种都不同，前世林远饲养的都是外三元，而现在则是内三元。

　　这就对林远的技术提出了一定的要求。

　　为此林远先按照根据市面上卖的饲料，复刻出了一款基础饲料，用于大部分的仔猪喂养，保证仔猪能在六个月时间涨到200斤以上，达到出栏的水平。

　　同时又根据自己的经验，优化出了四款原料和原料配比略有不同的饲料，进行对比试验，看看用哪种饲料喂的猪，长的最快，最健康，瘦肉率最高。

　　在这个年代，出栏速度和瘦肉率才是王道。

　　而在研究饲料的同时，林远还没有忘记去各个养猪场巡查情况，毕竟现在可不像未来有各种监测设备实时监测猪的情况，全靠人力。

　　而林远招收的那些员工，只有少部分有在养猪场干活的经验，但在林远眼中也基本上聊胜于无，所以还得林远亲自来盯才行。

　　同时这样也可以防止养猪场的员工不按照自己的要求，喂猪和清洁猪圈。

　　这是林远重生后养的第一批猪，也是林远压上一切的一场豪赌，再怎么谨慎都不为过。

　　“老板好！”

　　小洼村养猪场，两名员工见林远过来，连忙对林远问好。

　　林远看着穿着工作制服，戴着口罩的两人，满意地点了点头。

　　“很好，规模化养猪不同于在自家院子养猪，一定要注意好卫生问题，每次喂猪、清理猪圈一定要穿好工作服、戴好口罩。

　　不仅猪不容易生病，你们自己也能舒服一些。”

　　随后林远前往了猪圈，查看猪的生长情况。

　　小洼村养猪场是林远实验自己四款优化饲料的地方，一共五十头猪，基础版饲料和四种优化饲料各自有10头猪进行对照实验。

　　经过一个月时间的喂养，已经可以初步看出差别了。

　　经过林远优化的四种饲料，每一种饲料的长肉速度都比按照这个时代饲料复刻出来的基础款饲料要快上不少。

　　长肉最快的那一组，甚至比用基础饲料喂养的猪大了一圈。

　　按照林远估算，如果喂基础款饲料的话，需要六个半月才能长到200斤，达到出栏的标准，而长肉最快的那一组则不到六个月就能出栏。

　　最关键的是，两者的食量并没有太大差距，这快出来的半个月时间可都是赤裸裸的利润。

　　按照料肉比，也就是吃的饲料的数量和长肉的数量的比重计算的话，五款饲料的料肉比分别是2.9、2.85、2.8、2.8、2.7。

　　林远优化后效果最好的饲料，比基础款的饲料涨一斤肉，要少消耗二两饲料。

　　按照200斤出栏的标准的话，一头仔猪要涨180斤肉，用优化后的饲料能少吃36斤，这里面最少有10块钱的利润。

　　而这还是林远养猪场内部，使用了林远前世未来的一些规模化养殖控制料肉比的管理方法，本身料肉比控制的就比较好。

　　实际上这个年代土三元的料肉比普遍都在3.0以上，并且成本比起林远自己制作配合饲料成本还要更高。

　　可以说同样养一头猪，光饲料上，林远的成本就能比外面少几十块，这意味着别人亏损，林远可能稍微盈利；别人微利，林远可以爆赚；别人爆赚，林远可以血赚。

　　可以说，即使没有接下来的猪周期，林远也能凭借优秀的成本控制立于不败之地。

　　而叠加93年和94年的猪肉行情，更能迎风而起，让财富实现指数级的提升。

　　“这里的五十头猪从今天开始，全部换成5号饲料，如果一个月后依然没有问题，其他分厂也全部跟进！”

　　“是，老板！”

　　随着饲料落地，养猪场的员工们也完全进入工作状态之后，本来连续忙碌了一个多月的林远现在终于可以稍微放松一下，安心等待几个月后，第一批仔猪出栏，财富自然跃升到六位数就行。

　　但是或许每一个重生者都自带一种强烈焦虑感，让林远很难彻底放松下来，再加上这个年代娱乐资源的匮乏，让林远晚上再次在书房做起了规划。

　　“租养猪场用了三千八，饲料用了一万六，猪仔用了七千二，工人工资需要预留一万五，这样我手里能用的钱还有小一万块，得想办法利用起来。

　　从成本的角度考虑，以后肯定是要像前世原牧集团那样走自己繁殖猪仔的路线，这笔钱或许可以用来扩建母猪猪舍，多养几头母猪。”

　　以林远现在的体量肯定没有办法走国营农场那种全链条的繁衍路线，即先要有几头土猪，用土猪和洋公猪交配生产二元猪，再用二元猪和洋公猪交配生产三元猪的模式。

　　（杂交猪只能这样繁殖，不能用两头二元猪或者三元猪直接繁殖后代，不然会造成血统退化，想要稳定产出三元猪只能一步一步来，走这种全链条的繁衍路线）

　　所以林远只打算先引进一批土二元母猪，和几头杜洛克公猪作为种猪配种。

　　这样无论是资金压力还是管理难度都会小上很多。

　　等到这批猪老了，不能用了，已经是几年之后了，那时林远如果没有破产的话，养猪场应该已经建立起完善的繁衍体系了。

　　“不过不管是母猪还是公猪，价格好像都有点儿贵呀。”

　　林远一边在本子上划拉着，一边皱了皱眉头。
