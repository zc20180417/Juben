## Original Excerpt
第4章 强势护妻，当众宣告追求，苏家气疯

晚宴现场的震惊还未散去。时鸢站在台上，清冷耀眼，如同无人能及的月光，将台下脸色惨白的苏家三人，衬得像跳梁小丑。刘美兰手脚冰凉，想上前说话，却又怕被全场目光钉死，只能死死攥着苏雨柔，气得浑身发抖。苏振宏僵在原地，脸上一阵红一阵白，想上前攀关系，又想起自己之前说的那些嫌弃话，每一句都像耳光一样扇在自己脸上。苏雨柔更是眼底猩红，嫉妒得快要发疯。就在这时——人群自动分开一条道路。傅斯年迈步上前，一身黑色高定西装，气场凛冽，目光自始至终只落在台上的时鸢身上，没有分给旁人半分。他没有理会苏家，径直走到舞台边缘，伸手，动作自然而强势，将时鸢从台上轻轻扶了下来。这一幕，让全场倒吸一口冷气。傅氏总裁！海城只手遮天的男人，竟然对这位神秘设计师，如此呵护备至？时鸢微微蹙眉：“傅先生，自重。”傅斯年却攥紧她的手腕，指尖带着不容挣脱的温度，抬眼看向全场，声音低沉有力，透过安静的会场，清晰地落在每一个人耳中：“介绍一下，这位时鸢小姐，不是外人。”“她是我傅斯年认准的人，从今天起，我正式追求她。”“谁敢为难她，就是与我傅斯年为敌。”轰——一句话，炸翻全场！所有人都惊呆了。谁不知道傅斯年深情，为了失踪的苏清月守身如玉，从不近女色。如今竟然当众宣告，要追求这位顶级设计师？而最崩溃的，是苏家一家三口。刘美兰当场就忍不住了，冲上前尖声道：“傅斯年！你疯了！她是清月的妹妹，是我们苏家不要的女儿，你怎么能——”“闭嘴。”傅斯年冷冷回眸，眼神锋利如刀，吓得刘美兰瞬间噤声。他语气冰冷，毫不留情：“苏家不要的人，是我傅斯年捧在手心的宝。”“你们嫌弃她、贬低她、觉得她配不上苏家，可你们苏家，十个都配不上她一根手指头。”这话太狠，直接把苏家的脸面踩在地上碾碎。苏振宏气得脸色铁青：“傅斯年！我们是你的长辈，你怎么能这么说话！”“长辈？”傅斯年嗤笑一声，护着时鸢往身后站了半步，姿态护短到极致，“生而不养，冷眼相对，嫌亲疏养，你们也配称长辈？”“从今往后，时鸢由我护着，你们苏家，再敢对她有半句不敬，傅氏将立刻终止与苏家所有合作。”终止所有合作！苏家本就靠着傅氏扶持，一旦断了，立刻一落千丈！苏振宏踉跄一步，差点摔倒。刘美兰面如死灰，想说什么，却一句话都骂不出来。苏雨柔更是嫉妒得眼睛发红，死死盯着时鸢，恨不得冲上去撕碎她。凭什么？凭什么这个突然冒出来的女人，能成为万众敬仰的设计师，还能得到傅斯年这样的偏爱？时鸢看着身前替她挡下所有风雨的男人，眼底微动，却依旧保持着清冷。她轻轻挣开他的手，淡淡开口：“傅先生，我的事，我自己能解决。”傅斯年却不退让，目光深情又偏执：“我知道你很强，可我就是想护着你。”“清月是过去，你才是我的现在和未来。”公开偏爱！当众护短！直接和苏家撕破脸！全场目光齐刷刷落在苏家身上，有嘲讽，有看戏，有鄙夷。曾经风光无限的苏家，在这一刻，彻底沦为整个海城的笑柄。刘美兰气得眼前一黑，直接扶着苏雨柔差点晕过去。苏振宏胸口剧烈起伏，却一句话都不敢反驳。傅斯年懒得再看他们一眼，温柔地看向时鸢，语气瞬间放软：“我送你回去。”时鸢没有拒绝，也没有答应，只是淡漠地转身离开。傅斯年紧随其后，如同最忠诚的守护者。两人一走，全场哗然。苏家三人站在原地，被无数目光指指点点，颜面尽失，气得快要爆炸，却连一句反抗的话都不敢说。
