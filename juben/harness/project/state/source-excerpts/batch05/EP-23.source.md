## Original Excerpt
夜色如墨，霓虹将海城的晚高峰晕染得一片璀璨。时鸢刚结束一场私密的设计洽谈，黑色风衣衬得她身姿清瘦挺拔，侧脸线条冷白利落，一双眼淡漠得不带半分烟火气。她低头整理着手稿，指尖划过画纸上精致的剪裁线条，无人知晓，这个低调到近乎不起眼的女人，就是圈内炙手可热、从不以真面目示人的顶级设计师——鸢。更无人知晓，她脚下这座她刚刚扎根的城市，藏着她从未谋面的血亲。刚走到街角，一道强势的黑影骤然将她笼罩。男人身形高大挺拔，西装革履，周身散发着生人勿近的凛冽气场，一双深邃的眼眸死死锁在她脸上，带着近乎疯狂的震颤与失而复得的狂喜。是傅斯年。海城无人敢惹的傅氏掌权人。他伸手，一把攥住时鸢的手腕，力道大得几乎要捏碎她的骨头，声音沙哑得不成样子：“清月，你终于回来了。”时鸢眉峰微蹙，用力挣扎，冷声道：“先生，你认错人了。” 认错？傅斯年怎么可能认错。这张脸，这双眼睛，这轮廓……分明是他失踪了整整两年的妻子，苏清月。他不顾她的抗拒，直接将她打横抱起，语气偏执又温柔：“别闹了，跟我回家。”时鸢心头一冷。她不知道，自己这张与亲姐姐一模一样的脸，会成为她踏入苏家漩涡的开端。她更不知道，那个所谓的“家”里，等着她的不是久别重逢的温情，而是偏心入骨的父母，和一个早已鸠占鹊巢的养妹。而她隐藏在平凡外表下，总裁与顶级设计师的双重身份，终将在不久的将来，狠狠撕碎所有偏见与不公。

　　傅斯年的车一路驶入市中心最昂贵的别墅区。时鸢坐在副驾，手腕上还残留着男人方才攥紧的力道，她垂着眼，指尖轻轻摩挲着袖口，眼底没有半分慌乱，只有一片冷静的漠然。她从不惹事，可也从不怕事。车子停在苏家别墅门前时，时鸢才淡淡开口，声音冷而清晰：“傅先生，我最后说一次，我不是苏清月。”傅斯年侧过头，目光落在她与妻子如出一辙的眉眼上，喉结滚动，语气依旧偏执：“是不是，进去就知道了。”门被打开。苏家父母苏振宏与刘美兰一看见时鸢，脸色骤然一变。刘美兰先是愣了几秒，随即快步上前，眼神复杂地打量着她，没有久别重逢的激动，反倒带着几分审视与疏离：“你……你和清月长得真像……”站在一旁的养女苏雨柔立刻挽住刘美兰的胳膊，眼眶微红，声音柔弱得能掐出水：“爸，妈，她会不会是……姐姐当年走失的妹妹？”一句话，点醒了所有人。苏振宏眉头紧锁，上下扫过时鸢身上简约低调的穿搭，语气带着不易察觉的嫌弃：“你叫什么名字？多大了？什么时候丢的？”没有心疼，没有拥抱，只有盘问。时鸢抬眼，目光平静地掠过眼前这对所谓的亲生父母，薄唇轻启：“我叫时鸢。”没有说自己的原名，也没有主动攀亲。刘美兰立刻皱起眉，语气尖了几分：“时鸢？不是我们苏家的姓。你这些年过得很不好吧？穿得这么朴素，也没什么气质，跟雨柔比起来差远了。”苏雨柔立刻低下头，装作不好意思的样子，眼底却闪过一丝得意。她从小被苏家宠到大，享尽了千金的一切，绝不可能让这个突然冒出来的真千金抢走一切。“妈！”苏雨柔轻轻拉了拉刘美兰，“姐姐刚回来，你们别这么说……”这一劝，反倒让刘美兰更心疼养女，对时鸢越发不满：“本来就是，你看看雨柔，从小琴棋书画样样精通，举止端庄，哪像她，一身生人勿近的冷硬，一点大家闺秀的样子都没有。”时鸢站在原地，指尖微顿。心底最后一丝对亲情的期待，彻底凉透。她原本还在犹豫，要不要坦白身份，要不要认回这个家。现在看来，完全没必要。傅斯年看着这一幕，眉头第一次紧紧皱起。他忽然发现，眼前的时鸢，和他记忆里温柔软糯的苏清月截然不同。她冷静、强大、骨子里带着一股生人勿近的矜贵，绝非普通人家养出来的女孩。更重要的是——他刚才无意间瞥见，她袖口露出的腕表，是全球限量三只、价值千万的定制款。一个穿着普通、却戴得起千万腕表的女人……真的像苏家父母说的那样，过得落魄不堪吗？时鸢淡淡收回目光，转身就走：“既然你们认错了，那我告辞。”“站住！”苏振宏厉声呵斥，“我们还没做亲子鉴定，你不准走！”时鸢脚步一顿，侧过脸，逆光之下，那张清冷绝美的脸上，勾起一抹极淡、极冷的笑。“可以鉴定。”“但我丑话说在前头——就算我是你们的女儿，我也没打算，认这个家。”话音落下，全场死寂。苏家父母脸色铁青，苏雨柔眼底慌了一瞬。傅斯年却看着她的背影，心脏猛地一跳。他忽然意识到，自己好像……真的认错了人。可他却不想放手了。这个叫时鸢的女人，远比他想象的，要神秘、要强、要耀眼得多。而她隐藏的双重身份，很快就会让这群看不起她的人，狠狠打脸，悔不当初。
