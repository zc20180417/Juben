## Original Excerpt
第49 章 远丰产业园

“那是当然，高镇长，我对饲料厂可是寄予厚望，我还指望它给养猪场的扩张提供弹药呢。

　　不过想要真正把它建起来，还是需要领导们的支持才行。”

　　林远看着高伟良，脸上也露出了笑容。

　　“好说，林远你有什么需要我提供帮助的，尽管提就行，我们镇政府绝对全力支持你。

　　昨天你在电话里提到的土地的事情，你现在有什么想法？”

　　高伟良喝了一口茶，目光看向林远和李星和。

　　林远脸上露出微笑，然后打开了手里的文件夹，推到了高伟良面前。

　　“高镇长，这是我们设计的饲料厂的建设方案，一期的话，我们计划投资200万，建立一个年产一万吨的中型饲料厂。

　　位置则选在白水镇西边靠近国道的这块荒地，主要面向白水镇进行招工，一期的话只需要10亩地。

　　不过未来随着饲料厂持续盈利，肯定会继续扩张的，这座饲料厂按照我们的设计，其最终的年产量为10万吨，连带配套设施，大概需要100亩地。”

　　高伟良微微皱眉，脸上闪过一丝纠结，对于林远能建这么大的饲料厂心里肯定是非常乐见其成的。

　　只是林远这一要就要上百亩地，然后慢慢扩建的作风也让他非常头疼。

　　某种程度上来说，这是有点儿擦边的。

　　“对于你的计划，我原则上是支持的，不过还是需要和县里谈一谈。”

　　高伟良说到一半，看了一下林远的反应，停顿了一下才继续开口道。

　　“你也不用担心，只是饲料厂的投资太大，光镇里做不了主，同时配套的水电这些也需要县里支持才行。”

　　按照现在饲料的价格，一吨饲料的价格最少也有1000块，要是饲料厂彻底建成之后，一年的营收就有一个亿。

　　而荷县的GDP现在也才刚突破10亿，一栋饲料厂的GDP就差不多占荷县的十分之一了，这肯定不能由高伟良自己做决定。

　　而且镇上的能力也有限，找县里也能帮林远要更多的帮助。

　　“嗯嗯，我理解，高镇长，那就麻烦您了。”

　　林远点了点头，恭敬地说道。

　　回到远丰之后，林远便将饲料厂的筹备工作交给了李星和，毕竟李星和才是专业的，对于饲料厂需要什么更加清楚。

　　另一边，高伟良将林远计划投资建饲料厂的消息带到县里之后，果然也引起了巨大的轰动。

　　县委书记、县长都被惊动了。

　　“不就是100亩地吗？给他批！还有三通一平，也都要做好了，咱们要让人看到咱们县里的诚意。”

　　会议室里，荷县的县委书记王云峰大手一挥便同意了林远的土地申请。

　　“地的事情都好说，以远丰的实力和饲料水平，想要将饲料厂的产能扩建到10万吨问题不大。

　　只是白水镇的这块地虽然在国道旁边，但是位置还是差了一点，城关镇这边的产业园现在也需要企业入驻，里面的各种配套都比白水镇那里好多了。”

　　这时城关镇的书记扶了一下眼镜，提出了另一个问题。

　　王健康和高伟良听到这话后，表情变得有些不对劲。

　　“张书记，你这话说得就不对了，选择建在哪里是林远的自由，而且城关镇这边可不能直接给林远批100亩地，毕竟林远第一期的工程占地也只有10亩地。

　　要是落在真落在城关镇的产业园，那可不利于远丰未来的扩张，要是后续远丰打算扩张却发现没有合适的地之后，谁知道林远会不会像养猪场那样换个县建一个新的饲料厂。

　　而白水镇这边就不会有这种问题，别说10万吨产能，就算林远未来打算将饲料厂产能扩张到100万吨，也有足够的地。

　　另外，我觉得产业园这种东西，不止城关镇可以搞，白水镇同样可以建，我们白水镇的优秀企业也是有很多的。”

　　高伟良站起身，向着周围的人侃侃而谈。

　　“没错，我们也要考虑一下企业家本身的想法嘛，林远毕竟是白水镇的人，想要造福家乡的心情可以理解。

　　而且我觉得伟良说的也很有道理，我们完全可以直接和林远对接，在白水镇建立一个远丰产业园，未来说不定还能成为咱们荷县新的发展引擎呢。”

　　方雨桐这时也跟着开口附和道。

　　“但是...”这时又有人想开口。

　　“好了，咱们开这个会就是讨论县里要不要给远丰批地，还有给远丰什么样的支持，让远丰搬家的想法就不要有了。”

　　王云峰开口打断了众人的争论。

　　“关于方县长说的产业园的事情，我觉得可以好好考虑一下，咱们荷县还有很多潜力可以挖掘的。

　　白水镇虽然现在配套差一点，但是无论是交通条件还是地理位置都非常好，四通八达，去曹州还有泉城、彭城都非常方便，完全具有建立产业园的条件。

　　另外，这段时间荷县，特别是白水镇的养殖、果树种植、蔬菜种植都发展得很好，但是我们也不能只满足于提供初级产品上，也可以试着搞一下食品加工嘛。

　　这个产业园就可以向着这个方向发展，由咱们县里和远丰牵头，然后联系荷县的企业家们一起搞一下。”

　　见王云峰这个书记都开口了，其他人自然也不会再提出反对，众人开始围绕产业园的事情讨论了起来。

　　一个产业园的建设方案渐渐出炉。

　　......

　　“高镇长，你说县里要在白水镇建立一个产业园吗，还要让远丰冠名？”

　　办公室里，当林远收到高伟良的消息时，整个人都被震惊到了，嘴里喝的水直接喷了出来。

　　“哈哈，惊喜吧，林远，这可是县里考虑到远丰对荷县经济的重大贡献特批的，你们远丰可得好好出出力啊！”

　　高伟良自己也没有想到自己只不过是突发奇想，没想到真的成了，这下林远可不能就建一个10万吨产能的饲料厂了。

　　按照产业园第一期500亩的面积，林远怎么也得解决一半吧。

　　“我知道了，高镇长，我会努力的！”

　　林远挂断了电话，瘫倒在了老板椅上，这下是真的出了一个大风头了，不过，经过这一次的事情，远丰的名气也能更上一层楼了。

　　就是500亩的产业园，以远丰的体量，该怎么去把他给填满。
