## Original Excerpt
第45 章 偶遇海洋大学教授

本来这么重要的事情最好刘玉芳也跟着一起来的，但是远丰现在确实没有能够独当一面的人才，刘玉芳必须得留下坐镇。

　　火车上，林远看着车窗外的飞雪，心里难得宁静了下来。

　　“小伙子，你是要去哪儿呀？”

　　林远旁边一个穿着中山装，戴着眼镜的老人，笑着和林远打了声招呼。

　　“琴岛？您呢？”

　　“巧了不是，我也是去琴岛，看你这样子，你是去琴岛旅游？”

　　“算是吧！”

　　林远面带微笑，没有说自己是去招人的。

　　“琴岛可是好地方啊，虽然现在是冬天，但是风光也非常不错，栈桥、五四广场、天主教堂、崂山这些都很不错，还可以泡泡海水温泉，现在是旅游淡季，人也少，更适合好好欣赏美景...”

　　听到林远的目的，韩文峰聊天的兴趣也提了上来，滔滔不绝地讲了起来。

　　“您是琴岛本地人？”

　　林远听着韩文峰讲述青岛的各种景点，也没有露出不耐烦的神色，毕竟这个年代坐火车基本没啥能干的，和人聊天还能缓解一下无聊。

　　“哈哈，算是半个吧，我老家是曹州的，现在在青岛教书，差不多有十年了？”

　　韩文峰笑了笑，然后端起水杯喝了一口，说了太久都有点儿口渴了。

　　“看您这气质，你是在大学当老师吗？”

　　林远眼睛一亮，没想到自己坐个火车都能碰到大学老师。

　　“没错，我就在海洋大学教书，你也是大学生？哪个大学的？”

　　韩文峰也笑着说道，自从林远坐下后，他就感觉林远气质非常不错，也是一个文化人。

　　林远摇了摇头：“不是，我是中专生，学的畜牧，去年刚毕业，现在开了一家养猪场。”

　　“哦？没想到你这么年轻就是老板了，看你是在曹州上的车，你知道咱们曹州最近可出了一个不得了的年轻人。

　　也是中专毕业，开了一家养猪场来过完年应该就是咱们鲁省最大的养猪场了，说起来，他的年纪”

　　说到这，韩文峰表情古怪地看了一眼林远。

　　“你该不会就是远丰的老板林远吧。”

　　“哈哈，没错，我就是林远，这次去琴岛也不是去旅游的，而是去见几个人。”

　　林远笑着承认了自己的身份，他觉得韩文峰如果真的是海洋大学的老师，那绝对有结交的价值。

　　“你是去琴岛招人的吧？”

　　韩文峰眼珠子一转，像是想到了什么。

　　“您怎么知道？”

　　“你要见的人是不是有一个姓韩，应聘的是你们技术中心的负责人？”

　　“老人家，您贵姓？”

　　“免贵姓韩！”

　　林远瞪大了眼睛：“难道您是？”

　　“哈哈，没错，巧了不是，韩泽成就是我儿子，研究畜牧的，刚从国外回来，本来是打算去农大当老师的，没想到你们的人找到了他，就打算去看看。”

　　韩文峰脸上也露出了笑容，笑着解释道。

　　“这真是太巧了，韩教授，看您水快喝完了，要不要我去给您接点儿？”

　　林远脸上带着讨好的表情，伸手就要去拿韩文峰的水杯。

　　韩文峰连忙握紧了水杯：“还有一半呢，不需要加水，再说了你一个大老板给我接水也不合适呀。

　　另外，我儿子那边我可说不上话，能不能让他进远丰，还得看你自己才行。”

　　“瞧您这话说的，尊老爱幼是咱们的传统美德，帮您倒杯水怎么了。不过咱们这么有缘，就算您不帮我，给我说说韩博士的想法总没问题吧。”

　　“我哪儿知道他的想法，他要去你们远丰的事儿，还是我媳妇儿给我说的。”

　　“这样啊。”

　　看来韩文峰和韩泽成的关系可能并没有像自己和林建军那样和谐，林远心里有些失望，不过很快便调整回来，总归是有收获的。

　　对于韩泽成这个高学历的专业人才，林远还是很看重的，技术中心太需要一个有足够实力的负责人了。

　　随后林远便和韩文峰聊了起来，打听起韩泽成的各种信息，以增加自己招揽的到韩泽成的可能性。

　　“对了，韩教授，还没有问过你是研究什么的？”

　　突然林远看向了韩文峰，如果是子承父业的话，那韩文峰应该也是研究畜牧的，并且实力还要更强，毕竟不是什么人都能在985高校当教授的。

　　“我是搞水产的，和那个逆子研究的东西不是一个方向的。说起来我现在都有些羡慕那群搞养猪的家伙了，我们可没有远丰这样的企业愿意给我们提供资金。”

　　韩文峰先听林远突然提到自己，先是愣了一下，随后才调侃道。

　　“都是搞养殖的，说不定什么时候我们远丰就进军水产行业了，到时候我找您合作，您可不能拒绝。”

　　“怎么会？你不是我们这个圈子的不知道，现在啥项目不需要经费，我听说你们远丰公布要搞校企联合研发的消息时，农大那群养猪的都乐疯了，就等着你们远丰上门了。

　　我那个逆子愿意去远丰，很大程度上也是因为各个高校和研究所的资金不足。”

　　说到这儿，韩文峰叹了口气，语气都低沉了不少。

　　“放心吧，韩教授，咱们国家发展很快，总会好起来的。”

　　注意到韩文峰情绪不对，林远连忙安慰道。

　　不过单从远丰的未来看，形式要比自己林远想的还要好不少，不管是招人还是搞校企联合研发，都会轻松不少。

　　可惜现在远丰的钱大部分都压在扩建养猪场上了，能拿出来的钱有限，包括未来，主要的资金也是要用于扩建养猪场，这也是重资产模式的无奈。

　　即使想要猛砸研发，也得优先扩张规模才行。

　　“没错，困难只是暂时的，未来一定会好起来的。”

　　韩文峰也是很快便调整了过来，振奋地说道。

　　“说起来，小林，对于校企联合研发这件事儿，你是怎么想的？除了个别老古董，大部分老师对这个模式可都是很好奇的。”

　　“这个模式，我也是在摸索中，暂时也没有形成完整的方案，不过底层的就是我们出钱，高校负责研发的那一套东西，需要考虑的只是成果如何分配，这个就需要之后再仔细研究了。”

　　林远思索了片刻，给出了一个非常粗略的答案。

　　“可惜海洋大学都是研究水产的，估计短时间是没有机会从远丰这里拿到钱了。”

　　韩文峰突然叹了一口气，脸上带着可惜。
