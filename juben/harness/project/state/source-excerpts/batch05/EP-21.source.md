## Original Excerpt
第41 章 年终总结（上）

虽然现在运输条件不太好，从远东将粮食运回来成本比较高，但是种植大豆这种经济价值比较高的农作物，成本还是比国内低一些的。

　　不过这个需要等养猪场这边更稳定，林远的资金更雄厚之后再说，以现在林远的实力，虽然也勉强能玩儿，但是考虑到其牵扯到的精力和产出，性价比并不高。

　　当然现在是十一月份儿，玉米和大豆都是春天种植的，现在就算有这个计划，也得等到明年才能执行。

　　现在的话，就只能先继续忍受暴涨的饲料价格了。

　　了解完饲料的情况，并做好预案之后，林远又开始继续他之前做到一半的事情，即确定新的养猪场的地址。

　　至于食品厂那边，林远暂时不打算进入这个旋涡之中，让工人自己推选出几个管理层，组织生产，并让万家帮忙售卖之后，林远便暂时将其放在一边，让其自行运转。

　　又过了半个月时间，林远和王立军几乎跑遍了大半个曹州，终于确定了新的养猪场的地址。

　　新的养猪场选在了磐县西部的乐庄，除了各方面条件都非常优越之外，最大的优势就是磐县政府的支持。

　　磐县这边以相当优惠的价格，一口气批了500亩地，后续林远打算继续扩张的话，也预留了非常多的空间

　　县里的几家银行，还一起给林远批了20万的低息贷款，不得不说经济大县的实力就是不一般。

　　随着合同签订，林远也没有浪费时间，直接联系新的施工队进场施工。

　　相比大洼村的养猪场，磐县的这家养猪场对土地的利用效率还要更高一点，相比之前那种一栋或者说一间猪舍只有100头的模式。

　　新的养猪场每一栋猪舍都能容纳1000头猪，已经有现代化养猪场的影子了，是林远根据后世的记忆设计出来，然后请琴岛的设计公司设计的图纸。

　　预计这五百亩地，就能建成一个配套完善的，存栏量能达到十万头的大型养猪场。

　　不过这种养猪场在方便管理的同时，成本相比之前那种模式也高了很多，再加上配套的污水处理系统，林远的120万也只能建造两座。

　　好在接下来养猪的利润很不错，并且政府也有免税政策，预计一年时间就能回本并且小赚一笔，要是继续扩张还能进一步摊平建造成本，加快回本速度。

　　接下来一段时间，林远便开始在荷县和磐县两头跑，虽然两边的工作都不忙，但还是需要林远在场，并且时不时签一份文件。

　　大洼村这边还好，有刘玉芳在，并且养猪场现在也已经基本形成了自己的，不需要林远操心太多。

　　磐县那边是真的需要林远全程跟进，才能保证建筑公司能够完全按照自己的标准和要求将养猪场落地，实在是没有合适的人能够代替林远坐镇。

　　就这样，时间缓缓来到了1993年12月。

　　随着新一批超市顺利开业，在外面奔波了几个月的林建军终于又回到了林家。

　　两人闲聊中开始为1993年做起了年终总结。

　　相比积累期的1992年，1993年，林家无疑是取得了一个跨越式的发展。

　　林远的养猪场存栏量从500头，扩张到了5000头，到1994年就能突破到两万两千头。

　　按照现在的猪价，每年的营收能达到三千万，已经接近去年白水镇的GDP了。

　　饲料厂和发酵厂作为养猪场的配套，暂时忽略不计。

　　零食厂的话，有万家的渠道在，已经逐渐扭亏为赢，并且随着万家的扩张营收不断增长。

　　同时林远也已经请了设计公司在重新设计产品的包装，等新包装上市后，销量预计还能有一个爆发式增长，一年千万营收应该没有太大问题。

　　而林建军在林远的帮助下，事业发展也是非常顺利。

　　首先是最开始的建军砖厂，现在应该叫建远建材了，林远和林建军各占比50%。

　　虽然因为年中时的调控政策萧条了一段时间，但是后面凭借林远养猪场的订单，很快盘活，并在原有砖厂的基础上，低价收购了一个小型水泥厂。

　　两者加起来，每年的营收达到了200万的水平。

　　未来随着林远的养猪场继续扩张，对建筑材料需求增加，可能还会继续扩张。

　　此外林远还在筹备成立建远建筑作为自家专用的施工单位，只不过现在一方面资金有些紧张，另一方面养猪场和超市都追求快速扩张，暂时被搁置了下来。

　　接着就是林远和林建军合资的建远贸易还有建远物流，目前建远贸易仍然只有曹州市一个点位。

　　不过上个月林远担心两人的疯狂扩张让资金链断裂，为了让自己的现金流更加健康，又抽空跑了一趟羊城，谈了一些电器的订单，康庄的店铺也增加了几家。

　　服装、鞋子、家电还有电子产品这些品类加起来，每个月的营收加起来有四百万，利润更是能达到200万，让林远和林建军可以动用的现金流更加充沛，为两人的扩张提供了充沛的弹药。

　　而建远物流作为建远贸易的配套公司，也随之扩张了几倍，目前拥有10辆6米货车往返于羊城和曹州。

　　最后就是万家超市了，最初的荷县四家店铺，在第一个月创造了300万的营收后，后续每个月的营收也维持在了150万，每月利润大概在30万左右。

　　随后林建军又在曹州市区开了一家大型超市，在皲县、梁县、磐县三个县，按照荷县的模式各开了一中三小四家超市。

　　所有超市都在12月初正式开始营业，虽然没有节假日的加持，但凭借着强力的营销活动，还是创造了恐怖的营收。

　　开业第一天，曹州市的万家超市因为面积达到了2000平，并且曹州的市民消费能力也更强，营业额直接突破了100万，达到了120万，引起了巨大的轰动。
