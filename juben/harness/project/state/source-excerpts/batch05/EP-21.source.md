## Original Excerpt
第16章 佳偶成双

傅斯年将时鸢紧紧拥在怀中，戒指的光芒与两人眼底的温柔交织，整个会议室都被甜蜜填满。苏父苏母笑得合不拢嘴，连连点头，对这个女婿满意到了极致。苏清月站在一旁，看着妹妹终于得偿所愿，眉眼温柔，轻轻拭了拭眼角。可就在这时，大厅外忽然传来一阵沉稳而有力的脚步声。一道颀长挺拔的身影，逆光出现在门口。男人穿着一身深灰色高定西装，气质温润却自带强大气场，眉眼清俊，目光穿过人群，直直落在苏清月身上，瞬间便柔成了一汪春水。他没有看任何人，只望着苏清月，声音低沉又宠溺，带着跨越山海的思念：“月月，我来接你回家。”苏清月的丈夫——陆知衍，回来了。全场瞬间安静。时鸢愣住，抬头看向姐姐。苏清月原本平静温婉的脸上，骤然染上一层浅浅的红晕，一向冷静强势的她，此刻竟露出了小女儿般的娇羞与慌乱。陆知衍大步走来，无视所有人的目光，径直走到苏清月面前，伸手便将她轻轻拥入怀中，动作自然又亲密。“处理完那边的事，立刻飞回来了。”他低头，鼻尖蹭了蹭她的发顶，语气带着心疼，“让你一个人面对这么多事，辛苦了。”苏清月轻轻推了推他，耳根微红：“这么多人看着呢……”“看着更好。”陆知衍低笑，声音不大，却清晰有力，“我要所有人知道，你苏清月，不是孤身一人，你有我。”这一刻，时鸢才恍然大悟。原来姐姐在国外这些年，并非独自漂泊，她早已拥有了属于自己的幸福与依靠。难怪她回国后气场沉稳、行事果决，因为她的身后，一直站着这样一位深情又强大的丈夫。陆知衍这才松开妻子，礼貌地看向苏父苏母，微微躬身，礼数周全：“爸妈，抱歉现在才正式见面。”随后他看向时鸢与傅斯年，伸出手，笑容温和：“我是陆知衍，清月的丈夫。以后小鸢和清月，我一起护着。”傅斯年伸手与他相握，两大强势男人对视一眼，瞬间达成默契。“以后，两家便是一家。”苏父苏母此刻惊喜交加，眼泪再次落下——不仅两个女儿平安归来、恩怨了结，大女儿还带回来如此优秀可靠的丈夫，小女儿也被傅斯年真心呵护、求婚成功。一家团圆，佳偶成双。苏清月靠在陆知衍怀里，轻声向众人解释：“我在国外被人暗中保护的那几年，是知衍一直陪在我身边，帮我查真相、攒势力，也是他支持我回国，陪我一起把所有亏欠都讨回来。”陆知衍低头看着她，眼底满是纵容：“你的心愿，就是我的事。你要回家，我便陪你回家；你要护妹妹，我便陪你一起护。”时鸢看着眼前相拥的姐姐与姐夫，再看看身边紧紧牵着自己的傅斯年，心底被巨大的温暖填满。曾经颠沛流离、孤立无援的姐妹俩，如今一个有傅斯年倾尽所有宠爱，一个有陆知衍跨越山海守护。阴谋散尽，恶人伏法，亲情圆满，爱情成双。苏清月轻轻握住时鸢的手，陆知衍与傅斯年并肩站在她们身后。两对璧人，一家人整整齐齐，站在晨光之中，光芒万丈。陆知衍温柔开口：“接下来，该给你们办一场盛大的婚礼。小鸢和傅总，你和我，我们一起。”傅斯年低头，在时鸢额间印下一个轻吻：“好。两场婚礼，双倍幸福。”
