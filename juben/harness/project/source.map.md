## Batch 01 (EP01-05): 错认入局，断亲掉马

### EP01: 街头错认，车门锁局

**source_chapter_span**: 第1章开头至姜照眠被带入沈家前后，可自然吃进初到沈家门口的第一轮压迫。

**must-keep_beats**:
- 开场必须直接进入异常冲突：陌生男人在街头错认姜照眠，现场拦人，不能先铺背景。
- 姜照眠不认识顾砚舟，介绍前只能称他为“先生”“这位先生”或“你”，不能提前喊顾先生或全名。
- 顾砚舟因她酷似沈知微而失控，坚持带她回沈家确认，姜照眠明确拒绝被当成替身。
- 姜照眠用行动争取离开，但被车门、中控、保镖或路线形成现实阻力。
- 结尾必须形成强钩：车门锁死或车已驶向沈家，姜照眠被迫卷入一个她尚不知道的豪门漩涡。

**knowledge_boundary**: 姜照眠只知道自己被陌生人错认，不知道沈知微、沈家真千金、顾砚舟姓名与真实关系；顾砚舟只知道她长得像沈知微，不知道她真实身世；观众可以知道这是错认入口，但角色不能提前说破亲子关系。

**must-not-add / must-not-jump**: 不得提前做亲子鉴定结论；不得让姜照眠提前知道顾砚舟姓名；不得提前公开设计师“眠”身份；不得把沈雨棠的完整阴谋提前摊牌。

**ending_function**: locked_in

### EP02: 沈家验身，亲情当场碎裂

**source_chapter_span**: 第1章后段至第2章亲子鉴定、沈家认亲后的第一次冷待与冲突。

**must-keep_beats**:
- 开场直接升级：姜照眠被带到沈家，不是被欢迎，而是被审视、盘问或质疑来历。
- 沈怀章与许兰因在亲子鉴定和现实压力间摇摆，确认血缘后也没有真正给出亲情。
- 沈雨棠第一次以“沈家女儿”的姿态占位，表面委屈，实际引导父母防备姜照眠。
- 姜照眠从期待最低限度解释，转为看清沈家只想控制她、补偿她、安置她。
- 结尾落在姜照眠第一次明确划线：她不是回来争宠的人，也不会跪着认亲。

**knowledge_boundary**: 姜照眠本集可知道沈家与自己有血缘关系，但不知道当年失踪真相；沈家人知道她是真千金，却仍把沈雨棠的情绪放在前面；顾砚舟开始知道自己错认，但还没有真正理解姜照眠。

**must-not-add / must-not-jump**: 不得提前让沈家真诚悔悟；不得提前揭露沈雨棠全部作恶证据；不得让姜照眠马上接受沈家身份。

**ending_function**: confrontation_pending

### EP03: 家宴羞辱，设计师掉马

**source_chapter_span**: 第2章至第3章宴会、礼服、设计师身份线索公开。

**must-keep_beats**:
- 开场用宴会现场羞辱或服装事件切入，姜照眠被当作上不了台面的“外来者”。
- 沈雨棠借宾客、礼服、身份差距制造难堪，逼姜照眠在公众面前出丑。
- 姜照眠不靠解释求饶，而是用专业细节或作品归属反制。
- 顶级设计师“眠”的身份被现场关键人物或证据逼近公开，沈家和宾客态度反转。
- 结尾卡在身份刚被认出或即将被完全确认，沈雨棠第一次真正慌乱。

**knowledge_boundary**: 宾客和沈家大多数人起初不知道姜照眠就是“眠”；姜照眠知道自己身份但不主动炫耀；顾砚舟可通过细节重新评估她，但不能替她包办打脸。

**must-not-add / must-not-jump**: 不得提前进入旧案绑架线；不得让沈雨棠马上彻底失败；不得让顾砚舟替姜照眠完成全部反击。

**ending_function**: reversal_triggered

### EP04: 顾砚舟护短，姜照眠不领情

**source_chapter_span**: 第3章后段至第4章顾砚舟态度转变、沈家补偿和追求线开始。

**must-keep_beats**:
- 开场从宴会余波继续：沈雨棠试图把翻车解释成误会，沈家想大事化小。

- 姜照眠拒绝被“补偿”收买，要求的是边界、尊重和真相。
- 结尾形成新压力：顾砚舟意识到想靠权势靠近她无效，沈雨棠转向更隐蔽的反扑。

**knowledge_boundary**: 姜照眠知道顾砚舟姓名和身份后，可以称呼顾砚舟；顾砚舟知道自己错得具体，尚未获得信任；沈家人知道她有价值，但未真正理解伤害。

**must-not-add / must-not-jump**: 不得让姜照眠快速原谅顾砚舟；不得让沈家一集内完成悔改；不得提前安排恋爱确定关系。

**ending_function**: confrontation_pending

### EP05: 求和失败，姜照眠断亲

**source_chapter_span**: 第4章至第5章沈家求和、女主公开切割、沈雨棠下一轮谋划。

**must-keep_beats**:
- 开场必须是沈家主动求和或安排“家庭修复”场面，但现场仍把沈雨棠放在姜照眠前面。

- 姜照眠逐条拆穿所谓亲情的条件：需要她让、忍、退，却无人要求沈雨棠付出代价。
- 姜照眠明确断开对沈家亲情幻想，选择自己离开或只保留必要法律关系。
- 结尾给出阶段钩：沈雨棠发现软招无效，开始盯上姜照眠的事业或设计资料。

**knowledge_boundary**: 姜照眠已确认沈家不会自然站到自己这边；沈家父母仍把“两个女儿都要”当成公平；沈雨棠知道姜照眠不好控制，但不知道她下一步底牌。

**must-not-add / must-not-jump**: 不得提前让沈知微归来；不得提前揭开旧案主谋；不得让姜照眠失去独立判断。

**ending_function**: emotional_payoff

## Batch 04 (EP16-20): 真相落地，情感回收

### EP16: 姐姐反证，养女露尾

**source_chapter_span**: 第15章至第16章沈知微反证、沈雨棠关键漏洞。

**must-keep_beats**:
- 开场用审讯、听证或家庭对质切入，沈知微必须在压力下自证。
- 沈知微拿出反证，但现场有人质疑证据来源或完整性。
- 姜照眠补上被忽略的细节，把沈雨棠拉入同一证据链。
