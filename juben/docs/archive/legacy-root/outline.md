# 《墨凰谋：庶女上位录》故事大纲

## 剧情简介
庶女亲见生母惨死，替嫡姐入宫，以宠为刃、以情为棋，从卑微庶女一路杀到皇后太后，夺尽天下。

## 基本信息
- 剧本方向：古装宫斗 / 复仇爽文 / 大女主上位 / 权谋反转
- 集数：60集

## 故事大纲
大周永和年间，宰相府庶女沈青鸾亲眼看着生母柳姨娘被主母沈夫人活活打死，父亲沈丞相却为保官位视若无睹，嫡姐沈明珠更将她当作脚下泥。那一夜起，沈青鸾便发誓：她不要做任人碾碎的庶女，要让所有害过她的人一个个失去一切。三年后，皇帝选秀，嫡姐畏惧入宫送死，沈青鸾主动请命替姐入宫。对沈夫人而言，这是把一枚可用可弃的棋子送进后宫；对沈青鸾而言，这却是她最完美的复仇入口。她要借帝王之手，杀家门之仇；再借后宫之争，登天下之位。

**第1-12集：[血夜立誓，替姐入宫]**
暴雨夜，柳姨娘惨死柴房，十四岁的沈青鸾躲在窗外，眼睁睁看着主母夺走母亲的陪嫁手镯，也看清了父亲的冷漠和沈家的薄情。三年后，皇帝选秀，嫡姐沈明珠死也不肯进宫，沈青鸾顺势请命，以“替姐分忧”的姿态入局。入宫后，她故意扮丑示弱，躲过第一轮风头，在宸妃横行的后宫里装出怯懦病弱的模样。可她真正要做的，是让所有人都低估她。她在御花园以退为进，引起皇帝萧景珩注意；第一次侍寝便精准拿捏帝王的征服欲，让萧景珩记住这个“表面柔顺、实则看不透”的女人。很快，宸妃设宴羞辱她，逼她当众献舞，她却借皇帝撞破此局，反手让宸妃失宠。紧接着又借巫蛊案反噬宸妃，把这位后宫第一恶妃打入冷宫，完成入宫后的第一场漂亮立威。

**第13-24集：[借刀杀人，家仇入宫]**
立住脚跟后，沈青鸾开始反杀家中仇人。她表面上对沈夫人百般孝顺，实则借宫中之势一步步拆掉沈家的根。她利用皇帝的信任查出镇国将军通敌，顺势毁掉沈明珠原本想攀附的婚事；又暗中收集沈丞相贪腐罪证，让父亲从权臣高位一夜跌落，被流放边疆。沈夫人与沈明珠无路可走，只能继续求她这个“庶女”施救。与此同时，后宫里更危险的对手浮出水面。淑妃下慢性毒，皇后开始试探，沈青鸾则以病诱宠、借毒反杀，稳步抬升位份。她索性将无处可去的沈明珠接进宫，表面姐妹情深，实则要把这个最恨她的嫡姐拖入自己最擅长的修罗场。第20集附近，皇后借沈明珠之手盯死沈青鸾，忠心丫鬟翠竹为护主惨死，沈青鸾抱着翠竹尸体跪在雪地里，第一次彻底撕下柔弱伪装，立誓要让后宫所有操控她命运的人，连本带利血偿。

**第25-36集：[后宫猎杀，大仇得报]**
翠竹之死彻底点燃沈青鸾的杀心。她不再只为自保，而是开始主动织网。她一边引淑妃与皇后相互猜忌，一边纵着沈明珠生出不该有的野心，让嫡姐一步步沦为自己的刀。沈夫人原以为远离京城便能保命，却被沈青鸾借流放之地的官员之手折磨得生不如死；沈明珠以为入宫后能夺回属于自己的风光，却发现自己不过是皇后和沈青鸾之间的一枚试子。沈青鸾还盯上了皇后的软肋太子，她借“意外”撬动储位危机，让皇后乱中出错。淑妃、沈夫人、沈明珠接连覆灭，皇后也因太子被废而失势。到这一阶段结束时，沈青鸾终于站在后宫权力顶端，家中血债血偿，她也被萧景珩推上后位，完成从庶女到皇后的第一层逆天改命。

**第37-48集：[登后不止，谋国夺储]**
成为皇后后，沈青鸾并未停手。她深知后位不是终点，只要太子和前皇后还在，她就永远不算真正安全。于是她开始用更隐蔽、更帝王式的方式收拢权力。她一面装作福薄自轻，激起萧景珩不顾朝臣反对也要立她为后的执念；一面在立后之后故意纵太子骄纵跋扈，让他一步步失德失民心，最终亲手把太子送上被废之路。被废太子被她温柔收留，又死在她日日喂下的慢性毒里，前皇后也随之崩溃，被她送入冷宫。与此同时，萧景珩对她的爱与依赖日渐加深，沈青鸾却在亲手绣给他的香囊中悄悄埋下慢毒。她要的不是一世帝宠，而是萧景珩死后，她还能以正统之名继续掌天下。可就在这一过程中，帝王偶尔流露的真心，开始让她生出一瞬迟疑：她究竟是在利用萧景珩，还是也曾在某个瞬间被他的偏爱打动过？

**第49-60集：[太后临朝，独揽天下]**
沈青鸾无法生育，却偏要用一个“皇子”稳住自己的万世基业。她先以假流产除掉丽嫔，再秘密借腹生子，用宫女阿碧的孩子换来名正言顺的太子身份，随后灭口绝后患。萧景珩病重之际，终于向她挑明：他一直知道她入宫是为复仇，也知道她手上沾满鲜血，可他仍然爱她。沈青鸾第一次被这份近乎荒唐的爱震了一下，却还是在萧景珩死后毫不犹豫扶幼帝登基，自己垂帘听政。朝臣骂她妖后，她便杀到无人敢言；小皇帝长大想亲政，她便废帝夺权，把整个大周都攥进掌心。晚年时，她终于从一个求生复仇的庶女，变成真正意义上的孤家寡人。她杀尽了仇敌，也赢到了权势、帝位和千古之名。可在驾崩前，她才承认，自己这一生真正留下的遗憾，不是杀人太多，而是从未学会怎样去爱。故事以她驾崩收束，留下一个极具争议的结论：她是毒妇，是明君，是妖后，也是这个时代最清醒、最残酷、最成功的胜利者。
