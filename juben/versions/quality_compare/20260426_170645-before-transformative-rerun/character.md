# Character Reference

用于记录人物关系、立场、经历与镜头抓手，不负责规定固定台词。

## 填写原则
- 优先写人物处境、关系压力、欲望、软肋、外在抓手
- 不要把人物小传写成剧情复述，也不要直接替角色发明口号式台词

## 核心人物

### 顾闻川
- 功能定位：重生归来的实业掌舵人，负责把未来认知转化为现场选择和组织能力。
- 欲望：救回家庭命运，抢住 90 年代农业、食品和贸易窗口，建立可持续的实业集团。
- 软肋：起步阶段没人相信他；越往后越容易被旁人误解成投机取巧。
- 镜头抓手：撕合同、按住报价单、盯着出栏账、在会议桌上拆分产业链、在期货屏幕前提醒收手。
- 写作边界：不能写成全知神仙。每次判断都要落到一个可验证行动：买仔猪、扩栏、谈贷款、建厂、签人、设风控。

### 顾建岳
- 功能定位：父亲与乡土根基，代表上一世被时代误伤的人。
- 欲望：让家里稳一点，别让儿子把全家拖进风险。
- 软肋：相信眼前现金和熟人经验，对新行业、新组织、新金融工具天然警惕。
- 镜头抓手：攥合同、蹲猪圈、摸账本、看着厂房沉默、在关键节点替儿子压住场面。
- 写作边界：不能降智成只会反对。他的质疑要有现实依据，认可也要来自亲眼见到结果。

### 阎成泰
- 功能定位：开局压力人物，代表“卖掉养猪场、追逐表面机会”的上一世错误路径。
- 欲望：低价拿下养猪场或推动交易完成。
- 镜头抓手：合同、印泥、笑里藏刀的催签。
- 写作边界：不必长期复杂化，早期用来制造逼签现场和顾闻川的第一场反击。

### 周培民
- 功能定位：地方权力与政策背书。
- 欲望：看见真实税收、就业和产业项目，不愿为年轻人的空话背锅。
- 软肋：地方发展压力大，既想抓机会又怕出事。
- 镜头抓手：视察猪场/厂房、翻报表、当众追问资金和安全。
- 写作边界：支持必须有交换条件和审慎过程，不能无条件当主角外挂。

### 曹婉秋
- 功能定位：职业经理人，推动顾闻川从家庭作坊走向企业组织。
- 欲望：找到能真正放权且有增长空间的平台。
- 软肋：不接受拍脑袋式管理，会追问流程、财务和职责边界。
- 镜头抓手：摊开组织架构、当场指出账目漏洞、接过关键人事安排。
- 写作边界：要专业、克制、敢反问，不写成崇拜型女助理。

### 邵景和
- 功能定位：饲料厂执行负责人。
- 欲望：把产能、成本、品控和销售跑通。
- 镜头抓手：生产线、原料仓、成本表、质量检测。
- 写作边界：务实，不空喊口号；他的戏应围绕“能不能按时、安全、赚钱地生产”。

### 方雨晴
- 功能定位：食品与方便面品牌负责人。
- 欲望：把远丰从原料型企业推向消费品牌。
- 镜头抓手：样品面、包装袋、渠道报价、广告方案。
- 写作边界：不能只报数字。她要提出市场阻力：竞品围堵、渠道费用、产能爬坡。

### 梁策
- 功能定位：期货和风险对冲负责人。
- 欲望：用规则赚钱，同时避免企业死在价格波动里。
- 镜头抓手：期货账户、仓位表、风控线、浮盈后的沉默。
- 写作边界：不是赌徒。每次操作都必须绑定风险控制和退出条件。

### 唐靖
- 功能定位：硬执行型人才，补足企业纪律和外部硬仗能力。
- 欲望：把战场上的执行力转化为企业秩序。
- 镜头抓手：站姿、纪律表、现场处置突发冲突。
- 写作边界：少说狠话，多用行动建立压迫感。

### 宋衡
- 功能定位：农业科研资源入口。
- 欲望：让科研成果真正落地，但不愿被资本短期化。
- 镜头抓手：实验田、种子袋、科研资料、办公室追问。
- 写作边界：不能被钱一句话打动，要看顾闻川是否懂长期投入。

### 许明舟
- 功能定位：大豆种业转化人才。
- 欲望：离开不顺的旧体系，把技术推广真正做起来。
- 镜头抓手：种子样本、推广方案、被搁置的研究材料。
- 写作边界：不是普通打工人，他加入远丰应是双向选择。
