# 第26集

人物表（本集出场）
顾闻川：男主
顾建岳：顾闻川父亲
纪文峥：顾闻川小舅，退伍军人
周社长：荷县信用社负责人
曹行长：市农行负责人
工作人员若干

场26-1 内景 纪家院子-日
人物：顾闻川、纪文峥

△ 顾闻川敲门。门开时，纪文峥穿着训练服，额头上还有汗。

顾闻川：小舅，你刚才是在训练？

纪文峥：刚从部队回来，不活动一下，身体不自在。先进来坐。

△ 客厅里，顾闻川把“订单种粮试点”几页纸放到桌上。

顾闻川：我想请你帮远丰管一个新部门。

纪文峥：我？

顾闻川：远丰要和农户签订单，提供种子、肥料，按合同收玉米。先在白水镇试点，后面再往其他乡镇扩。

纪文峥翻着文件，指尖停在“一万亩”上。

纪文峥：我真的可以吗？

顾闻川：可以。这不是坐办公室的活，要下村、见人、讲信用。你是退伍军人，农户更愿意信你。

纪文峥沉默几秒，点头。

纪文峥：那我试。

顾闻川：不是试，是负责。远丰会配农学大学生和财务给你。

场26-2 内景 顾家客厅-夜
人物：顾闻川、顾建岳

△ 顾建岳把万家近期营收表推给顾闻川。数字一行比一行大。

顾建岳：琴岛、泉城几家店，开业不到两个月，现金流就起来了。外面现在看我的眼神都不一样。

顾闻川：这是好事。

顾建岳：好是好，可现在谁都觉得顾家有钱。亲戚借钱，地方找投资，银行也来问贷款。钱多了，压力也跟着来。

顾闻川看着报表。

顾闻川：远丰也要借一笔大的。

顾建岳抬头。

顾闻川：先清掉旧贷，再把大洼村和磐县养猪场整体抵押，争取一千万。顾氏建材那边也清贷后再抵押，贷一百五十万。

顾建岳：你比我还敢。

顾闻川：不是敢，是必须。猪价窗口不会一直等远丰。

顾建岳：银行看的是抵押和还款能力，不是你的野心。

顾闻川：所以我先把材料做硬。

场26-3 内景 荷县信用社办公室-日
人物：顾闻川、周社长、工作人员

△ 周社长看着还款单，脸上带笑。

周社长：顾总，目前远丰贷款一共五十万，你确定提前还清？

顾闻川：确定，辛苦您安排人处理一下。

周社长：还完之后，是不是还有新打算？

顾闻川把一摞材料推过去。

顾闻川：我想抵押大洼村和磐县养猪场，贷款一千万，用于扩建远丰养殖。顾氏建材也想抵押工厂，贷款一百五十万。

△ 周社长翻了几页，笑容慢慢变苦。

周社长：顾总，你这是给我出难题。一千万，我们县信用社拿不出来。

顾闻川：我理解。我今天主要是清旧贷。

周社长合上材料。

周社长：贷款我办不了，但我可以给市农行打个电话。远丰这种企业，不能因为信用社盘子小耽误了。

顾闻川起身。

顾闻川：这个情，我记下。

场26-4 内景 市农行行长办公室-日
人物：顾闻川、曹行长

△ 曹行长翻看资产资料。养猪场、猪舍、土地批文、出栏记录、产业园规划一份份摊开。

曹行长：顾总，远丰的情况，周社长已经介绍过。抵押物足，现金流也好，但扩张太快，是风险。

顾闻川：所以贷款用途我写得很清楚。养殖扩建、建材扩建，专款专用。

曹行长：一千万不是小数。银行要看你能不能还。

顾闻川：现在猪价、出栏计划、母猪扩繁表都在这里。就算猪价回落，远丰还有饲料、食品和产业园项目做支撑。

曹行长抬眼。

曹行长：利息方面，养猪场这笔可以定到三点二。建材那笔五点三。你能接受？

顾闻川：能接受。

△ 曹行长把钢笔递过去。

曹行长：那就签合同。

场26-5 内景 远丰办公室-夜
人物：顾闻川、纪文峥

△ 贷款合同和订单种粮试点表并排放在桌上。

纪文峥：白水镇第一批农户已经开始登记。有人问，远丰会不会秋后压价。

顾闻川：合同价写清楚，远丰不压价。质量不合格，也要有检测依据。

纪文峥：明白。

△ 顾闻川拿起产业园规划，手指停在“新城配套、食品加工、饲料扩建”几项上。

顾闻川：荷县给了远丰一张很大的门票。一千万贷款只是进场钱，后面每一步都要证明远丰配得上。

纪文峥：我这边先把粮源抓住。

顾闻川：对。没有粮，猪场、饲料厂、食品厂都会被价格卡脖子。

△ 窗外，产业园方向的施工灯亮了一排。
