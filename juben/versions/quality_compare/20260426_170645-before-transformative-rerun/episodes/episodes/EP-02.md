# 第2集

人物表（本集出场）
顾闻川：男主，接手养猪场
顾建岳：顾闻川父亲
杜老板：县里猪贩子
过磅工若干

场2-1 外景 顾家养猪场-清晨
人物：顾闻川、顾建岳

△ 猪圈门一开，百来头育肥猪挤在栏边。顾建岳拿着本子，带顾闻川一栏一栏看。

顾建岳：这一百头能出栏。十三头母猪，三头不能产仔，今天一起卖，剩下十头给你留着。

顾闻川（蹲下看猪蹄和背膘）：母猪不能动。以后仔猪要靠它们。

顾建岳：你先把眼前这摊子管住。别昨天嘴硬，今天就露怯。

△ 外面传来拖拉机声。五辆拖拉机停在门口，杜老板拎着鼓鼓的皮包下来。

杜老板（笑）：顾老哥，这批猪你终于舍得卖了。最近县里缺肉，我可等急了。

顾建岳：老杜，我信你才叫你来。价格别让我吃亏。

杜老板：市面上一块七一斤，我给你一块七毛五。

△ 顾建岳眼睛一亮，刚要点头，顾闻川伸手挡住账本。

顾闻川：不够。

场2-2 外景 顾家养猪场-清晨
人物：顾闻川、顾建岳、杜老板

杜老板（看着顾闻川）：这就是你儿子？听说不去报到了，回来养猪？

顾建岳：顾闻川，学兽医的。以后猪场交给他。

杜老板：有魄力。不过谈价不是课堂。

顾闻川：那就谈市场。县里猪肉上午就卖空，周边倒闭的猪场多，能一次拉走一百头的更少。杜老板拿到这批猪，转手就有利润。

杜老板（笑意淡了）：年轻人，二道贩子赚的也是辛苦钱。

顾闻川：辛苦钱该赚，但不能拿我爸不常去县城压价。一块九。

顾建岳（低声）：闻川，别喊太狠。

顾闻川（没退）：爸，一百头猪，不是散户两三头。量在这里，他也省车、省人、省时间。

△ 杜老板看了看猪圈，又看顾建岳。

杜老板：一块九不可能。一块八，今天全拉走，现钱。

△ 顾建岳眼神变了。顾闻川伸手。

顾闻川：成交。过磅。

场2-3 外景 养猪场过磅处-日
人物：顾闻川、顾建岳、杜老板、过磅工

过磅工：二百一十斤！

过磅工：二百零八斤！

过磅工：二百二十二斤！

△ 猪一头头上车。顾建岳盯着秤，脸上的笑压不住。杜老板蹲在一旁算账，算盘珠子拨得飞快。

杜老板：一百头育肥猪，二万一千六百四十二斤。三头母猪一千一百斤。凑整，四万零五百。

△ 他打开皮包，一沓一沓现金拍在木桌上。顾建岳手指摸过钞票边缘。

顾建岳：就这一次，多了两千多。

杜老板（苦笑）：顾老哥，你这儿子不好糊弄。

顾闻川：以后还会打交道。价公道，猪就先给你。

△ 拖拉机队开走，猪圈一下空了大半。

场2-4 外景 空猪圈前-日
人物：顾闻川、顾建岳

△ 风从空栏里穿过去。顾建岳把钱包好，塞进内兜。

顾闻川：爸，这钱你打算怎么用？

顾建岳（看他）：我就知道你盯上这笔钱了。

顾闻川：猪价已经在动。现在多留一头猪，明年就多一份本钱。

顾建岳：砖厂要建，家里也要留底。今天这笔，我分你一万。再加家里给你留的两万，三万块，你自己折腾。

△ 顾闻川接过钱，没有笑得太满，只把钱压进账本。

顾闻川：我先把猪场扩起来。

顾建岳：记住，今天只是卖得好，不是以后都好。

顾闻川：所以才要赶在别人反应过来之前动手。
