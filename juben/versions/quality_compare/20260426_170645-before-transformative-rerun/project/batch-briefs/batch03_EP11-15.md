## Episode Mapping
- EP-11：第16章 早期管培生 至 第17章（曹婉秋加盟节点）
  - 【动作】顾闻川复盘现有人员和账目，发现扩张速度已经逼近管理上限。 -> 【人物】曹婉秋登场，不被前景打动，先追问授权、财务和组织边界。 -> 【对抗】顾建岳担心外人管钱管人，顾闻川必须当场给出用人理由。 -> 【钩子】曹婉秋提出加入条件：要么真放权，要么别谈现代企业。
  - knowledge_boundary:
    - 曹婉秋知道顾家企业在扩张，但不知道后续所有产业布局。
    - 顾建岳只知道曹婉秋专业，仍不完全信任她。
- EP-12：第23 章 资金问题 至 第24 章 赚一笔快钱
  - 【阻力】扩张资金缺口暴露，曹婉秋用账目逼顾闻川面对现金流。 -> 【关系】顾建岳认为该收一收，顾闻川坚持窗口期不能错过。 -> 【动作】顾闻川提出短期快钱方案，但必须明确风险和退出点。 -> 【钩子】曹婉秋要求顾闻川把“快钱”写成可执行计划，否则她不会签字推进。
  - knowledge_boundary:
    - 曹婉秋只根据账面和计划判断，不知道顾闻川的前世经验。
    - 顾建岳知道缺钱，但不知道顾闻川后续会通过贸易、物流、食品继续扩张。
- EP-13：第26 章 回到养猪场，准备扩张 至 第27 章 进军建筑领域的可能
  - 【动作】顾闻川回到养猪场检查扩张准备，确认主业不能因为快钱线失控。 -> 【信息】他把建筑领域机会放进产业链判断，而不是单纯换赛道。 -> 【阻力】顾建岳质疑顾闻川步子太大，曹婉秋也要求项目优先级。 -> 【钩子】顾闻川画出“养殖现金流 + 建筑机会 + 后续贸易”的三线安排。
  - knowledge_boundary:
    - 团队知道顾闻川考虑建筑机会，但不知道后续服装贸易和物流公司。
    - 顾闻川不能提前说出完整远丰集团终局。
- EP-14：第28 章 新的品类，鞋子和萝卜裤 至 第29 章 贸易公司和物流公司
  - 【动作】顾闻川把服装/鞋类机会变成具体采购、销售和周转方案。 -> 【阻力】曹婉秋追问库存风险，顾建岳担心看不懂的货砸手里。 -> 【结果】贸易公司和物流公司思路成型，顾闻川开始补齐流通能力。 -> 【钩子】资金快周转即将落地，但任何一步压货错误都会拖垮现金流。
  - knowledge_boundary:
    - 团队只知道短期贸易品类，不知道后续边境贸易和致远贸易。
    - 外部商户只把顾闻川当新入局者，不能提前认可其长期实力。
- EP-15：第30 章 资金到位 至 第31章（周培民再次视察节点）
  - 【结果】短期资金回笼，扩张资金到位，前几集资金压力得到释放。 -> 【动作】周培民再次视察，看到顾闻川不是只会投机，而是把钱回到实业。 -> 【关系】曹婉秋获得更多实际管理权，顾建岳开始接受外部人才。 -> 【钩子】周培民提醒：越做大，越要经得起更高层级审视。
  - knowledge_boundary:
    - 周培民看到阶段成果，但还没有市级视察和食品厂安全危机。
    - 曹婉秋掌握管理权，但不掌握所有战略决策。

## Hard Constraints
- 不能把曹婉秋写成无条件崇拜顾闻川。
- 不能提前让她解决所有管理问题。
- 不能让快钱方案没有成本和风险。
- 不能提前写资金已经到账。
- 不能弱化养猪场主业。
- 不能跳过建筑领域的进入阻力。
- 不要把贸易写成无风险捡钱。
- 不能提前成立远东农场或期货部门。
- 不能把政府背书写成永久保护伞。
- 不能提前跳到市长视察。
