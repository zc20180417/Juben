## Episode Mapping
- EP-16：第32 章 万家超市 至 第33 章 火爆的开业
  - 【动作】顾闻川推动万家超市开业，把流通能力从设想推到现场。 -> 【阻力】开业前有人质疑货源、客流和价格体系撑不住。 -> 【爽点】超市开业火爆，顾闻川的渠道判断第一次被市场验证。 -> 【钩子】爆火不是终点，补货、管理和扩张压力同时压上来。
  - knowledge_boundary:
    - 消费者只知道新超市便宜方便，不知道远丰后续食品布局。
    - 团队只看到开业成功，还不知道扩张带来的供应链压力。
- EP-17：第34 章 超市的扩张计划 至 第35 章 疯狂扩张，存栏量突破两万
  - 【动作】顾闻川制定超市扩张计划，同时推动养猪场存栏突破两万。 -> 【阻力】曹婉秋指出现金、人员、供应链三线都在吃紧。 -> 【信息】顾闻川把超市和养殖互相支撑的逻辑讲清。 -> 【钩子】规模扩大后，仔猪、防疫和稳定供应成为下一道坎。
  - knowledge_boundary:
    - 团队知道两条线并进，但还不知道仔猪繁育体系和市长视察。
    - 顾建岳可以担心规模风险，不能提前知道后续年终架构。
- EP-18：第36 章 建立仔猪繁育体系 至 第37 章 市长视察（上）
  - 【动作】顾闻川推动建立仔猪繁育体系，避免扩张被外部仔猪供应卡住。 -> 【阻力】投入大、见效慢，顾建岳和管理层都担心拖累现金流。 -> 【信息】顾闻川把长期控制力讲成“以后别人卡不了我们的脖子”。 -> 【钩子】市级视察消息传来，繁育体系必须经得起更高层审视。
  - knowledge_boundary:
    - 市级领导尚未正式到场。
    - 团队只知道繁育体系的重要性，不知道后续饲料安全危机。
- EP-19：第37 章 市长视察（上） 至 第38 章 市长视察（下）
  - 【动作】市级视察到场，顾闻川必须用现场、账目、就业和规划回答问题。 -> 【阻力】视察不是表扬会，领导追问风险、产能和地方价值。 -> 【结果】顾闻川用实业结果赢得更高层级关注。 -> 【钩子】更大机会出现：收购县食品厂，把农业端推向食品加工端。
  - knowledge_boundary:
    - 市级领导只根据现场表现判断，不知道顾闻川未来会布局远东和期货。
    - 团队知道食品厂机会将至，但尚未完成收购。
- EP-20：第39 章 收购县食品厂 至 第40 章 饲料安全问题
  - 【动作】顾闻川推动收购县食品厂，远丰从养殖端进入加工端。 -> 【阻力】旧厂资产、人员、债务和产能问题暴露。 -> 【危机】饲料安全问题压到桌面，远丰信誉第一次遭遇系统性考验。 -> 【钩子】顾闻川必须当场决定：遮掩保短期，还是公开查清保长期。
  - knowledge_boundary:
    - 外界只知道远丰扩张和安全风波，不知道年终总结后的架构升级。
    - 团队不知道顾闻川会如何处理危机。
