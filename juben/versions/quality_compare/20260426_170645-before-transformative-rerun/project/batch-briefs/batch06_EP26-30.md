## Episode Mapping
- EP-26：第54章（顾建岳被重新看待节点）至 第56 章 荷县新城和产业园
  - 【关系】顾建岳因远丰做大被外界重新看待，但压力也落回家庭。 -> 【动作】顾闻川争取 1000 万贷款，必须用资产、订单和产业园计划换信用。 -> 【阻力】银行和地方都担心远丰扩张过快。 -> 【钩子】荷县新城和产业园机会打开，远丰必须证明自己配得上这张门票。
  - knowledge_boundary:
    - 银行只看风险和抵押，不知道后续方便面、期货收益。
    - 顾建岳知道儿子做大了，但不掌握所有金融安排。
- EP-27：第57 章 饲料厂估值 至 第58 章 新版方便面
  - 【动作】邵景和拿出饲料厂产能和估值数据，证明制造端价值。 -> 【动作】方雨晴推出新版方便面，食品线从工厂产品转向品牌产品。 -> 【阻力】团队必须同时面对原料上涨、竞品围堵和渠道费用。 -> 【钩子】饲料厂投产与方便面上市窗口撞在一起，远丰必须双线打赢。
  - knowledge_boundary:
    - 邵景和只负责饲料执行，方雨晴负责食品品牌，两人不知道期货线即将展开。
    - 市场只看到新产品，不知道远丰后续资本和贸易布局。
- EP-28：第59 章 饲料厂投产与开业 至 第60 章 战斗英雄刘浩
  - 【动作】饲料厂投产开业，顾闻川盯现场、订单和质量。 -> 【阻力】产能爬坡、设备磨合和外部观望同时出现。 -> 【人物】唐靖入场，用纪律和执行力补远丰扩张后的现场管理短板。 -> 【钩子】远乡方便面即将铺开，品牌硬仗比建厂更难。
  - knowledge_boundary:
    - 唐靖刚进入远丰，只知道需要整顿执行，不知道集团全局。
    - 外界只看到开业，不能提前知道远丰家底和实业成立。
- EP-29：第61 章 远乡方便面 至 第62 章 远丰的家底
  - 【动作】方雨晴推动远乡方便面铺货，渠道和消费者反馈同时到来。 -> 【阻力】竞品、货架费、经销商信任都不是一句广告能解决。 -> 【信息】顾闻川复盘远丰家底，明确哪些钱能动、哪些钱不能动。 -> 【钩子】远丰需要从多项目组合升级为真正公司主体。
  - knowledge_boundary:
    - 经销商只看产品和利润，不知道远丰后续期货和远东布局。
    - 团队知道家底情况，但还不知道远丰实业正式成立后的新边界。
- EP-30：第63 章 远丰实业 至 第64 章 远东农场负责人
  - 【结果】远丰实业成立，把养殖、饲料、食品、贸易、物流纳入更清晰架构。 -> 【动作】顾闻川讨论远东农场负责人，开始把视线投向更远资源端。 -> 【阻力】团队质疑远东布局跨度太大，管理和信任都跟不上。 -> 【钩子】顾闻川提出需要金融对冲能力，否则原料价格会反噬企业。
  - knowledge_boundary:
    - 团队知道远丰实业成立和远东农场设想，不知道期货具体操作。
    - 远东农场负责人尚未完全证明能力。
