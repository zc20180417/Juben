## Episode Mapping
- EP-21：第40 章 饲料安全问题 至 第42 章 年终总结（下）
  - 【动作】顾闻川要求查清饲料安全问题，现场处理责任和流程漏洞。 -> 【结果】年终总结把危机、利润、扩张问题集中摊开。 -> 【关系】曹婉秋推动制度化管理，顾建岳接受不能再靠熟人经验。 -> 【钩子】顾闻川宣布远丰需要新架构和专业人才。
  - knowledge_boundary:
    - 团队知道安全风波被处理，但还没有形成完整新架构。
    - 外界只看到远丰经受考验，不知道后续高校人才线。
- EP-22：第43 章 人才问题 至 第46 章 面试
  - 【信息】顾闻川发现扩张最大瓶颈从钱变成人才。 -> 【动作】远丰新架构成型，岗位、权限和业务线重新划分。 -> 【人物】顾闻川偶遇海洋大学教授线索，并通过面试筛选专业人才。 -> 【钩子】专业人才愿不愿意加入，取决于远丰是否真有长期产业计划。
  - knowledge_boundary:
    - 候选人只知道远丰在扩张，不知道后续产业园和方便面品牌计划。
    - 顾闻川可以表达长期愿景，不能提前给出全部后续成果。
- EP-23：第47 章 1994，猪价开启狂飙模式 至 第48 章 对饲料厂和食品厂的规划
  - 【爽点】1994 猪价狂飙，顾闻川早期判断得到大规模验证。 -> 【反转】顾闻川没有沉迷猪价，而是要求饲料厂、食品厂同步规划。 -> 【阻力】团队担心摊子太大，原料、产能、销售都可能卡住。 -> 【钩子】顾闻川把产业园作为下一阶段承载平台。
  - knowledge_boundary:
    - 团队知道猪价大涨，但不知道产业园能否落地。
    - 外界只看到远丰赚钱，不能提前知道食品估值和融资。
- EP-24：第49 章 远丰产业园 至 第50 章 食品厂的发展
  - 【动作】顾闻川推动远丰产业园规划，把饲料、食品、物流协同落到空间上。 -> 【阻力】土地、资金、审批和产能爬坡同时压来。 -> 【信息】方雨晴接住食品厂发展线，提出消费端不能只靠产能。 -> 【钩子】食品线需要融资和估值，远丰要接受外部资本检验。
  - knowledge_boundary:
    - 方雨晴知道食品厂发展压力，但还没有完成融资。
    - 政府和团队只看到产业园规划，不知道后续千万贷款。
- EP-25：第51 章 拉投资，食品厂的估值 至 第53 章 益农食品
  - 【动作】顾闻川和方雨晴推动食品厂估值与融资谈判。 -> 【阻力】投资方质疑远丰只是地方企业，食品品牌能否出圈。 -> 【信息】蔬菜加工厂、益农食品线补足食品板块，不只做单一方便面。 -> 【钩子】顾闻川需要更大贷款和更硬资产，才能支撑下一轮扩张。
  - knowledge_boundary:
    - 投资方只根据当下食品厂和渠道判断，不知道后续饲料厂估值。
    - 团队知道食品线扩张，但不知道贷款能否拿下。
