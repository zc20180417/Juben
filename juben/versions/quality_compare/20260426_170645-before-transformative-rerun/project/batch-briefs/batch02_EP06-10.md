## Episode Mapping
- EP-06：第7章 砖厂的恐怖利润 至 第8章 卖猪，惊人的收益
  - 【动作】顾闻川核算砖厂和卖猪两边收益，把账本摆到顾建岳面前。 -> 【爽点】卖猪收益超出村里预期，旁人开始改口。 -> 【关系】顾建岳从反对转为追问下一步怎么守住风险。 -> 【钩子】顾闻川提出养猪场不能只扩大规模，还要掌握上游和现金流。
  - knowledge_boundary:
    - 村里人只看到顾家赚钱，不能提前知道后续产业园。
    - 顾建岳开始信任顾闻川，但还不是完全放权。
- EP-07：第9章 养猪场扩张计划 至 第10章 镇长来访
  - 【动作】顾闻川制定养猪场扩张计划，目标从家庭作坊变成镇级项目。 -> 【阻力】周培民来访，不听空话，只问环保、防疫、就业和贷款风险。 -> 【信息】顾闻川用出栏数据、用工计划和税收预期回应。 -> 【钩子】周培民没有当场拍板，而是要求顾闻川拿出更硬的证明。
  - knowledge_boundary:
    - 周培民只知道顾家养猪场近期赚了钱，不知道顾闻川后续跨行业扩张。
    - 顾闻川可以谈地方产业价值，不能提前许诺超出当前阶段的成果。
- EP-08：第11章（周培民支持节点）至 第12章 投资砖厂的想法
  - 【对抗】周培民在支持前继续追问风险，顾闻川必须把项目收益和就业说清。 -> 【结果】镇里给出有限支持，顾闻川拿到扩张所需的政策空间。 -> 【信息】顾闻川把砖厂机会纳入现金流安排，而不是盲目转行。 -> 【钩子】顾建岳发现儿子已经在同时盘养殖和建材两条账。
  - knowledge_boundary:
    - 周培民知道顾闻川有产业想法，但还不知道食品厂、超市、远东贸易。
    - 顾建岳知道砖厂能赚钱，但不清楚顾闻川会把它当资金跳板。
- EP-09：第13章 首富 至 第14章 招兵买马
  - 【爽点】顾家阶段性成为本地显眼的赚钱户，“首富”称呼带来面子也带来压力。 -> 【动作】顾闻川意识到靠父子两人扛不住扩张，开始招兵买马。 -> 【阻力】来应聘的人看重短期钱，未必能跟上顾闻川的制度要求。 -> 【钩子】顾闻川提出要找能管企业的人，而不是只找能干活的人。
  - knowledge_boundary:
    - 外人只知道顾家赚了钱，不知道顾闻川下一步要搭现代公司架构。
    - 新人只能知道当前岗位，不提前知道全剧产业线。
- EP-10：第15章 春节 至 第16章 早期管培生
  - 【关系】春节节点展示亲戚和村里人对顾家的态度变化。 -> 【动作】顾闻川提出管培生/年轻人培养思路，让企业不再只靠熟人。 -> 【阻力】顾建岳担心年轻人不稳，老工人也怕新人抢位置。 -> 【钩子】顾闻川把第一批培养名单摆上桌，企业组织化正式开始。
  - knowledge_boundary:
    - 亲戚和村民只看到顾家变有钱，不能知道后续集团化。
    - 管培生只进入早期培养，不具备后续高管能力。

## Hard Constraints
- 不要把收益写成无限资金。
- 不要提前出现职业经理人团队。
- 不能让政府支持无条件落地。
- 不能提前写市长视察。
- 不能把砖厂写成主业替代养猪场。
- 不能提前写县级、市级领导全面背书。
- 不能提前让曹婉秋入职。
- 不能把顾家首富写成全国性财富。
- 不能把管培生立刻写成成熟管理层。
- 不能跳到食品厂或超市开业。
