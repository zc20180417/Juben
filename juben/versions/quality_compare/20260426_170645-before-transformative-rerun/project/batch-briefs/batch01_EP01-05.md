## Episode Mapping
- EP-01：第1章 首富从继承养猪场开始
  - 【动作】顾闻川冲进门，赶在阎成泰和顾建岳签字前拿走卖场合同。 -> 【信息】顾闻川不能说重生，只能用猪价、建材风险和家底承压说服父亲。 -> 【关系】顾建岳认为儿子年轻胡闹，阎成泰催签制造当场逼选。 -> 【钩子】顾闻川撕开合同局面，承诺一个月内让养猪场赚到第一笔硬钱。
  - knowledge_boundary:
    - 顾建岳不知道顾闻川重生，只知道儿子突然强硬反对卖场。
    - 阎成泰只知道交易被搅黄，不知道顾闻川掌握未来行情。
    - 顾闻川可以提市场判断，不能提前说出后续所有产业布局。
- EP-02：第2章 卖猪
  - 【动作】顾闻川亲自盯卖猪现场，要求按上涨预期重新谈价。 -> 【对抗】买方试图压价，顾建岳担心猪砸在手里。 -> 【信息】顾闻川用供需、春节消费和周边猪源不足解释判断。 -> 【结果】第一批猪卖出高于预期的价格，顾建岳第一次动摇。
  - knowledge_boundary:
    - 顾建岳只看到这一次卖猪结果，不能知道未来猪价全线狂飙。
    - 买方只按当下市场压价，不知道顾闻川的未来认知。
- EP-03：第3章 扩大规模 至 第4章 招工和贷款
  - 【动作】顾闻川提出继续扩栏，不把第一笔钱落袋为安。 -> 【阻力】顾建岳担心猪价回落、疫病和人工不够，要求顾闻川给出可执行方案。 -> 【信息】顾闻川拆出猪舍、仔猪、人工、周转周期和贷款缺口。 -> 【钩子】父子来到银行/信用社门口，扩栏从家庭争论变成外部融资压力。
  - knowledge_boundary:
    - 顾建岳知道卖猪赚了，但还不知道顾闻川后续会做饲料、食品、贸易。
    - 银行/信用社人员只看抵押和还款能力，不能无理由相信顾闻川。
- EP-04：第4章 招工和贷款 至 第5章 准备就绪，购买仔猪
  - 【动作】顾闻川一边招工一边谈贷款，把扩栏计划拆成可落地清单。 -> 【阻力】村里人观望，担心养猪场扩大后赔光，顾建岳也担心人手失控。 -> 【信息】顾闻川坚持先把猪舍、防疫、饲料采购和人员职责定清楚。 -> 【结果】仔猪进场，养猪场从保住变成主动扩张。
  - knowledge_boundary:
    - 工人只知道顾家要扩大养殖，不知道顾闻川的全部商业计划。
    - 顾建岳可以认可执行细节，但仍保留对规模风险的担心。
- EP-05：第6章 配合饲料 至 第7章 砖厂的恐怖利润
  - 【信息】顾闻川提出配合饲料思路，把养猪优势从等行情变成控成本。 -> 【对抗】顾建岳和工人质疑新配方不稳，担心猪出问题。 -> 【动作】顾闻川现场调整原料比例和喂养流程，要求用小批量先验证。 -> 【钩子】砖厂利润线被抛出，顾闻川意识到下一笔快钱不只在猪圈里。
  - knowledge_boundary:
    - 众人只知道顾闻川懂一些饲料配比，不能知道他会做成饲料厂。
    - 砖厂机会只是刚被看见，不能提前写投资完成。

## Hard Constraints
- 不能提前写镇长支持、贷款、饲料厂或后续公司成立。
- 不能把顾建岳写成立刻完全相信顾闻川。
- 不能提前写配合饲料技术成型。
- 不能把第一笔收益夸张成足以支撑后续所有扩张。
- 不能跳过招工、猪舍和贷款这些扩栏前置难题。
- 不能让贷款无阻力秒批。
- 不能提前写成现代企业制度完全成型。
- 不能跳过防疫、饲料、人工这些现场阻力。
- 不能把配合饲料直接写成成熟商品。
- 不能提前成立远丰或食品线。
