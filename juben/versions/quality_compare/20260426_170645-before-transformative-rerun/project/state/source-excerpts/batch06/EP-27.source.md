## Original Excerpt
第57 章 饲料厂估值

思考了一会儿，林远缓缓开口：

　　“我之前确实没有考虑过这个问题，不过刚才我仔细想了想，饲料厂这边确实可以引进一部分外部的资金用于扩张。

　　我这边计划让出三成的股份，由县里和我父亲那边平分，不过具体的估值我还得回去好好思考一下才能给您回复。”

　　高伟良点了点头：“好，除了投资之外也可以贷款，现在饲料厂也已经建好了，可以用来抵押进行贷款了，当然食品厂那边也是一样。”

　　相比股份，高伟良更关心饲料厂和食品厂的扩张，毕竟这可都是实实在在的GDP和税收。

　　相比起来那点儿分红，还不如留着给工厂自己进行扩张呢。

　　“好的，高县长，我回去会好好计划一下两个工厂扩张的事情的。”

　　和高伟良道别后，林远并没有像往常一样先返回大洼村，而是打算直接去饲料厂看看。

　　至于交通工具，则是林远心爱的小摩托。

　　“老板，你现在也是一个大老板了？啥时候买一辆小轿车，一直开着个摩托车，一点儿老板的样子都没有。”

　　镇政府楼下，林远的摩托车旁边，高芳芳鼓着嘴吐槽道。

　　“我这是发扬艰苦奋斗的精神，现在远丰正是发展的关键时期，还没有到该享受的时候。”

　　林远听到高芳芳的话，笑容僵在了脸上，但还是嘴硬道。

　　重生一年多，一直都是骑自行车或者开摩托车，林远都已经习惯了，自然也就忘了买车的事情。

　　“行了，回头我就买一辆，现在只有摩托车，你坐不坐，不坐就自己想办法回去。”

　　林远拍了拍摩托车的后座，试图结束这个话题。

　　“哎，摊上你这样的老板，算我倒霉。”

　　高芳芳叹了一口气，随后坐在了摩托车的后座上，双手扶住了摩托的后备箱。

　　“坐稳了！”

　　林远回头看了，一眼高芳芳，提醒了一句，然后发动了摩托车，向着白水镇的方向驶去。

　　行驶在公路上，感受着迎面吹来的风，林远心情都舒畅了不少。

　　现在这个季节确实是开摩托车的好季节，不冷不热，路上的车也没有后世那么多，不用吃尾气，可以尽情地拥抱自然。

　　十几分钟后，两人到了产业园外面，林远将摩托车停在路边，看着那一辆辆向饲料厂仓库运输原料的货车，不禁再次感慨白水镇的优越条件。

　　本身白水镇就位于曹州这个产粮大市，而曹州又处于几省交界处，隔壁的西边的豫省更是中原粮仓，原料非常丰富。

　　而市场方面，无论是东边的鲁省还是南边的苏省，都是龙国经济前三的大省，市场也非常广阔。

　　在发展食品加工产业方面，说得上是得天独厚了。

　　推着摩托车，刚走进饲料厂的大门，林远便远远看到李星和向自己走来。

　　“李哥，你该忙忙，不用每次都这样，我来得次数也不少了，各个地方我都认识。”

　　两人走近，林远笑着和李星和打了一个招呼。

　　“林总，我也是刚从办公室出来，正要去生产线那边看看，听人说您来了，我就自己过来了，我现在刚好有事情想找您商量一下。”

　　李星和笑着解释道。

　　“这么巧吗，那咱们就一起去车间看看吧，边走边聊。”

　　林远脸上闪过一丝意外，将摩托交给了保安，示意李星和带路。

　　“好的，林总，您跟我来。”

　　李星和脸上露出笑容，伸手示意让林远两人跟着自己走。

　　“林总，我找您是想和您商量一下饲料该怎么定价，现在市面上的普通饲料小猪料一斤是9毛到一块，育肥料一斤是七毛到八毛。

　　不过咱们的饲料无论是长肉速度还是瘦肉率都要比市面上普通的饲料更好，所以这个价格我想和您一起确认一下。”

　　走在路上，李星和缓缓开口说出了自己的问题。

　　林远听后，也微微皱眉，这也确实是一个问题。

　　作为对外出售的饲料，和远丰自己用的饲料还是有一些区别的，是韩泽成入职远丰后的第一个成果。

　　为了赚更多的利润，其原料成本比自家用的要低不少，但是其效果依然能比市面上普通的饲料好不少。

　　无论是消耗的饲料数量，还是出栏时间都能比普通饲料少5%。

　　其价格和普通的饲料一样肯定是不合理的，但是远丰的饲料毕竟是第一次卖，并没有将口碑给攒起来，如果价格太高容易出现滞销的情况。

　　思考了一会儿，林远缓缓开口：“咱们的成本本来就低，就算不卖高价，也能赚取足够的利润，最开始的时候就对标普通饲料的价格吧。

　　小猪料一斤一块，育肥料一斤八毛，前三个月为了打响名气，价格再各降一毛钱。”

　　虽然刚开始就把价格定的低，不方便后面涨价，但是现在猪饲料因为通货膨胀的原因，价格本来就不正常。

　　只要等后面市场上其它饲料降价时，自己不降或者少降就能将价格定在合理区间了。

　　“好的，林总。”

　　李星和听林远说完，也松了一口气。

　　要是林远把价格定太高的话，销售压力可就全放到他这个饲料厂的负责人身上了。

　　现在这样的价格的话，就不用担心饲料卖不出去了。

　　“对了，我这次来饲料厂，是想和你商量一下饲料厂后续扩张的事情。”

　　林远突然想起了自己的目的，继续说道。

　　“饲料厂每年的利润预计有多少？”

　　“林总，现在饲料厂每个月能生产1000吨饲料，营收在180万左右，因为咱们的配方比较优秀，净利润大概能达到12%的水平，每月利润为21万，一年的利润大概在250万。

　　按照市盈率计算的话，饲料厂的估值为2500万。

　　不过考虑到咱们有多个核心配方，再加上净利润足够高，估值应该还能再提高一半，即使报价3000万也有人愿意投资。”

　　李星和思考了一会儿，报出来一串让林远惊喜的数字。
