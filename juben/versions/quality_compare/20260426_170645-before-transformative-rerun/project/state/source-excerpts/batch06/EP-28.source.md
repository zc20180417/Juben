## Original Excerpt
第59 章 饲料厂投产与开业

“喂，儿子？找我啥事儿？”

　　很快，林建军的声音便从电话里传出。

　　“老林，谁啊！”

　　“我，儿子！”

　　没等林远开口，电话那头再次有谈话的声音传出。

　　“爸，你现在在忙吗？是不是打扰到你了。”

　　林远听着电话那头的噪音，微微皱眉。

　　“参加了一个饭局，不用在意，有啥事儿直接说就行。”

　　林建军那边的噪音减轻了不少，应该是换了一个地方。

　　“也没啥大事儿，就是我打算把饲料厂也拿出来融资，计算让出三成股份，由你和县里平分，你这边儿有兴趣吗？”

　　林远也没有浪费时间，直接进入了主题。

　　“当然有兴趣，不过我这边一时半会儿估计没办法回荷县，回头我让我公司的法务过去办理一下合同。”

　　林建军语气带着一丝惊喜，能赚钱的生意，他没有理由错过。

　　“行，那咱们就这么说定了！”

　　挂断了电话，林远深深地呼了一口气，他也不知道自己这样把股份都撒出去的做法到底正不正确，会不会影响自己对这些子公司的掌控。

　　尤其是林远最初搞这些公司的目的是想给养猪场输血，支持养猪场的扩张的，随着自己这么一搞，几家工厂都变成了完全独立的公司了。

　　其发展速度肯定能比原来更快，发展上限或许也能比原先更高，成为比肩远丰养殖的大企业都说不定，只是远丰以后想要再抽调利润只能通过分红或者专利费的形式了，要复杂很多。

　　“既然已经决定了，那就不能后悔，接下来要做的就是努力发展好几家企业！”

　　林远握了握拳，悄悄给自己打了一个气。

　　几天后，林建军安排的人抵达白水镇，林远带着他和远丰的法务一起前往了县城。

　　很快，一份全新的融资合同顺利出炉。

　　荷县国资局和万家集团各出资450万，获得远丰饲料15%的股份。

　　随后林远又从农行搞到了200万的抵押贷款和100万的经营贷款。

　　一共1200万资金，全部投入到饲料厂的二期工程之中。

　　考虑到建造新的生产线需要几个月时间，而饲料厂的第一个万吨生产线已经开始运转，并创造利润了。

　　所以不需要考虑囤积原料和保留流动资金的问题，这1200万可以全部用于厂房建设还有引进设备。

　　林远和李星和商量之后，打算直接引进一条10万吨的饲料生产线，将饲料厂的年产量提高到10万吨。

　　刚刚离开不久的施工队再次回到了产业园的工地之上，并且这次出动的人数还要更多，之前在建食品厂还有蔬菜加工厂厂房的施工队也都投入到了饲料厂的扩建之中。

　　相比其他地方的施工队，荷县的施工队因为远丰的存在获得可以说是相当滋润，规模也在不断逆势扩张。

　　大量的农民工也进一步带动了荷县的就业，刺激了荷县的经济。

　　可以说整个曹州，没有一个地方不羡慕现在的荷县的。

　　而林远看着庞大的工地，再次考虑要不要建立自己的建筑公司。

　　“还是算了！”

　　林远摇了摇头，现在房地产市场太过低迷，远丰的工厂和猪舍的建造需求虽然一直都有，但也存在很大的不确定性。

　　不能保证建筑公司始终有订单，相比起自己搞建筑公司节省的成本，其风险还是太大了。

　　还是等荷县的经济进一步好转之后再说吧，以荷县的发展态势应该能比国内其他地方先一步从寒冬中走出来。

　　等荷县打算搞新城的时候，拉上万家一起建立一家建筑公司才是最合适的。

　　饲料厂的扩建，并没有影响其正常开业。

　　1994年5月10号，饲料厂产线最终调整完成，开始全力运转。

　　随着林远和李星和一起按下开关。

　　车间里机器爆发出巨大的轰鸣声，一袋袋印着“远丰”二字的饲料不断从生产线上掉下来，然后被工人用叉车拉走，等待销售。

　　1994年5月20号，第一批300吨饲料生产完成。

　　在李星和的安排下，50吨饲料被货车从饲料厂拉走，运往了白水镇远丰直营的饲料店进行零售。

　　剩余的250吨则会按照订单，由远丰的人送货上门，直接从饲料厂拉到对应的养猪场。

　　上午八点，白水镇，远丰饲料门店正式开门营业，早已因为远丰的名气等候多时的养殖户们，瞬间涌入了饲料厂！

　　“大家不要挤，有序排队！”

　　饲料店的员工拿着一个外放喇叭，试图维持一下秩序。

　　只不过效果并没有想象中的那样好。

　　“给我拿10袋饲料！”

　　一名穿着蓝色工装的中间人冲到了柜台前面，挥舞着手中的钞票大声喊道。

　　“我先来的，先给我，我要20袋！”

　　旁边马上有人将其挤开，随后报出自己的订单。

　　“大家不要着急，我们这次备货很充足！足足有300吨，六千袋，足够满足大家需求了！”

　　“这一批卖完了，下一批不知道还得等多久，先到先得！”

　　人群中马上就有人喊道。

　　远丰饲料厂的产量和质量，在白水镇不是秘密，虽然效果很好，但是每天就30吨，完全不够分，因此所有人对于这批饲料都势在必得。

　　对于这种情况，林远和李星和也没有预想到，他们猜到饲料门店第一天开业会很火，但是也没有想到会这么火。

　　两人还是低估了现在的猪价对于饲料行业的影响了。

　　高昂的猪价，让各地的养殖户都更舍得使用饲料来加快猪的成长，而远丰的饲料质量又这么好，他们肯定不能错过。

　　不得已，林远和李星和又紧急从饲料厂那边调来了几个人维持现场秩序，两人连带着高芳芳也都兼职了一把售货员。

　　好在当初建远贸易开建立的时候，林远也当过售货员卖衣服，因此卖起来还算是得心应手。

　　林远熟练地在计算器上敲下一个个数字，随后填好一张张单子。

　　“小丰村大民养猪场，50袋小猪料，50袋育肥料。”

　　“赵老板，你收好条子，今天下午远丰的人就能送货上门，到时候一手交钱一手交货！”

　　林远笑着将一份订货单递给了赵大民。
