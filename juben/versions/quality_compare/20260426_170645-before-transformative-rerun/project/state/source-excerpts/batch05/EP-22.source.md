## Original Excerpt
第43 章 人才问题

林远听到这话，眼睛一亮：“爸既然你找了猎头，那为什么只给万家找人，建远贸易、建远物流还有建远建材，以及我打算成立的建远建筑，都需要专业的经理人来打理。

　　咱们现在又不缺钱，完全可以一次性找齐。”

　　其它几家公司虽然不是目前的重点，但是不代表其没有发展潜力。

　　只是林建军和林远两人精力有限，只能维持一个架子，没有办法控制其快速发展、扩张。

　　不如找一些职业经理人来执掌这些公司，林远只需要在合适的时候，结合自己后事的眼光，提供一些建议就行。

　　“好主意，我现在就和猎头公司那边联系。”

　　林建军也知道其他几家公司的潜力，以林家现在的底蕴，完全可以试着走出去看看，就算失败了，最坏的结果也就是回到曹州继续混日子，该赚的钱也不会少赚。

　　想到这，林建军当即拿出了自己的大哥大给猎头那边打了过去。

　　“喂，我是万家的林建军，之前联系过你们？”

　　“林总，您好，您打电话过来是要询问我们这边招人的进展吗？”

　　“不是这个，我打电话是想让你们帮我多找一些职业经理人，除了能够执掌万家的职业经理人之外，再帮我找一个贸易公司总经理、一个物流公司总经理、一个建材公司总经理、一个建筑公司的总经理。

　　至于我公司的具体情况，我相信以你们的实力应该能调查清楚，然后帮我物色到合适的人选，待遇不是问题，猎头费也不会少，你们大胆去谈！”

　　林建军说完，手机那头呼吸声都重了不少。

　　“林总，您放心，我们是专业的猎头公司，绝对不会让你失望的！”

　　猎头公司的人兴奋地保证道。

　　“嗯嗯，我相信你们，如果你们这一次找的总经理能让我满意，后续财务、法务、销售需要的人我也都找你们。”

　　给猎头画了一张饼，并得到了再三保证后，林建军挂掉了电话。

　　“希望猎头公司那边速度能快一点儿，这样我也能轻松不少了。”

　　“爸，把猎头的联系方式给我一份儿呗。”

　　林建军刚挂掉电话，林远便凑了过来，笑着说道。

　　随着养猪场的规模扩大，需要的人才也是在直线提升，别的不说，至少在磐县的新养猪场肯定需要一个新的厂长管理。

　　还有母猪、饲料这些，最好都有专门的人管理，光靠刘玉芳和林远两个人精力确实有些不足。

　　猎头那边虽然找那些专业的研究人员难度稍微大一点儿，但是找几个专业的经理人或者懂技术的管理还是没有问题的。

　　林远打算明天回养猪场就和刘玉芳商量一下。

　　.....

　　次日，在家里短暂休息了几天的林建军再次离开了林家，前往了曹州，处理银行贷款的事情。

　　与之一起的还有郑淑芳，林远又变成了留守儿童，谁让养猪场就建在大洼村呢？

　　“唉”，看着空荡荡的家，林远长长的叹了口气，无奈地走出家门，在路上买了一个煎饼后，便直接前往了养猪场。

　　办公室里，刘玉芳现在还没有处理完手头的工作，林远便自己研究起远丰需要的人才情况。

　　研究了一会儿，在本子上写写画画了一会儿，林远不禁揉了揉头，远丰成立的时间还是太短，扩张速度又太快，人才缺口也确实很大。

　　除了之前和林市长提的科研人员之外，职业经理人、专业的财务、法务甚至销售、采购都存在不足。

　　也不知道那家猎头的水平怎么样，同时对于这个时代的人才储备情况，林远也没有太过乐观，相比后世的人才过剩，现在无疑是人才最稀缺的时期之一。

　　“除了外部招募之外，内部培养也不能放松。”

　　林远将笔摔在本子上，深呼了一口气。

　　作为一个重生者，林远的眼光还是比较长远的，今年暑假，除了高伟良介绍过来的那批中专生之外，林远又招募了一批高中生进行定向培养。

　　没办法，养猪场规模摆在这儿，靠县中专分配过来的那三瓜俩枣，还有刘玉芳偶尔挖过来的几名专业人才根本不够用。

　　面对现在数万头的存栏量，甚至未来数十万的存栏量，只能先内部培养。

　　“说起来这批高中生也已经培养了几个月了，也该去看看他们的成长情况了。”

　　相比第一批，这一批“管培生”的运气明显要差了一点，林远这个大老板没有时间像上一批那样一点点教导他们。

　　只能先跟着之前的老员工学习，学习速度和学习强度都比不了，几个月过去也才勉强上手，能在老员工的指导下处理一下专业的问题。

　　随着林远忙起来之后，更是将他们彻底放在一边。

　　“砰砰砰”

　　突然，敲门声响起，将林远从思索中拉了出来。

　　“进！”

　　随后刘玉芳拿着一份文件走了进来。

　　“林总，你说的那个人才问题我也注意到了，刚好我这段时间做了一份计划书，可以和这个问题一起解决。

　　再过一个多月咱们的养猪场就能扩建完成了，到时候规模将是现在的几倍，成为一家真正的大型养猪场，所以我打算再对咱们养猪场的组织架构进行一次调整。

　　刚好趁着这次机会，咱们一起定下来。”

　　刘玉芳一边说着，一边将手里的文件递给林远。

　　林远打开文件，眼中闪过一丝意外，刘玉芳设计的架构竟然已经和未来的差不多了。

　　按照刘玉芳的规划。

　　远丰养猪场未来主要将由五个部门组成，分别是技术中心、生产中心、后勤中心以及独立的兽医部、饲料部。
