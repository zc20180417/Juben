## Original Excerpt
第40 章 饲料安全问题

“我之前也在市食品厂当过副厂长，对这一行也有一定了解，我之前看了一下县食品厂的设备和工艺，其虽然并不算顶尖，但也绝对没有那么差。

　　特别是之前有万家的帮助，竟然还能亏损，这绝对是不合理的。

　　我猜测是县食品厂的厂长和管理层，想要自己低价将食品厂买下来，故意将食品厂经营亏损的。

　　这次是我们大意了。”

　　刘玉芳缓缓开口道，之前无论是林远还是方雨桐，都没有注意到一个细节。

　　那就是有万家超市这个通天代，县食品厂到底是烂到了什么程度，才能做到继续亏损的。

　　这个烂不止可以烂在技术和观念，更可能是烂在人上。

　　林远深呼了一口气。

　　“刘姐，你说的很有道理，确实有这个可能，我们接下来也可以从这方面入手，看看之前先食品厂的管理层有没有搞什么动作。”

　　......

　　事实上，作为一个在国有企业里面摸爬滚打这么多年，并一直做到市食品厂副厂长的女人，刘玉芳的猜测相当准。

　　时间回到几天前。

　　林远打算收购县食品厂的消息传到了县食品厂。

　　县食品厂厂长赵玉刚将自己的茶杯狠狠地摔在了地上。

　　“该死，老子辛苦了这么久，竟然让一个毛头小子给截胡。

　　不行，不能让这小子这么舒服，我赵玉刚的便宜可不是那么好占的。”

　　只是林远的养猪场几个月后才会卖猪，林市长又刚刚视察过，想报复在林远最核心的养猪场上报复林远也确实不容易。

　　而只在食品厂上给林远添堵，赵玉刚又有些不甘心，于是便想出来一条毒计，既能报复林远，又不容易牵扯到自己身上。

　　随后便有了远丰仓库里的那些玉米。

　　作为食品厂的厂长，他对食物的性质还是很清楚的，很容易就想到了将玉米捂热迫使其变质的方法。

　　不需要去市面上购买任何毒药，只需要从农民手里收购一批玉米，然后将其捂热后，卖给为远丰提供饲料的供应商就行。

　　行踪隐蔽，几乎没有任何被发现的可能。

　　等林远的养猪场出事了，到时候县里肯定不会再把食品厂继续交给林远了，最好的方式就是由他这个厂长买下了。

　　只是赵玉刚完全想不到，林远两人竟然直接用仇恨锁定法将怀疑目标放在他身上。

　　而一旦有了怀疑目标，再进行针对性调查的话，想要找到问题可就容易多了。

　　“林总，背后的人确定了，就是赵玉刚。”

　　林远办公室里，刘玉芳笑着对林远说道。

　　“我们拿赵玉刚的照片给供应商那边的人看，负责收购的摊贩确实见过赵玉刚。他一个城里人，还是县食品厂的厂长，好好的为什么会去卖玉米？”

　　林远听完也点点头：“没错，将这些信息反馈给警方吧，相信他们很快就能顺着线索搞到更多的证据。

　　还有咱们对于赵玉刚故意让食品厂亏损的消息也告诉警方，并且不排除赵玉刚担任厂长期间存在严重贪污行为的可能。”

　　“好的，林总。”

　　刘玉芳点头应了下来，随后转身离开了林远的办公室。

　　“我是不是也该找一个秘书了？”

　　看着刘玉芳的背影，林远在心里想到，按理说这种事情是不该刘玉芳这个厂长来做的，作为养猪场厂长只需要管好养猪场就行。

　　林远这边提供的情报引起了警方极大的重视，这里面涉及到的东西可不少。

　　甚至赵玉刚投毒的事情已经不是最重要的了，毕竟没有造成什么实际损失。

　　反而赵玉刚在食品厂的行为展露出来的问题更大，整个荷县那么多亏损的国营企业，是只有赵玉刚一个人在这么做，还是所有人都在这么做。

　　这可是对国有资产的系统性侵占，再重视也不为过。

　　为此荷县的县委书记韩宇星亲自挂帅，由纪委书记高小华配合，展开了针对荷县国有企业管理人员贪污腐败问题的专项整治行动。

　　而提前露出破绽的赵玉刚便是最好的突破口。

　　不过为了不引起其他人的警觉，在确定赵玉刚其确实存在投毒行为之后，荷县警方便先以投毒的名义抓捕了赵玉刚，然后再想办法从其口中了解其侵吞国有资产的细节。

　　不过这些暂时都与林远无关了，哪怕整治行动后面会不会继续向着曹州甚至鲁省扩展，也和林远无关。

　　他就是一个普通的养猪场老板而已。

　　现在摆在林远这个养猪场老板面前的还有一个更严峻的问题。

　　今年受通货膨胀影响，涨价的不止有猪肉，还有粮食和饲料的价格。

　　相比上半年，现在的粮价提高了至少30%，玉米的价格现在已经超过五毛一斤了，豆粕的价格更是达到了恐怖的一块多一斤。

　　虽然凭借远丰优秀料肉比和成本控制，利润还是相当可观，但是饲料价格一直这么涨也不是一个办法，这损失的可都是实实在在的利润。

　　还有之前的饲料问题也给林远提了一个醒，得将饲料来源控制在自己手里才行。

　　一方面可以进一步降低成本，另一方面则是能更好地保证供应和质量。

　　只不过，种植粮食比养猪场更加敏感，养猪场在领导的支持下，还能扩张到万头甚至十万头的规模。

　　换成粮食的话，虽然没有明确法律规定不能大规模承包，但是不鼓励大型工商企业参与粮食种植领域这句话，已经将意思表达得很明确了。

　　考虑许久，林远打算拿出前世很多企业都会使用的一种模式即“订单种粮”，由企业向农户下订单，提供种子、肥料，并按照合同价格收购。

　　虽然比不上自己直接下场种，但是也能在很大程度上控制成本了，并且还能带动农民致富，也能得到各级政府的支持。

　　除了订单种粮之外，林远还打算去一个很多重生者都喜欢去的地方种地，国内不能大规模的承包土地种粮，不代表国外不行。
