## Episode Mapping
- EP-06：第5章至第6章顾砚舟追求、赔罪与姜照眠建立边界。
  - 顾砚舟带着资源或歉意上门，却被姜照眠要求先为强行带走道歉。 -> 姜照眠把合作、赔偿、私情分开，不接受含混的偏爱。 -> 沈家或外部人物误以为她攀上顾砚舟，制造新的羞辱。 -> 顾砚舟出手解决现实麻烦，但把选择权交还给姜照眠。 -> 结尾落在顾砚舟第一次真正学会“等她同意”。
  - knowledge_boundary:
    - 顾砚舟知道自己要重建信任
    - 姜照眠可以感受到他变化，但不确认感情
    - 外人仍把她看作沈家风波中心。
- EP-07：第6章至第7章设计发布会、资料争夺、沈雨棠第一次公开反噬。
  - 开场用发布会现场事故切入：作品、灯光、模特或资料被动手脚。 -> 沈雨棠借“替沈家遮丑”或“维护名声”掩盖破坏行为。 -> 姜照眠现场调整，用专业能力把事故变成高光。 -> 证据不能只是屏幕一亮就赢，必须有现场阻力：断电、话筒被抢、负责人推责或宾客质疑。 -> 结尾让沈雨棠留下可追查破绽，为沈知微归来铺路。
  - knowledge_boundary:
    - 姜照眠可怀疑沈雨棠但未掌握全链条
    - 顾砚舟可辅助保护现场秩序
    - 沈家父母仍想压下丑闻。
- EP-08：第7章至第8章沈知微归来、姐妹关系建立、沈家旧伤重新打开。
  - 开场以沈知微突然出现或与姜照眠面对面切入，利用两人相似外貌形成强视觉冲击。 -> 沈知微不是单纯温柔姐姐，她要确认姜照眠有没有被沈家再次伤害。 -> 沈家父母试图把两个亲生女儿的归来包装成团圆，却被旧账打断。 -> 姜照眠与沈知微在短对话中建立初步信任，但彼此都保留观察。 -> 结尾落在沈知微发现当年失踪或姜照眠被弃并不简单。
  - knowledge_boundary:
    - 姜照眠知道沈知微身份
    - 沈知微知道姜照眠是真妹妹并怀疑沈家处理不当
    - 沈雨棠意识到自己地位同时被两人威胁。
- EP-09：第8章至第9章沈雨棠偷取或陷害设计资料、姐妹联手拆局。
  - 开场是资料失窃或姜照眠被指控抄袭，直接把事业线推入危机。 -> 沈雨棠利用沈家内部权限或旧关系制造“证据链”，让姜照眠短时间处于被动。 -> 沈知微不急着安慰，而是用她掌握的沈家信息帮姜照眠找破口。 -> 姐妹联手设置现场反证，让沈雨棠的说法当众自相矛盾。 -> 结尾阶段性小爽点：沈雨棠第一次在父母和外人面前无法圆谎。
  - knowledge_boundary:
    - 姜照眠与沈知微彼此信任上升
    - 沈家父母开始动摇但仍不愿彻底放弃沈雨棠
    - 沈雨棠知道姐妹联手后更急。
- EP-10：第9章至第10章沈家权力重排、旧案线索浮现。
  - 开场是沈家内部会议或公司权责变动，姜照眠与沈知微不再只是被安排的人。 -> 沈雨棠试图用“多年养育之恩”阻止权力转移。 -> 姜照眠提出只接自己能负责的部分，沈知微补上沈家长期隐瞒的旧账。 -> 顾砚舟带来外部线索，但不能替姐妹做决定。 -> 结尾抛出旧案线索：当年失踪、错抱或绑架背后另有人获利。
  - knowledge_boundary:
    - 姜照眠与沈知微知道旧案有问题，但不知道周振岳具体身份
    - 顾砚舟掌握部分调查线索
    - 沈雨棠知道旧事可能威胁自己。
