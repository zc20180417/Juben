  - knowledge_boundary:
    - 姜照眠知道陆知衡与沈知微关系
    - 沈知微知道自己可重新选择人生
    - 外部危机基本解除。
- EP-22：婚礼筹备、家庭修复与姐妹自主选择。
  - 开场用婚礼筹备冲突切入：流程、座次、姓氏或父母补偿方式再次触碰旧伤。 -> 姜照眠与沈知微共同决定婚礼怎么办，不让沈家用盛大弥补过去。 -> 沈怀章和许兰因学习退后，接受女儿们自己定规则。 -> 顾砚舟与陆知衡分别配合，而不是替她们拍板。 -> 结尾落在婚礼前夜，姜照眠看到过去与现在形成对照。
  - knowledge_boundary:
    - 双方关系稳定
    - 父母仍在修复中
    - 姐妹已经拿回重大生活选择权。
- EP-23：婚礼前夜、设计师身份与亲情和解的最终准备。
  - 开场是婚纱或礼服出现问题，姜照眠用自己的设计能力完成最后一次高光。 -> 沈知微陪她处理，不再是危机中的战友，而是家人。 -> 沈家父母送出迟来的祝福，但姜照眠只接受真诚，不接受赎罪表演。 -> 顾砚舟看到她为自己设计未来，而不是被任何人安排。 -> 结尾卡在婚礼大门打开前。
  - knowledge_boundary:
    - 所有人都知道姜照眠的能力和边界
    - 亲情修复进入可持续状态
    - 婚礼尚未正式开始。
- EP-24：婚礼现场、双重誓言、情感高潮。
  - 开场直接进入婚礼现场一个强情绪瞬间，不要铺陈宾客。 -> 姜照眠与顾砚舟的誓言要回应“错认、尊重、选择”三层关系变化。 -> 沈知微与陆知衡的誓言回应“失散、等待、重逢”。 -> 沈家父母在台下接受女儿们已独立，不再抢占中心。 -> 结尾落在双重誓言完成，全场情绪释放。
  - knowledge_boundary:
    - 主线真相已清
    - 角色之间没有隐藏误会
    - 本集承担情绪高潮而非新悬念。
- EP-25：全书尾声，婚后与家庭关系闭环。
  - 开场用婚礼后安静但有行动的场面切入：姜照眠回望来路或回到工作室。 -> 姜照眠确认自己不是“被沈家找回的人”，而是自己走回光里的人。 -> 顾砚舟、沈知微和家人分别给出不越界的陪伴。 -> 过去的旧物、街头、车门或礼服形成首尾呼应。 -> 结尾闭环：姜照眠拥有姓名、事业、亲情边界与爱情选择，不再被弃。
