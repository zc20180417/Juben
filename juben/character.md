# 人物小传

## 主要角色
**沈青鸾（女主 / 庶女 / 后宫上位者）** 十七岁入宫，眼尾一颗朱砂痣，越素越显锋利。她最擅长的是忍、等、借刀，不把底牌一次说完。生母柳姨娘被害死，是她一切复仇和上位的原点。她入宫最初只为借权报仇，走到后面却一步步把目标变成掌权。这个角色必须同时有被逼出来的狠、对代价的感知，以及偶尔被情感击中的裂缝。

**萧景珩（男主 / 皇帝）** 少年登基，掌控欲强，多疑，习惯用试探和压迫确认别人是否可信。他对沈青鸾从好奇、征服欲，慢慢滑向依赖与失控。这个人物既是帝王，也是女主最锋利的一把刀，必须同时撑住权力压迫与后期情感失衡。

## 反派角色
**沈夫人（主母 / 家宅大反派）** 代表家宅旧秩序，用规矩与体面包装恶毒。她害死柳姨娘，长期把沈青鸾压在脚下，是女主最早要踩碎的那堵墙。

**沈丞相（父亲 / 利己权臣）** 冷眼旁观柳姨娘之死，只在“有用”时才想起沈青鸾是自己的女儿。他不是最响的恶人，却是最冷的共谋者。

**沈明珠（嫡姐 / 家仇对象 / 中期宫斗反派）** 自小被捧着长大，骄纵、短视、贪恋高位。她对沈青鸾的压迫带着嫡庶优越感，入宫后则逐步从家中对手变成宫中替身反派。

**宸妃（前期后宫大敌）** 将门之女，嚣张跋扈，最爱当众立威。她适合成为女主用帝王之手完成第一次立威的磨刀石。

**淑妃（中期慢刀子反派）** 表面温婉，实则习惯借别人之手杀人。她不靠声势压人，而靠毒、谣言和猜忌让人慢慢失势。

**皇后（后宫旧秩序之主）** 端庄稳重，真正危险处在于她懂得用名分、规矩和储位杀人。她代表的是整个后宫秩序本身。

## 重要配角
**柳姨娘（生母 / 复仇原点）** 江南绣娘出身，温婉清丽，手极巧。她死得很早，却一直以旧物、旧话、旧伤的形式活在女主后半生里。

**翠竹（贴身丫鬟 / 情感支点）** 直来直去，忠心外露，是沈青鸾在深宫里最少数能放下一层防备的人。她的死不是单纯虐点，而是女主从“借势”转向“主动猎杀”的真正转折。

**云太医（暗棋盟友）** 清瘦谨慎，早年受过沈青鸾恩惠，因此在太医院中暗中为她效力。他既是功能型盟友，也承担“见证女主越来越狠”的作用。

**太子萧承乾（储君 / 国本障碍）** 年少骄横，既蠢又狠，是皇后和储位主线的关键连接点。沈青鸾对他的处理，标志着她从后宫斗争正式进入国本斗争。

**阿碧（宫女 / 借腹生子工具人）** 出身寒微，性格怯懦，只想平安活下去。她被卷入换子与储位布局，是后期权势代价的承受者。

## 关系核心
- 沈青鸾 × 萧景珩：试探、利用、失控、放不下
- 沈青鸾 × 翠竹：主仆之上，接近亲人
- 沈青鸾 × 沈夫人 / 沈明珠：家仇、羞辱、反击
- 沈青鸾 × 皇后：名分与秩序之争
- 沈青鸾 × 太子：从利用到清除的国本斗争
