# Character Anchor

## 改名映射

- 姜照眠 -> 原著时鸢
- 顾砚舟 -> 原著傅斯年
- 沈雨棠 -> 原著苏雨柔
- 沈知微 -> 原著苏清月
- 沈怀章 -> 原著苏振宏
- 许兰因 -> 原著刘美兰
- 陆知衡 -> 原著陆知衍
- 周振岳 -> 原著林振雄

## 姜照眠

- 功能：女主，真千金，顶级设计师“眠”。
- 外在：清冷、利落、低调，不主动解释身份。
- 内核：有边界感，不求迟来的亲情，不靠哭诉赢同情。
- 弱点：对“被选择、被保护”仍有迟疑，亲情伤口深。
- 写作要点：她的反击要短、准、冷，不长篇输出委屈；她的高光来自行动、身份和证据。
- 禁忌：不能写成撒娇求宠型女主；不能让男主替她完成所有打脸。

## 顾砚舟

- 功能：男主，权势守护者，错认入口制造者。
- 外在：强势、克制、压迫感强。
- 内核：前期有执念和越界，后期学会尊重姜照眠的主体性。
- 弱点：最初把姜照眠投射成旧人，容易用“保护”包装控制。
- 写作要点：他可以拦人、护人、压场，但要逐渐承认“脸像不等于人归他”。
- 禁忌：不能一直霸总式强抢；不能让他成为全剧唯一解题人。

## 沈雨棠

- 功能：主反派，假千金，白莲式操控者。
- 外在：柔弱、会哭、会替别人说好话。
- 内核：贪婪、嫉妒、占有欲强，害怕失去沈家资源。
- 升级路径：暗踩 -> 栽赃 -> 勾结外部 -> 转移财产/旧案黑手 -> 终极掉马。
- 写作要点：每次作恶必须带来新后果，不能只重复撒娇。
- 禁忌：不洗白，不突然悔悟成受害者。

## 沈知微

- 功能：亲姐姐，亲情补偿与智谋盟友。
- 外在：温婉、沉稳、有阅历。
- 内核：强保护欲，查案能力强，能温柔也能出手。
- 写作要点：她不是只会抱妹妹哭的人，要能推动证据线和旧案线。
- 禁忌：不能抢走女主主线，但要成为姐妹并肩的关键支点。

## 沈怀章

- 功能：沈父，权威压迫与后悔线。
- 外在：重面子、讲家族利益、语气硬。
- 内核：前期偏听偏信，后期被现实打醒。
- 写作要点：他的悔悟要付出代价，不能一句“爸爸错了”就和解。

## 许兰因

- 功能：沈母，情感偏心与迟到补偿。
- 外在：更情绪化，容易被沈雨棠带节奏。
- 内核：既有母性，又有势利和偏见。
- 写作要点：她的伤人台词要来自“规矩、教养、感恩”的压迫。

## 陆知衡

- 功能：沈知微丈夫，后段情绪闭环。
- 外在：温润但强大，成熟稳定。
- 内核：支持妻子查真相，愿意一起保护姜照眠。
- 写作要点：登场要带来“姐姐也被好好爱着”的情绪满足。

## 周振岳

- 功能：外部同谋/旧案线推手。
- 外在：老练、贪婪、善于甩锅。
- 内核：以为自己能操盘沈家和姐妹，实际被沈雨棠利用。
- 写作要点：用于制造后段危机，最终和沈雨棠一起被证据反杀。
