# Character Reference

用于记录人物关系、立场、经历与镜头抓手，不负责规定固定台词。

## 填写原则

- 优先写人物处境、关系压力、欲望、软肋、外在抓手
- 不要把人物小传写成剧情复述，也不要直接替角色发明口号式台词
- 本轮是原创化改编：保留人物功能，不保留原著姓名、行业身份、具体事件外壳

## 核心人物

### 陆行川

- 功能定位：重生归来的供应链创业者，负责把未来认知转化为冷链、加工、渠道、原料和组织能力。
- 欲望：保住家里旧冷库，借 90 年代县域食品工业窗口建立可持续供应链集团。
- 软肋：起步阶段没人相信他；越往后越容易被误解成投机或赌运气。
- 镜头抓手：按住印泥、撕转让合同、盯温控表、站在冷库机房听机器声、在货架前看断货、在原料风控表上画退出线。
- 写作边界：不能写成全知神仙。每次判断都要落到可验证行动：订单、设备、检测、渠道、贷款、风控、授权。

### 陆振海

- 功能定位：父亲与旧厂根基，代表上一世被时代误判压垮的人。
- 欲望：尽快把烂冷库变现，别让家里继续背债。
- 软肋：相信眼前现金和熟人经验，对新渠道、新设备、新金融工具天然警惕。
- 镜头抓手：攥合同、看电表、摸结霜的管道、翻旧账本、在工人面前替儿子压住场。
- 写作边界：不能降智成只会反对。他的质疑要有现实依据，认可必须来自亲眼见到结果。

### 段启荣

- 功能定位：开局压力人物，代表低价吃下旧资产的倒爷路径。
- 欲望：趁陆家急用钱，低价拿走冷库和地皮。
- 镜头抓手：转让合同、印泥、皮包、催盖章的手。
- 写作边界：不必长期复杂化，早期用来制造逼签现场和陆行川的第一场反击。

### 贺远桥

- 功能定位：地方权力与政策背书。
- 欲望：看见真实就业、税收、安全生产和产业项目，不愿为空话背锅。
- 软肋：地方发展压力大，既想抓机会又怕出事故。
- 镜头抓手：检查冷库、翻检测单、追问消防和用电、在县里会议上压住反对声。
- 写作边界：支持必须有交换条件和审慎过程，不能无条件当主角外挂。

### 梅知夏

- 功能定位：职业经理人，推动陆行川从家庭小厂走向组织化企业。
- 欲望：找到能真正放权、又能让她做成事的平台。
- 软肋：不接受拍脑袋式管理，会追问流程、财务和职责边界。
- 镜头抓手：摊开组织架构、指出账目漏洞、拿走人事章、当场设立追责表。
- 写作边界：要专业、克制、敢反问，不写成崇拜型女助理。

### 沈砚山

- 功能定位：生产与品控负责人。
- 欲望：让冷库、速冻线、包装和交付稳定跑起来。
- 镜头抓手：制冷机、温控表、速冻线、检测单、维修扳手。
- 写作边界：务实，不空喊口号；他的戏围绕“能不能按时、安全、低成本交付”。

### 乔南枝

- 功能定位：品牌和渠道负责人。
- 欲望：把无名小厂的速冻产品送上终端货架，并让消费者愿意复购。
- 镜头抓手：冷柜、包装样袋、铺货单、渠道报价、样板店货架。
- 写作边界：不能只报数字。她要提出市场阻力：竞品封锁、渠道费、陈列位置、口味反馈。

### 韩策

- 功能定位：大宗原料采购和风控负责人。
- 欲望：用规则锁住成本，但不让企业死在波动里。
- 镜头抓手：仓位表、采购单、风控线、浮盈后的沉默。
- 写作边界：不是赌徒。每次操作都必须绑定企业采购需求、仓位纪律和退出条件。

### 秦峥

- 功能定位：硬执行型人才，补足车队、仓储和现场秩序。
- 欲望：把战场上的执行力转化为企业纪律。
- 镜头抓手：车队排班表、仓门、封条、现场制止冲突。
- 写作边界：少说狠话，多用行动建立压迫感；不是打手。

### 叶承稷

- 功能定位：食品工程与加工科研资源入口。
- 欲望：让科研成果真正落地，但不愿被企业短期化。
- 镜头抓手：实验记录、样品柜、保鲜测试、工艺参数。
- 写作边界：不能被钱一句话打动，要看陆行川是否懂长期投入和科研边界。

### 许望舒

- 功能定位：加工适配与原料转化人才。
- 欲望：离开不顺的旧体系，把技术推广真正做起来。
- 镜头抓手：原料样本、工艺表、被搁置的实验资料。
- 写作边界：不是普通打工人，加入应是双向选择。
