# 人物小传

## 主要角色
**沈青鸾（女主 / 庶女 / 后宫上位者）** 十七岁入宫，肌肤冷白，眉眼精致，眼尾一颗朱砂痣，越是素净越显得惊心动魄。她表面柔顺安静，说话轻缓，永远带着三分怯意，实则心思极深，隐忍克制，擅长看人下菜、借刀杀人，对自己狠，对别人更狠。她幼年亲眼目睹生母柳姨娘被主母活活打死，从那一夜起便把“活下来、爬上去、让所有人血债血偿”刻进骨血里。入宫对她不是求宠，而是借权复仇；皇帝对她不是归宿，而是最锋利的一把刀。她一路从答应爬到皇后、太后，完成从受害者到权力掌控者的蜕变。她与萧景珩的关系始于试探和利用，终于复杂而迟来的情感震动，是全剧最危险也最动人的情感线。

**萧景珩（男主 / 皇帝）** 二十五岁上下，少年登基，容貌俊美而压迫感极强，常年身着玄色龙袍，眉眼冷峻，一开口便让人不敢直视。他天性多疑，极重帝王权术，习惯掌控一切，对女人的态度向来是赏玩与利用并存，从不轻易动心。可沈青鸾偏偏不按他的规则来，她越退越让他想逼近，越难懂越让他着迷。他对沈青鸾从好奇、征服欲，慢慢变成依赖与偏爱，最后甚至清楚她在利用自己，仍旧放不下她。作为帝王，他是沈青鸾借势上位的台阶；作为男人，他又是她一生中唯一真正接近“爱”的人。这个角色必须同时具备帝王威势和深情失控，才能撑住全剧感情与权谋两条主线。

## 反派角色
**沈夫人（主母 / 家宅大反派）** 四十出头，保养得宜，衣着华贵，笑时慈眉善目，不笑时像一尊冷硬佛像。她出身名门，最看不起柳姨娘这样的低贱妾室，认为整个相府后宅都该由自己掌控。她心狠手辣，擅长用“家法”“规矩”“体面”包装自己的恶毒，是害死柳姨娘的罪魁祸首。她对沈青鸾从来只有利用和打压，把这个庶女当成一条可随时打死的狗。后期她被流放后仍不改高高在上的姿态，最终在沈青鸾精心设计的报复中失去一切，是女主家仇线最核心的仇敌。

**沈丞相（父亲 / 利己权臣）** 五十岁上下，外表儒雅端方，实则最懂趋利避害。他不亲自动手，却默认主母虐杀妾室，只因一个绣娘的命在他眼里不值一提。他对沈青鸾的态度冷淡而功利，只在她有用时才想起这是自己的女儿。这个人物最可恨之处不在张牙舞爪，而在“冷眼旁观”和“父权共谋”。他从朝堂高位跌入流放泥潭，正是沈青鸾对“男人掌权却假装无辜”的第一记重击。

**沈明珠（嫡姐 / 家仇对象 / 中期宫斗反派）** 十八岁，明艳张扬，珠翠满头时像一朵被富贵堆出来的牡丹，漂亮却浮躁。她自小被捧着长大，骄纵跋扈，最擅长把庶妹踩在脚下找优越感，从不觉得沈青鸾和自己是同类。她既怕进宫送死，又不甘心失去高门婚配和荣华富贵，贪婪、自私、短视，是最容易被人利用的那种反派。入宫后，她以为自己终于有机会翻盘，结果一步步沦为皇后和沈青鸾博弈中的棋子。她和沈青鸾之间既有家仇，又有“嫡庶之争”“谁才配站上高位”的镜像关系，是最有撕扯感的女性对手之一。

**宸妃（前期后宫大BOSS）** 二十岁上下，将门之女，容貌艳丽，气势逼人，爱穿最张扬的颜色，走到哪里都像在示威。她嚣张跋扈，脾气暴烈，仗着家世和宠爱横行后宫，最爱踩新入宫的嫔妃立威。她不是最聪明的敌人，却是最适合拿来做第一阶段“立威祭旗”的人。她的存在让观众迅速理解后宫规则有多残酷，也给沈青鸾提供了第一个能漂亮反杀的舞台。她被借巫蛊案打入冷宫，是女主第一场高光复仇的标志性胜利。

**淑妃（中期心机反派）** 二十五岁左右，长相温婉清秀，声音柔和，礼数周全，像后宫里最不会惹事的那一类女人。可她真正厉害之处，在于从不亲自出头，擅长借别人之手完成自己的谋算，是比宸妃更难缠的慢刀子型敌人。她惯于下毒、布谣言、引猜忌，让别人一步步死在看不见的地方。她与沈青鸾之间的对抗不是硬碰硬，而是静水深流式的智斗，最适合拉开中段宫斗质感。

**皇后（终极后宫反派）** 三十岁左右，端庄华贵，永远一副母仪天下、波澜不惊的姿态，看起来最守规矩，实际上最懂利用规矩杀人。她掌着后宫秩序，也掌着储位和嫡庶名分，是沈青鸾真正意义上的天花板型对手。她不屑亲自脏手，却总能让别人替自己动刀，是宸妃和沈明珠背后真正的操盘者之一。她最大的软肋是太子，也是她覆灭的起点。这个角色代表的是正统、名分和旧秩序，只有把她拉下来，沈青鸾才算真正改写命运。

## 重要配角
**柳姨娘（生母 / 复仇原点）** 去世时不过三十来岁，出身江南绣娘，容貌温婉清丽，手巧心善，最鲜明的印记就是与沈青鸾一模一样的朱砂痣。她身份卑微，却从不轻贱自己，哪怕被打到临死，也没有真正低头。她虽然很早退场，却是贯穿全剧的精神幽灵：沈青鸾每一次往上爬，都是在替她讨回尊严。她不是单纯的“死去的母亲”，而是女主所有狠和冷的根源。

**翠竹（贴身丫鬟 / 忠仆 / 情感支点）** 十六七岁，圆脸大眼，动作利落，说话直来直去，跟在沈青鸾身边时总透着一股不服输的机灵劲。她是少数真心把沈青鸾当亲人而不是主子的人，知道主子心狠，却始终不退。她既是沈青鸾在深宫里最早的温度，也是观众代入女主情感的出口。后期她为护主惨死，这一死会直接推动沈青鸾从“为复仇而活”进一步转向“主动猎杀一切拦路者”，是全剧一个极重要的情绪转折点。

**云太医（暗棋 / 医者盟友）** 二十七八岁，清瘦儒雅，不善高声言语，做事谨慎，眼底却藏着很深的感恩和审慎。早年受过沈青鸾恩惠，因此在太医院中暗中为她效力。毒、药、病、孕、假孕，他几乎参与了沈青鸾后期每一步最不能见光的计划。这个角色既是工具型盟友，也承担“见证女主越来越狠”的旁观功能。他越知道真相，就越明白自己已与她绑在一条船上。

**太子萧承乾（储君 / 后位之后的下一道障碍）** 十五岁左右，出身高贵，年少骄横，从小被母亲和地位惯坏，既蠢又狠。少年人的傲慢让他很容易被纵坏，也很容易被推到失德失民心的深渊。他不是最强的敌人，却是最关键的政治障碍，因为他的存在决定了皇后和储位仍旧牢牢压在沈青鸾头上。沈青鸾对他的处理，标志着她从后宫斗争正式升级到国本斗争。

**阿碧（宫女 / 借腹生子工具人）** 十八九岁，出身寒微，性格怯懦，最大的愿望不过是平安活下去。她因为怀孕而被沈青鸾看中，短暂地被好吃好喝供起来，以为自己终于遇上贵人。她对权谋全然无知，只是一枚被卷进帝后棋局的可怜棋子。这个角色的命运很残忍，也最能体现沈青鸾后期为了权势已不再给任何人留退路。

## 其他角色
**丽嫔** 二十出头，容貌艳俗，依附强者而活，最会跟风踩人。她本事不大，恶意不小，是最适合被拿来做“流产局”牺牲品的爪牙型反派。

**幼帝萧景睿** 婴孩登基，少年时被当作权力符号养大。名义上是天子，实则从出生起就是沈青鸾稳固太后之位的工具，长大后想挣脱掌控，却反而成为她废帝夺权的最后一块踏板。
